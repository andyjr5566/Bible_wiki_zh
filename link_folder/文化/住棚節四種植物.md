# 住棚節四種植物

## 定義

利未記23章40節規定，住棚節第一日要拿「美好樹上的果子」和「棕樹上的枝子」，與「茂密樹的枝條」並「河旁的柳枝」，在神面前歡樂七日。在猶太傳統中，這四種植物被具體識別為香櫞（美好樹的果子）、棕樹枝、芭樂樹枝（茂密樹）和柳樹枝。法利賽人將這四種樹枝束成紮拿在手中搖一搖歡樂；而撒都該人則認為這些樹枝是為了支搭和妝飾棚子所用。這些植物象徵神賜福的繁榮、和平與新鮮，並見證了迦南美地豐盛的出產。
