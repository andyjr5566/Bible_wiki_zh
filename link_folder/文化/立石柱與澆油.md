# 立石柱與澆油

## 定義

古代近東以立石作記號或紀念碑，並以澆油表示奉獻、分別為聖的做法。雅各把枕過的石頭立起並澆油，用來記念神在該處向他顯現。

在創世記第二十八章，石柱不是神的形像，而是事件的記念。CT/GT 說明立石也可作立約記號，澆油則使該物成為神聖地點的標誌；後來迦南祭儀中的柱像崇拜受律法禁止，應與雅各此處的紀念行動區分。

## 主題發展

立石與澆油在此結合「記念」和「奉獻」兩層意義；其價值在於指向神的顯現，而不在石頭本身。
