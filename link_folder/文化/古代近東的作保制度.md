# 古代近東的作保制度

## 定義

「作保」表示擔保人為另一人的安全或債務承擔責任。「從我手中追討」使用責任追索的語言：若受保人不能交還，擔保人要親自承擔後果。創世記四十三章並未列出具體刑罰，重點是猶大把責任從便雅憫轉到自己身上。

## 主題發展

作保的價值不只在口頭保證，更在擔保人是否願意實際承受受保人的後果。
