# 古代近東神明裁判（致咒詛苦水的背景比較）

## 定義

「神明裁判（Ordeal）」在古代近東（如美索不達米亞、巴比倫、馬里等）是一種常見的法律審判機制，意指當缺乏人間見證人或直接證據時，將被告置於某種物理危險環境中（如手持熱鐵、腳踏紅犁頭、手伸滾油、喝毒水或投入大河），若被告未受傷害或活著浮起，即被認為獲得神明庇佑而被判無罪；反之則被定罪（GT《背景》、BH）。例如《漢摩拉比法典》規定被控行淫或行巫術者需投入河中接受河神裁判；馬里文獻亦記載神明裁判案例，要求神明喝飲混和城門泥土的水以約束履行誓約（GT《背景》）。然而，民數記第5章的「致咒詛苦水」條例與古代近東異教神明裁判有著本質上的巨大差異：(1) 無物理傷害與毒素：異教裁判多使用酷刑、毒藥或危險環境，非一般人體所能承受（致命或傷殘）；而以色列的苦水僅為聖水混和會幕塵土與墨水，水本身完全無毒害（GT、BH）；(2) 無罪推定原則：異教裁判預設被告有罪直至神蹟發生；摩西律法預設婦人清白，除非神親自施行超自然懲罰（BH）；(3) 排斥巫術迷信：異教裁判依賴法術與咒文強制神明；民數記條例完全排斥巫術，強調全能耶和華神親自作為公義審判者（GT《串珠》、BH）。
