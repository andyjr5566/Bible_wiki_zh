# 六日復造的結構

## 定義

創世記 1:3-31 記載六日復造，每日以「神說」開始，以「有晚上，有早晨，是第X日」結束。結構嚴謹有序。

## 核心摘要

### 與目前整理書卷的關聯

六日復造是有秩序的：先準備環境（光暗、穹蒼、水地），再填充生命（植物、光體、動物、人）。每一日都為下一日鋪墊。

## 主題發展

### 神學意義／研讀意義

- 神的創造有秩序，非混亂即興
- 六日結構：準備 → 填充 → 高峰（人）
- 猶太算法：從日落至次日日落為一日；先晚上後早晨
- 六日復造豫表信徒重生的經歷（光→分別→復活→屬天→得勝→成熟）
