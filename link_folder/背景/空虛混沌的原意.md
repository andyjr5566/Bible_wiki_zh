# 空虛混沌的原意

## 定義

「空虛混沌」（*tohu wa-bohu*，תֹהוּ וָבֹהוּ）是創世記 1:2 描述地審判後的光景。原文有「變成了」之意，表示 1:1 和 1:2 之間經過了變化。

## 核心摘要

「空虛混沌」（*tohu wa-bohu*，תֹהוּ וָבֹהוּ）是創世記 1:2 描述地審判後的光景。原文有「變成了」之意，表示 1:1 和 1:2 之間經過了變化。

### 與目前整理書卷的關聯

1:2 的「地是空虛混沌」原文有「變成了」之意，表示地原本美好，後因審判變成荒涼。這為六日復造提供了背景：神在審判後的荒涼中恢復萬有。

## 主題發展

### 神學意義／研讀意義

- 地變荒涼是神審判的結果：原初天使長（撒但）背叛神，牽連全地受審（賽十四12-15；結廿八15-19；啟十二4-9）
- 「淵」說明審判是用水淹滅（彼後三6）
- 前半段豫表人得救前的光景——空虛、黑暗、在審判權下
