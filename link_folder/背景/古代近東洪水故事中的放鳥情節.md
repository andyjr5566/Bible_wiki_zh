# 古代近東洪水故事中的放鳥情節

## 定義／基本資料

創八6-12挪亞放鳥探測水勢的情節，與古代近東洪水史詩（吉加墨斯史詩、阿特拉哈西斯史詩）有類似情節，但次序和意義不同。

## 觸發來源

- [[創世記/第8章|創世記 第8 章]]：6-12節

## 與目前整理書卷的關聯

### 創八與古代近東的對照

| 來源 | 放鳥次序 |
|------|----------|
| 聖經（創八） | 烏鴉→鴿子→鴿子→鴿子 |
| 吉加墨斯史詩 | 鴿子→燕子→烏鴉 |
| 阿特拉哈西斯史詩 | 類似但不完全相同 |

關鍵差異：聖經中烏鴉先放（不潔淨），鴿子後放（潔淨）；古代近東史詩中鴿子先放，烏鴉最後放且不再回來表示水退。

## 神學意義

聖經的放鳥次序具有潔淨/不潔淨的象徵意義，而非單純的探測工具。
