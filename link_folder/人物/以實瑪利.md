# 以實瑪利

## 定義／基本資料

以實瑪利（Ishmael），亞伯蘭與夏甲所生之子。名字意為「神聽見」。豫表按血氣生的（加四23），是阿拉伯人的祖先。

## 與目前整理書卷的關聯

### 創世記 第16章

- 耶和華使者豫言以實瑪利的性格：「像野驢，手攻打人」
- 以實瑪利住在眾弟兄的東邊
- 亞伯蘭八十六歲時生以實瑪利
- 來源：CT, GT, KC, BH

### 創世記 第21章

- 羅得戲笑以撒（9節）——屬肉體的逼迫屬靈的（加四29）
- 撒拉要求趕出夏甲和以實瑪利（10節）
- 神仍應許使以實瑪利後裔成立一國（13、18節）
- 在別是巴曠野水盡，神聽見童子聲音（17節）——以實瑪利＝「神聽見」的雙關
- 神開夏甲眼睛看見水井（19節），保佑他長大
- 成為弓箭手，母親從埃及娶妻（20–21節）
- 來源：CT, GT, KC, BH

### 創世記 第25章

- 以實瑪利十二族族長：尼拜約、基達、亞德別、米比衫、米施瑪、度瑪、瑪撒、哈大、提瑪、伊突、拿非施、基底瑪（13-15節）
- 照著村莊、營寨，作了十二族的族長（16節）
- 以實瑪利享壽137歲，氣絕而死，歸到他列祖那裡（17節）
- 子孫住處：哈腓拉至書珥，正在亞述道上，在眾弟兄東邊（18節）
- 以撒與以實瑪利共同埋葬亞伯拉罕於麥比拉洞（9節）
- 來源：cnbible, CT, GT, KC

- 以掃往以實瑪利家，又娶其女兒瑪哈拉；此時以實瑪利本人已去世。

- 來源：BH、KC、CT、GT
