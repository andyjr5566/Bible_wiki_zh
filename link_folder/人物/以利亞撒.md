# 以利亞撒

## 定義

出埃及記6:23「亞倫娶了亞米拿達的女兒……給他生了拿答、亞比戶、以利亞撒、以他瑪。」以利亞撒是亞倫的第三個兒子，後來接續亞倫為大祭司。

- 本章重點：民4:16 祭司亞倫的兒子以利亞撒掌管點燈的油、馨香的香、常獻的素祭、和膏油，並掌管全帳幕和其中一切所有的。以利亞撒為亞倫三子，掌管聖物供應。

## 主題發展

以利亞撒在兄長拿答、亞比戶獻凡火而死（利十1-2）後，接續亞倫為大祭司（民二十23-26）。他兒子非尼哈後來以熱心止息瘟疫，得著永恆祭司之約（民二十五10-13）。
