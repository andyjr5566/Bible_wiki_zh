# 迦南

## 定義

含的兒子，挪亞的孫子。因含看見挪亞赤身的事，受挪亞咒詛「必給他弟兄作奴僕的奴僕」。

### 與目前整理書卷的關聯

### 創世記第9章

- 含看見挪亞赤身，到外邊告訴兩個兄弟
- 挪亞咒詛迦南：「必給他弟兄作奴僕的奴僕」——參[[為何咒詛迦南而非含]]
- 咒詛的應驗：約書亞時代迦南人被以色列征服（書九23-27）；所羅門時代完全被征服（王上九20-21）

### 創世記第10章

- 迦南的後裔：西頓（長子）、赫、耶布斯人、亞摩利人、革迦撒人、希未人等
- 迦南的境界（19節）：北從西頓到南迦薩、東從所多瑪到拉沙
- 非利士人從迦斐托（克里特島）出來，首次在此提及

## 主題發展

咒詛迦南即咒詛含。挪亞在聖靈感動下不自覺說出豫言。
