# 比拉（所多瑪王）

## 定義／基本資料

所多瑪王比拉（Bera），名字意為「禮物」，但亦含「邪惡中的毀滅」之意（"gift, destruction in wickedness"）。其名與其城所多瑪的邪惡相呼應。
