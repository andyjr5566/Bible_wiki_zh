# 閃

## 定義

希伯來文 *Shem*（שֵׁם），意思是「名、名聲」。挪亞的長子（或次子，見待確認事項）。閃族（Semites）的祖先，以色列人及亞伯拉罕的先祖。

## 按書卷累積

### 觸發來源

- [[創世記/第5章|創世記 第5章]]：32 節

### 聖經出現

- [[創世記/第5章|創世記 第5章]]：32 節「挪亞活到五百歲，生了閃、含、雅弗」

### 與目前整理書卷的關聯

閃的後裔延續到亞伯拉罕，最終到基督（路三23, 36）。神被稱為「閃的神」（創九26）。

## 主題發展

### 神學意義／研讀意義

- 彌賽亞譜系經閃延續（太一1-17；路三23-38）
- 屬靈歷程：「閃」（名聲）——結實纍纍，滿得勞苦的果效

### 待確認事項

- 閃是否為長子有爭議：創十一10 顯示挪亞生長子時為 502 歲，故閃可能非長子
