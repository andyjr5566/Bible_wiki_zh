# 利百加

## 定義

利百加（Rebekah），彼土利之女，拿鶴的孫女。後來成為以撒的妻子，以實瑪利和以撒之母。豫表教會——神為祂兒子所預備的新婦。

### 創世記 第26章

- 以撒因怕利百加容貌俊美被殺，稱她為妹子（重蹈亞伯拉罕覆轍）
- 亞比米勒揭穿謊言並下令保護以撒夫婦
- 以撒耕種百倍收成，成為大富戶
- 非利士人嫉妒，塞住水井；亞比米勒要求以撒離開
- 以撒三次挖井（埃色、西提拿、利河伯），退讓不爭，神賜寬闊之地
- 神二次顯現於別是巴，以撒築壇求告、支搭帳棚、挖井
- 與亞比米勒立約互不侵犯
- 百歲時以掃四十歲娶赫人女子，常使以撒和利百加愁煩
- 來源：CT, GT, KC, BH

### 創世記 第27章

- 利百加聽見以撒要給以掃祝福，設計讓雅各騙取祝福（5-17節）
- 偏愛雅各（三十五28），對以撒說「你哥哥」「你父親」——語氣冷漠（CT）
- 教唆雅各欺騙丈夫，甘願承擔咒詛（13節）
- 得知以掃要殺雅各，安排雅各逃往哈蘭（42-45節）
- 對以撒抱怨赫人女子，趁機打發雅各離開（46節）
- 雅各一去二十年，母子再未相見（CT）
- 利百加的錯：未仰望等候神，想用手段成全神的旨意（CT）
- 來源：CT, GT, KC, BH
