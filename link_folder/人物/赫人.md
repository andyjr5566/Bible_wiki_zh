# 赫人

## 定義

赫人（Hittites），迦南地原住民之一，原文可能意為「恐怖」。是古代近東的強國，版圖從希伯侖到伯拉大河。

### 與目前整理書卷的關聯

### 創世記 第15章

- 神列舉十族作為迦南地的現有居民，赫人是其一
- 赫人後來成為亞伯拉罕購買麥比拉洞時的賣主（創廿三）
- 來源：GT

### 創世記 第23章

- 亞伯拉罕向住在基列亞巴（希伯崙）的赫人求購墳地
- 以弗崙（赫人，瑣轄之子）以400舍客勒高價出售麥比拉田連洞
- 交易在城門口當眾完成，有法律保障
- 赫人稱呼亞伯拉罕為「尊大的王子」「有神威的王子」
- 來源：CT, GT, KC, BH

### 創世記 第25章

- 亞伯拉罕被埋葬在麥比拉洞，這洞在幔利前、赫人瑣轄之子以弗崙的田中（9節）
- 來源：CT, GT, BH

### 創世記 第26章

- 以掃娶赫人女子猶滴和巴實抹為妻，表明不看重神的約
- 她們常使以撒和利百加心裡愁煩（極因敬拜外邦偶像）
- 來源：CT, GT, BH
