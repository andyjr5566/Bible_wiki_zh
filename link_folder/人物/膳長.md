# 膳長

## 定義

「膳長」是古埃及王宮中掌管飲食、負責法老膳食安全的高階官員。創世記40-41章記載膳長與酒政同得罪法老、被關在「王的囚犯被囚之處」，約瑟為他們解夢：膳長夢見三個筐餅、鳥吃頂上餅，約瑟解為三天後被掛木架、鳥吃肉。應驗後膳長被處死。

## 主題發展

膳長與酒政雙重見證結構：同囚、同夢、同求解、同應驗（第三天法老誕辰），卻生死不同命。KC、BH指出：此雙重結構強調神主權決定生死，約瑟只傳達神旨意。膳長被掛木架預表基督被掛木頭（加3:13），但膳長因己罪、基督為人罪。
