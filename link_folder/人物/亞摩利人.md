# 亞摩利人

## 定義

亞摩利人（Amorites），古代迦南地的重要民族之一，常作為迦南諸族的統稱。「亞摩利」意為「高處之人」或「西方人」。在創世記十四章中，亞摩利人指的是希伯崙一帶的部族，[[幔利]]、[[以實各]]、[[亞乃]]即為亞摩利人的首領。
