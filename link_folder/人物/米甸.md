# 米甸

## 定義

基土拉與亞伯拉罕的第四個兒子（創25:2）。名字意為「爭辯、衝突」。其後代成為米甸民族。

## 主題發展

米甸人在舊約歷史中扮演重要角色：
- 摩西逃往米甸，娶米甸祭司葉忒羅之女為妻
- 基甸時米甸人聯合亞瑪力人轄制以色列七年（士6:1-6）
- 神使用基甸三百人擊敗米甸全軍（士7章）
- 以賽亞預言米甸人將帶著黃金到錫安敬拜（賽60:6）
- 米甸人居住在阿拉伯半島西北部，從事香料貿易
