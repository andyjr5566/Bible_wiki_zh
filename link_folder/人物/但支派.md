# 但支派

## 定義

但支派是以色列十二支派之一，亞何利亞伯即出自此支派。在聖經歷史中，但支派並非如猶大或利未般的領導支派，但在建造會幕的聖工中，神同樣揀選了但支派的亞何利亞伯作為比撒列的助手，顯示神的工作包容了不同背景的人。

- 本章重點：民1:38-39 但支派 62,700 人，人數僅次於猶大。GT 丁良才指出但子孫的後代加增甚多——但原來只有一個兒子名戶伸（創46:23；民26:42），若以色列人在埃及共住四百三十年就沒有什麼難處。GT 精讀本指出但支派後來從以色列支派中脫離出去，獨自流離、拜偶像、行惡（士18:1-31），最後從啟示錄的十二支派名單中消失（啟7:5,8），這成就了雅各對但的預言（創49:16-18）。GT 背景註釋指出但支派為北營之首，因為但是妾生眾子最長的一個。BH 指出參孫是但支派最著名的成員（士13-16）。

## 主題發展

在出埃及記中，但支派因亞何利亞伯被選召而參與了會幕的建造。這表明神的恩典大於人的宗派或背景，即使是較不顯著的支派，神也賜予技能並裝備他們參與聖工。此原則呼應了新約中肢體互相配搭的教導。
