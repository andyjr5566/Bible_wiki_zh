# 瑪迦人（Maacathites）

## 定義

居住在巴珊地區北部的族群，與基述人並列，睚珥佔領亞珥歌伯全境時，其疆界達到瑪迦人的交界（申3:14）。後來以色列未趕出他們（書13:13）。

- 與本章關聯：本章以瑪迦人交界作為睚珥得地北界的地標之一。

## 主題發展

瑪迦人作為巴珊北界的鄰族，其交界確認了以色列得地的實際範圍。
