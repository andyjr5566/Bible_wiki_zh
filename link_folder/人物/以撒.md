# 以撒

## 定義／基本資料

以撒（Isaac），亞伯拉罕與撒拉所生之子，名字意為「喜笑」。他是應許之子，豫表基督。亞伯拉罕一百歲時生子以撒，應驗了神在創世記十五和十七章的應許。

## 與目前整理書卷的關聯

### 創世記 第17章

- 神應許明年此時撒拉必生以撒
- 「以撒」意為「那喜笑的」
- 以撒是應許之子，基督的豫表
- 來源：CT, GT, KC, BH

### 創世記 第21章

- 以撒出生！亞伯拉罕年100歲，撒拉生以撒（2–5節）
- 以撒第八日受割禮，照神所吩咐（4節）
- 以撒斷奶日設擺豐盛筵席（8節）
- 神說「從以撒生的，才要稱為你的後裔」（12節）——揀選以撒為約的繼承者
- 來源：CT, GT, KC, BH
