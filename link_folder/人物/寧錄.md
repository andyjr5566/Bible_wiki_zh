# 寧錄

## 定義／基本資料

古實的兒子，世上英雄之首，英勇的獵戶。原文意為「背叛者」，被視為第一個對抗神的帝國創立人，也是大罪人的豫表。

## 與目前整理書卷的關聯

### 寧錄的六項特徵

| 特徵 | 經文 | 屬靈意義 |
|------|------|----------|
| 名字 | 8節 | 「寧錄」意**背叛**→不法的人（帖後二8） |
| 能力 | 8節 | 世上英雄之首→大有能力（帖後二9） |
| 性情 | 9節 | 獵戶→兇暴殘忍（啟十三15） |
| 態度 | 9節 | 「在耶和華面前」英勇→**目中無神**、狂傲（帖後二4） |
| 工作 | 10節 | 國起頭是**巴別**→對抗並混亂神的國度 |
| 作法 | 11-12節 | 征服、建造黑暗權勢、聯合成大勢力範圍 |

### 寧錄的國度

- **巴別**（巴力的大門、混亂）→ 巴比倫的前身
- **以力、亞甲、甲尼** → 都在[[示拿地]]（巴比倫平原）
- 這是聖經內所提的**第一個國**
- 擴展至亞述：建造[[尼尼微]]、利河伯、迦拉、利鮮

### 屬靈教訓

- 撒但常利用神所棄絕的人，使他作反抗神的首領
- 世人自古崇拜英雄，但信徒不可過於高舉才幹的領袖
- 神所看重的不是「英勇的獵戶」，而是敬畏神、與神同行的人
- 凡動刀的，必死在刀下（太廿六52）
- 建造神國度的人都是**牧者**，為羊捨命（約十11、14）

### 與敵基督的豫表

寧錄是大罪人（敵基督）的豫表：
- 帖後二8：不法的人
- 啟十三：獸的權柄
- 史家約瑟夫主張他煽動建造巴別塔
