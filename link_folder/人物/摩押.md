# 摩押

## 定義／基本資料

摩押（Moab），羅得與大女兒亂倫所生之子。名字意為「由父而生」（從父親而來）。摩押人的始祖。

## 與目前整理書卷的關聯

### 創世記 第19章

- 大女兒灌醉父親同寢後懷孕所生（37節）
- 名字＝「由父而生」，名字本身即見證亂倫的起源
- 摩押人日後成為以色列的難處（巴蘭事件、申二十三3–6）
- 豫表不顧真理原則、仿傚世俗罪惡的異端邪派
- 來源：CT, GT, KC, BH
