# 約瑟

## 定義

雅各與拉結的長子（創30:22-24），名字意為「增添」或「願神再加」。創世記37-50章的主角。本章（創33:2,7）記雅各見以掃時，將拉結與約瑟安排在隊伍最後（最受保護位置），並記載約瑟與母親一同向以掃下拜。

## 按書卷累積

### 創世記
<!-- accumulation:創世記:33:start -->
#### 第33章
- 本章重點：約瑟作為拉結獨生子（當時），與母親在隊伍最後，享受最高保護優先權；向舅舅以掃下拜行禮。
- 與本章關聯：本章確立約瑟在雅各心中特殊地位的具體表現（安排在最後），呼應創30:22-24神顧念拉結、創37:3「雅各愛約瑟勝過愛諸子」。
