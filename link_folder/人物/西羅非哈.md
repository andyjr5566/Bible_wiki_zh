# 西羅非哈

## 定義

瑪拿西族人，約瑟的玄孫、瑪吉的曾孫、基列的孫子、希弗的兒子（民27:1）。死在曠野，不與可拉同黨，因自己罪死，無兒子，留下五個女兒（瑪拉、挪阿、曷拉、密迦、得撒）。其死因與無子狀況引發女兒求產業事件，進而促成以色列產業繼承律例的頒布。

## 主題發展
