# 夏娃

## 定義／基本資料

希伯來文 *Chavah*（חַוָּה），意思是「生命」或「眾生的母親」。夏娃是神用亞當的肋骨所造的女人，是眾生之母。

## 觸發來源

- [[創世記/第2章|創世記 第2章]]：18-24 節
- [[創世記/第3章|創世記 第3章]]：1-20 節

## 聖經出現

- [[創世記/第2章|創世記 第2章]]：22 節「耶和華神就用那人身上所取的肋骨造成一個女人，領他到那人跟前」
- [[創世記/第3章|創世記 第3章]]：20 節「亞當給他妻子起名叫夏娃，因為她是眾生之母」

## 與目前整理書卷的關聯

夏娃是神為亞當造的配偶，與他同等而相配。在創世記 3 章中，她被蛇引誘吃了分別善惡樹的果子，又給亞當吃，導致人類墮落。

## 神學意義／研讀意義

- 神用肋骨造夏娃，豫表主肋旁被槍扎，流出血和水（約十九34）
- 血＝赦罪；水＝分賜生命
- 教會是以基督復活的生命為材料建造而成的（弗五26-27）
- 夏娃更改神的話（創二 vs 創三），給魔鬼有機可乘
- 夏娃的六處更改（KingComments）：
  1. 省去「隨意」
  2. 省去「任何樹」
  3. 改「樹」為「果子」
  4. 把生命樹改為分別善惡樹在園中
  5. 增添「不可摸」
  6. 把「必定死」改為「免得死」
- 夏娃墮落的四層步驟：眼睛「見」→ 手「摘下」→ 口「吃」→ 「又給」別人
- 「夏娃」意為「生命」——表明亞當對神的應許（3:15）仍具有信心
- 夏娃豫表教會
