# 尼拜約

## 定義

以實瑪利的長子，也是瑪哈拉的兄長。創世記第二十八章用「尼拜約的妹子」進一步標明瑪哈拉在以實瑪利家族中的身分。

本章沒有記載尼拜約的行動；他的名字主要承擔家譜辨識功能，確認以掃所娶之妻出自以實瑪利的直系家族。
