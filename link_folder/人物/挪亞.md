# 挪亞

## 定義／基本資料

希伯來文 *Noah*（נֹחַ），意思是「安息」或「安慰」（字音與 nuach「安息」相近）。拉麥（塞特族）的兒子。五百歲生閃、含、雅弗。洪水後全家蒙保守，成為新人類的起頭。

## 與目前整理書卷的關聯

挪亞的名字由父親拉麥命名：「這個兒子必為我們的操作和手中的勞苦安慰我們；這操作勞苦是因為耶和華咒詛地。」反映族長們在咒詛地的勞苦中對安息的渴望。

## 神學意義／研讀意義

- **豫表災後得勝被提的信徒**——在洪水中蒙神保守（對比以諾被提免於洪水）
- 以諾 = 被提在審判之前（豫表教會被提）；挪亞 = 蒙保守經過審判（豫表以色列遺民）
- 「挪亞」的名字指向基督裡的終極安息（來四9-10）
- 屬靈歷程：「挪亞」（安息）——在主裡滿得安息
