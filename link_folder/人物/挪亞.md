# 挪亞

## 定義／基本資料

舊約洪水敘事的中心人物，神選召的義人，方舟的建造者，洪水後人類和萬物的重新開始者。

## 神學意義／研讀意義

### 信心與忍耐的典範

挪亞在方舟內等待神的指示，體現**信心與忍耐**。洪水始於二月十七日，到十月初一日山頂現出，再過 73 天山頂現出，正月初一日地無積水，二月二十七日地全乾。挪亞看見地乾了，仍未擅自行動，而是等候神的話。

### 基督的預表

挪亞在經文中被視為基督的預表：
- 方舟豫表基督
- 洪水豫表受浸的水與審判
- 方舟停於亞拉臘山豫表基督復活後升天

### 新造的開始

出方舟代表**新造的開始**。挪亞一家與萬物從方舟出來，象徵從死裏復活的人一舉一動有新生的樣式（羅馬書六4）。
