# 酒政

## 定義

「酒政」是古埃及王宮中掌管酒務、負責法老飲酒安全的高階官員。創世記40-41章記載酒政與膳長同得罪法老、被關在「王的囚犯被囚之處」，約瑟為他們解夢：酒政夢見葡萄藤三枝、出花結果、榨酒獻杯，約瑟解為三天後復職。應驗後酒政復職，兩年後向法老推薦約瑟。

## 主題發展

酒政與膳長雙重見證：同囚、同夢、同求解、同應驗（第三天法老誕辰），卻生死不同命。酒政復職、膳長被掛，凸顯神主權決定生死。酒政「兩年後」才想起約瑟（創41:9-13），見證神的時機精準——若早推薦，約瑟或許未預備好；若不推薦，約瑟永不見法老。
