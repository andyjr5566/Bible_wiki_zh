# 拿鶴

## 定義／基本資料

拿鶴（Nahor），亞伯蘭的兄弟，他拉的兒子。亞伯蘭出哈蘭時，拿鶴留在哈蘭。他的子孫中出了密迦和妾流瑪所生的十二個兒子（創二十二20–24），孫女[[利百加]]後來成為以撒的妻子。
