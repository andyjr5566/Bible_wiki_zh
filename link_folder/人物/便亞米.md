# 便亞米

## 定義／基本資料

便亞米（Ben-Ammi），羅得與小女兒亂倫所生之子。名字意為「我父之子」（族人之子）。亞捫人的始祖。

### 創世記 第19章

- 小女兒灌醉父親同寢後懷孕所生（38節）
- 名字＝「我父之子」，名字本身即見證亂倫的起源
- 亞捫人日後成為以色列的難處
