# 基述人（Geshurites）

## 定義

居住在巴珊地區北部邊界的族群，睚珥佔領亞珥歌伯全境時，其疆界達到基述人的交界（申3:14）。基述人後來未被以色列趕出，仍住在以色列中（書13:13）。

## 主題發展

基述人作為巴珊北鄰族群，其邊界成為以色列得地的標誌之一。
