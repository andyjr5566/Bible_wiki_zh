# 拉班

## 定義

拿鶴之子，利百加之兄。在聖經中常被描繪為貪婪且具有操縱性的性格。

## 核心摘要

## 按書卷累積

### 創世記 第24章
- 利百加之兄，在擇偶過程中主導家族決定。
- 看見金環金鐲後熱情接待老僕人。
- 認同擇偶之事出於耶和華。
- 來源：CT, GT, KC, BH

### 創世記 第25章

- 利百加是拉班的妹子（20節）
- 來源：CT, GT, BH

### 創世記 第27章

- 雅各逃往哈蘭投奔拉班（創二十七43）
- 利百加說「我哥哥拉班」（43節）
- 來源：CT, GT, BH
