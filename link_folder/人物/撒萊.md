# 撒萊

## 定義

亞伯蘭的妻子，名字意為「我的公主」。後改名撒拉（創十七15）。創十一30特別記載她不生育，沒有孩子。

#### 第12章
- 亞伯蘭下埃及時，因撒萊容貌俊美，怕被殺而要求她稱自己為妹子
- 撒萊被帶入法老後宮
- 耶和華降大災與法老全家，因撒萊的緣故
- 法老承認撒萊為亞伯蘭妻子，打發他們離開
