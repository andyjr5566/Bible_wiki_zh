# 撒萊

## 定義／基本資料

亞伯蘭的妻子，名字意為「我的公主」。後改名撒拉（創十七15）。創十一30特別記載她不生育，沒有孩子。

## 與目前整理書卷的關聯

### 撒萊不生育

- 創十一30「撒萊不生育，沒有孩子」
- 這強調神揀選的子民是藉**神的主權和介入**而來，不是自然的結果
- 撒萊的生育問題貫穿整個亞伯蘭的故事
- 最終以撒的出生完全是神蹟（創廿一1-7）

### 屬靈意義

- 撒萊的不生育豫表**人的肉體不能產生屬靈的後裔**
- 神的應許需要神自己來成就
