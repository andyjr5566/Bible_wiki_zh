# 亞瑪力人

## 定義／基本資料

亞瑪力人（Amalekites），以掃之孫亞瑪力後裔組成的遊牧民族，住在南地一帶，是以色列人的世仇。他們在以色列人出埃及後最先攻擊以色列人（出十七8-16），神宣告必世世代代與亞瑪力人爭戰。
