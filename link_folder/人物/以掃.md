# 以掃

## 定義／基本資料

以撒與利百加的長子，孿生兄弟中的哥哥。名字意為「有毛的」（創25:25），因出生時身體發紅、渾身有毛而得名。又名「以東」（創25:30，意為「紅」）。

## 與目前整理書卷的關聯

### 創世記 第25章

- 出生時身體發紅、渾身有毛，如同皮衣（25節）
- 善於打獵，常在田野活動（27節）
- 以撒愛他，因常吃他的野味（28節）
- 從田野回來累昏，為雅各熬的紅湯出賣長子名分（29-34節）
- 說「我將要死，這長子的名分於我有甚麼益處呢？」（32節）
- 起誓出賣長子名分，吃完紅湯便走了——**輕看了他長子的名分**（34節）
- 來源：cnbible, CT, GT, KC, BH

### 創世記 第26章

- 四十歲娶赫人女子猶滴和巴實抹為妻
- 她們常使以撒和利百加心裡愁煩（極因敬拜外邦偶像）
- 表明以掃不看重神的約，輕看长子名分
- 來源：CT, GT, KC, BH

## 主題發展

以掃在聖經中代表「屬肉體」的生命（與雅各的「屬靈」對比）：
- 出生在先、憑血氣行事、野性未馴
- 喜歡在世界活動（田野）
- 看重眼見之物（紅湯），輕視屬天福分（長子名分）
- 被希伯來書作者稱為「貪戀世俗」的代表（來12:16-17）
- 豫表人天然的生命

### 創世記 第27章

- 以掃打獵為父親準備野味（3、5、30-31節）
- 被雅各用詭詐奪去祝福（35節）
- 放聲痛哭兩次（34、38節），卻得不著門路使父親心意回轉（來十二17）
- 怨恨雅各，計劃在父親死後殺他（41節）
- 以撒給以掃次等祝福：居住貧瘠之地、倚靠刀劍度日、事奉兄弟（39-40節）
- 以掃說「他名雅各，豈不是正對嗎？」——承認雅各名字意為欺騙（36節）
- 來源：CT, GT, KC, BH

### 創世記 第28章

- 以掃看出父親不喜悅迦南女子，便在原有二妻之外另娶以實瑪利的女兒瑪哈拉。
- 來源把此舉理解為外在模仿雅各的婚姻安排，並未真正掌握約的要求。
