# 瑪吉（Machir）

## 定義

瑪拿西的長子（參創50:23，民26:29），本章記載摩西將基列地給了瑪吉（申3:15），作為瑪拿西半支派在約但河東的產業基礎。

## 主題發展

瑪吉代表瑪拿西支派的核心家族，其得基列地奠定了該支派在約但河東的勢力範圍。
