# 雅各

## 定義

以撒與利百加的次子，以掃的孿生弟弟。名字意為「抓住腳跟的人」、「詭計者」、「代替者」（創25:26）。日後改名「以色列」（創32:28）。

### 創世記 第25章

- 出生時手抓住以掃的腳跟（26節）
- 為人安靜，常住在帳棚裡（27節）
- 利百加愛他（28節）
- 熬紅湯要求以掃出賣長子名分（29-34節）
- 以「你今日把長子的名分賣給我罷」предлож（31節）
- 要求以掃起誓後給了餅和紅湯（33-34節）
- 來源：cnbible, CT, GT, KC, BH

## 主題發展

雅各在聖經中代表「屬靈的生命」（與以掃的「肉體」對比）：
- 出生在後，得救不在乎人努力
- 得救出於神揀選（「大的要服事小的」）
- 喜歡與主交通（安靜）、活在教會中（帳棚）
- 羨慕屬天福分（長子名分）
- 肯出代價追求（以餅和紅湯交換）
- 雖有屬靈渴望，但趁人之危的手段不正當
- 日後經歷極大磨練，蒙福是靠神的揀選與保守

### 創世記 第27章

- 雅各在利百加的設計下，穿以掃衣服、手包山羊羔皮，冒充以掃騙取父親祝福（15-29節）
- 三次說謊：自稱是以掃（19節）、說自己打獵回來（19節）、說「耶和華你的神使我遇見好機會」（20節）
- 以撒說「聲音是雅各的聲音，手卻是以掃的手」（22節）
- 雅各與父親親嘴，以撒聞衣服香氣後祝福（26-27節）
- 雅各一去二十年，母子再未相見（CT）
- 來源：CT, GT, KC, BH

### 創世記 第28章

- 以撒正式把亞伯拉罕的福傳給雅各，打發他往巴旦亞蘭娶妻。
- 雅各在伯特利首次直接遇見神；神重申土地、後裔、萬族得福及同在保守的應許。
- 他立石澆油並許願，顯出個人信仰的開始，也顯出其信心仍帶條件。
- 來源：BH、KC、CT、GT
