# 希伯

## 定義

閃的後裔，亞法撒之孫，沙拉之子。原文意為「渡過」，指從世界分別出來。希伯來人的祖先，以色列人嫡系祖先。

### 希伯的四項特徵

| 特徵 | 說明 |
|------|------|
| 名字 | 「希伯」意**渡過**，指從世界分別出來 |
| 源頭 | 亞法撒生**沙拉**（兵器、萌芽）→ 希伯，在生命中萌芽生長 |
| 工作 | 只從事生命的蕃衍與建造 |
| 順從 | 為兒子起名**法勒**（分散），表明全心順從神分散全地的旨意 |

### 「希伯來」的由來

- 「希伯來」意為「渡過來的人」
- 亞伯蘭是第一個被稱為「希伯來人」的（創十四13）
- 他從大河那邊渡過來，與偶像世界分別
