# 拿順

## 定義

拿順（Nahshon）是亞米拿達的兒子，猶大支派的首領與族長（v12；出6:23, 民1:7）。在民數記第7章會幕膏抹後行奉獻壇之禮時，拿順代表猶大支派在「頭一日」率先前來獻上豐富供物（v12-17；CT、GT）。猶大支派領先奉獻，乃是遵照以色列四方軍營起營與東邊第一隊領頭的屬靈秩序（民2:3；CT、GT）。拿順身為猶大支派首領，在曠野天路上率先響應神的感動，代表蒙揀選的軍隊奉獻禮物與祭物；在聖經救贖史上，拿順亦是大衛王與主耶穌基督的肉身先祖（代上2:10-15, 太1:4），豫表基督作為復活初熟的果子與全人類的引導者（GT）。

### 民數記

<!-- accumulation:民數記:7:start -->
#### [[04 民數記/第7章|第7章]]
- 本章重點：本章第12-17節記載拿順身為猶大支派首領，在奉獻壇之禮的頭一日率先獻上銀盤、銀碗、金盂及各類燔祭、贖罪祭、平安祭供物。
- 與本章關聯：為以色列十二支派首領行奉獻壇之禮拉開帷幕，確立猶大支派在軍營與奉獻事奉中領頭的榜樣。
