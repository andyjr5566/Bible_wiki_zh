# 列祖（亞伯拉罕、以撒、雅各）

## 定義

指亞伯拉罕（Abraham，אַבְרָהָם）、以撒（Isaac，יִצְחָק）、雅各（Jacob，יַעֲקֹב），申命記 8:1、18 提到神向列祖起誓應許之地並堅定約。

## 按書卷累積

### 申命記

- 本章重點：第 1、18 節提到神向列祖起誓應許之地，並為堅定約。
- 與本章關聯：作為約的對象，連結至「列祖起誓應許之地」、「神賜福為堅定約」。
