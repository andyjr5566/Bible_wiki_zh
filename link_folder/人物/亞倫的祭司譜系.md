# 亞倫的祭司譜系

## 定義

出埃及記6:23-25記載亞倫的婚姻和四個兒子，確立了亞倫祭司譜系的起點——利未支派與猶大支派的聯姻，象徵祭司體系和君王體系的結合。

- 本章重點：v23亞倫娶猶大支派首領拿順妹妹以利沙巴，四子拿答、亞比戶、以利亞撒、以他瑪；v25以利亞撒生非尼哈，確立大祭司傳承線

## 主題發展

亞倫的婚姻：拿順是猶大支派首領（民二3），後在耶穌家譜中（太一4）。CT精闢指出：利未支派與猶大支派聯姻，象徵祭司體系和君王體系的結合——預表基督一身兼祭司和君王（亞六13；來七1-17）。

亞倫四子命運：拿答、亞比戶獻凡火而死（利十1-2）；以利亞撒接續亞倫為大祭司（民二十23-26）；以他瑪繼續祭司職分（出二十八1）。

大祭司傳承線：亞倫→以利亞撒→非尼哈。非尼哈因什亭熱心止息瘟疫得「非尼哈之約」——永恆祭司應許（民二十五10-13）。

基督應驗：亞倫為首任大祭司預表基督為真正大祭司（來五1-10, 七23-28）；亞倫祭司有數目更替，基督祭司職「長久不更換」（來七24）；亞倫祭司需獻自己的罪，基督無罪一次獻上（來七27）。
