# 吹角節（七月初一）

## 定義

利未記23章24節規定，七月初一以色列人要守為聖安息日，要吹角作紀念，當有聖會。在以色列宗教曆中，七月是提斯利月，這日也是猶太人民事曆的新年元旦。吹角的用意是宣告喜樂的消息，號召四散的以色列人聚集，並提醒百姓預備迎接十日的贖罪日。此節期在摩西五經中首次於此處提及，所用的角通常是公羊的角，在特別嚴肅的場合中使用。預表上，吹角節表徵神的救贖計畫將從外邦教會轉回以色列人，提醒四散的以色列人悔改並招聚他們，也象徵基督再臨時的號筒聲響。

- 本章重點：本章23至25節首度設立七月初一的吹角節，規定守為聖安息日、吹角作紀念並獻火祭。
- 與本章關聯：吹角節開啟了七月分三個重要節期的序幕，預表以色列人未來的復興與悔改歸回。
