# 雅各二十年服事

## 定義
創31:38、41 雅各兩次列舉「二十年」服事拉班細節：v.38「我在你家這二十年，你的母綿羊、母山羊沒有掉過胎……公羊我沒有吃過」；v.41「為你的兩個女兒服事你十四年，為你的羊群服事你六年，你又十次改了我的工價」。丁良才註釋探討兩處「二十年」是否指同一時段；聖經精讀本指曠野牧人職責極重，雅各盡忠職守賠償野獸撕裂損失（v.39），白日乾熱黑夜寒霜不得合眼（v.40）。

#### 第31章
- 本章重點：雅各向拉班、眾弟兄公開列舉服事細節（v.36-42），作為忠心見證與神公義審判的依據；KC 指雅各忍耐不義、順服不公（彼前2:18），最終神為他伸冤
- 與本章關聯：二十年服事是本章對質的核心證據，見證神在人無法伸冤時主動介入

## 主題發展
雅各二十年忠心服事預表基督「為我們成了僕人形像」（腓2:7），忍受不義仍倚靠父神公義；彼前2:18-23 直接呼應此榜樣，勉勵信徒效法基督腳蹤。
