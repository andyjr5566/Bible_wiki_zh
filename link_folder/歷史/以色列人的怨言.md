# 以色列人的怨言

## 定義

出埃及記第14章 v11-12 記載以色列人在絕境中的怨言：「難道在埃及沒有墳地，你把我們帶來死在曠野嗎？你為什麼這樣待我們，將我們從埃及領出來呢？我們在埃及豈沒有對你說過，不要攪擾我們，容我們服事埃及人嗎？因為服事埃及人比死在曠野還好。」這是出埃及後第一次大規模怨言——百姓忘記神在埃及所行的十大神蹟，只看眼前環境，寧願回去做奴隸也不願面對死亡。KC 指出這是羅馬書7章的對應——信徒雖已重生，仍可能因罪的權勢而陷入絕望。

## 按書卷累積

### 出埃及記
<!-- accumulation:出埃及記:14:start -->
#### 第14章
- 本章重點：v11-12 以色列人看見埃及軍兵追來就發怨言——「難道在埃及沒有墳地」「服事埃及人比死在曠野還好」。KC 指出這是羅馬書7章的對應——忘記神的奇事、只看環境、寧願回歸奴役；CT 指出這是小信之人的典型反應。
- 與本章關聯：怨言是過紅海事件中人的失敗面——與神的信實形成對比，凸顯救恩全出於神。
