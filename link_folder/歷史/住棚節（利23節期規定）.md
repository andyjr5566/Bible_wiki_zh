# 住棚節（利23節期規定）

## 定義

利未記23章34至43節規定，七月十五日是住棚節，要向耶和華守節七日，第八日有嚴肅會。此節又稱收藏節，是每年最後的節期，慶祝秋天葡萄和橄欖的收成。以色列人要住在用樹枝搭成的棚裡七日，紀念當年神領以色列人出埃及在曠野飄流時曾使他們住在棚裡。節期充滿歡樂，第一日與第八日均為聖安息日。所羅門聖殿的獻殿禮也在此節舉行。預表上，住棚節表徵享受千年國度的安息與歡樂，但「棚」的暫時性也表明國度享受並非永遠，其後將進入永世。
