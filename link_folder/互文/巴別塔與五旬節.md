# 巴別塔與五旬節

## 定義

創十一1-9巴別塔的言語混亂與徒二1-11五旬節的言語合一形成逆轉對照。五旬節聖靈降臨，逆轉了巴別塔的審判。

### 屬靈意義

- 真正的合一來自聖靈的工作，不是人的努力
- 基督的救贖使不同語言、種族的人在祂裏面合而為一（弗二11-18；啟五9）
- 未來萬國將一同事奉神（番三9-10）
