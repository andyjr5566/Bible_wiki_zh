# 耶利米書四章

## 定義／基本資料

耶利米書四章4節提到「心裡的割禮」，呼籲以色列人行心裡的割禮，將心裡的污除掉。與創世記十七章的肉身割禮相呼應，表明割禮的真正意義在於心靈的潔淨。
