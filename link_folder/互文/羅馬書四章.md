# 羅馬書四章

## 定義／基本資料

羅馬書四章是保羅引用創世記十五章6節來論證「因信稱義」的核心經文。亞伯蘭信耶和華，耶和華就以此為他的義——這是稱義的基礎。

### 創世記 第15章

- 亞伯蘭信耶和華，耶和華就以此為他的義
- 保羅以此論證：稱義是本乎恩，因著信，不在乎行為
- 來源：CT, GT

## 待確認事項

- 亞伯蘭的割禮（創十七）與稱義的時序關係

### 創世記 第17章

- 亞伯拉罕九十九歲受割禮，作為立約的記號
- 割禮是稱義的「印記」（羅四11），不是稱義的原因
- 亞伯拉罕在受割禮前已被稱義（創十五6），割禮只是確認
- 來源：CT, GT
