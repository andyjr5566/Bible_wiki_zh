# 奉獻篷車與牛（利未人搬運分工）

## 定義

在摩西立完帳幕並抹膏使成聖的當天，以色列十二支派的首領主動帶來供物——「六輛篷子車和十二隻公牛」（每二首領獻一車，每首領獻一牛），奉到帳幕前（v1-3）。耶和華指示摩西收下這些車牛，「好作會幕的使用，都要照利未人所辦的事交給他們」（v4-5）。摩西隨即進行神聖分配：(1) 給革順子孫兩輛車、四隻牛（負責搬運比較輕的幕幔、帷子、門簾；v7）；(2) 給米拉利子孫四輛車、八隻牛（在以他瑪手下，負責搬運較重且固體的板、閂、柱子、座；v8）；(3) 惟獨哥轄子孫沒有分給車與牛，因為他們辦的是聖所的事，「在肩頭上抬聖物」（v9）。此事件展現了神的聖工乃藉著屬神子民合宜的奉獻與配搭來完成，且神照著各事奉家族的實際負載與需要，精準合理地預備與分配資源（CT、GT、KC）。
