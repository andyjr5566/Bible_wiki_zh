# 大坍、亞比蘭被地吞滅（可拉黨叛逆）

## 定義

申命記 11:6 記載流便子孫以利押的兒子大坍、亞比蘭（可拉黨核心人物，民數記 16 章）因叛逆摩西、亞倫，地開口吞滅他們、家眷、帳棚與一切活物。這事件展現神對內部悖逆的嚴厲管教，屬於「大能的手」在以色列營中執行公義的實例。

- 本章重點：摩西列舉大坍、亞比蘭被地吞滅（v.6），作為百姓親眼看見神管教威嚴的見證之一。
- 與本章關聯：與埃及神蹟、紅海拯救並列，強調神不僅審判外邦敵人，也潔淨自己百姓中的叛逆，勉勵當代人謹守誡命（v.8）。
