# 守逾越節（亞筆月、無酵餅、困苦餅）

## 定義

申命記 16:1-8 規定以色列人要在亞筆月（尼散月）守逾越節，將逾越節的祭牲（牛羊）獻於耶和華所選擇立名的居所；七日內吃無酵餅，稱為「困苦餅」，紀念神在夜間急速領他們出埃及，且不可在四境內見麵酵，祭肉不可留至早晨。
