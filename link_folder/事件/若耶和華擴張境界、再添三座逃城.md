# 若耶和華擴張境界、再添三座逃城

## 定義

第 8‑10 節記載：若耶和華照向列祖起誓擴張以色列境界，將應許之地全然賜下，百姓若謹守誡命、愛神遵行其道，便要在原有三座逃城之外再添三座，免得無辜血流於地上，流血的罪歸於百姓。

## 主題發展

此事件顯示神的應許與百姓順服的關係，並將逃城制度與領土擴張結合，強調公義社會的擴展。
