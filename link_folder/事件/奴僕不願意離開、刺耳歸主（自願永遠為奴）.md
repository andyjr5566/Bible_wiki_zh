# 奴僕不願意離開、刺耳歸主（自願永遠為奴）

## 定義

申命記 15:16-17 規定若奴僕在第七年說「我不願意離開你」，因愛主人與其家且在主人那裡很好，主人便拿錐子將奴僕耳朵在門上刺透，他便永遠為奴。婢女同例。這儀式在門框（家庭中心）公開進行，象徵自願委身、聽從主人聲音（耳朵）的永久契約關係。

## 主題發展

出埃及記 21:5-6 同樣記載此儀式。詩篇 40:6「你開通我的耳朵」被七十士譯本譯為「你為我預備了身體」，希伯來書 10:5 引用應用在基督道成肉身甘心順服。保羅自稱「基督耶穌的奴僕」（羅 1:1, 腓 1:1）屬靈化此意象。古近東法律（漢摩拉比法典 §282）也有刺耳標記奴僕身分，但聖經賦予自願愛的神學意義。
