# 毀滅迦南敬拜場所（高山、山岡、青翠樹下）

## 定義

耶和華命令以色列人進入迦南地後，必須徹底毀滅所有國民事奉神的場所，包括高山、小山、青翠樹下的祭壇、柱像、木偶（亞舍拉）、雕刻的神像，並將其名從那地方除滅（申12:2-3）。此行動旨在清除土地上的屬靈污染，為建立耶和華獨一的敬拜中心鋪路。

## 主題發展

此命令緊接申命記12:1「律例典章」的引言，作為進地敬拜改革的第一步：先除舊，後立新（中央聖所）。
