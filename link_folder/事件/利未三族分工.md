# 利未三族分工

## 定義

上帝指派利未支派三大家族——革順、哥轄、米拉利——在會幕四圍安營，各司其職，共同承擔會幕的拆卸、搬運、看守與架設事工。革順族（7,500名）駐西邊，掌管帳幕、罩棚、蓋、門簾、院子帷子與繩索等織物結構；哥轄族（8,600名）駐南邊，負責約櫃、桌子、燈臺、兩座壇、聖所器皿與幔子等最聖潔器物；米拉利族（6,200名）駐北邊，負責帳幕板、閂、柱子、帶卯座、院子柱子、橛子與繩索等骨架結構。祭司以利亞撒統領全體利未人，摩西、亞倫及其子孫駐東邊把守聖所入口。此分工彰顯神秩序與聖潔，預表新約教會肢體配搭事奉。

- 本章重點：本章詳細確立哥轄、革順、米拉利三族在會幕起營搬運上的精確職責分工：哥轄負責至聖物、革順負責織物與覆蓋、米拉利負責板閂骨架，並在祭司亞倫父子的統轄下完成8,580名任職男子的數點（v1-49）。
- 與本章關聯：三族分工是神對利未支派事奉秩序的完整制度化，確保每次起營移動都能有條不紊、絕無遺漏。

## 主題發展

民數記3章記載西乃曠野首次按月齡點算利未男丁並分配職任；4章進一步規定30-50歲壯丁具體搬運細節（革順用兩車四牛、哥轄肩抬聖物、米拉利用四車八牛）。後世大衛時期再按班次輪值（代上23-26章），但三族分工核心框架延續不變。
