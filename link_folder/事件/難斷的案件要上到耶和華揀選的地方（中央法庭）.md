# 難斷的案件要上到耶和華揀選的地方（中央法庭）

## 定義

根據申命記 17:8-13，城中若起爭訟、流血、毆打等難斷案件，當上到耶和華揀選的地方，尋求祭司利未人和當時審判官的判語，並照其指教謹守遵行，不可偏左右；違抗者必治死，以除惡、使民敬畏。

## 按書卷累積

### 申命記

<!-- accumulation:申命記:17:start -->
#### [[05 申命記/第17章|第17章]]
- 本章重點：規定難斷案件須上到耶和華揀選的地方，由祭司利未人與審判官判斷，判語不可偏離，違抗者治死。
- 與本章關聯：建立中央司法權威，確保判決公義，維持社會秩序。
