# 行軍次序

## 定義

民數記第2章記載耶和華吩咐摩西、亞倫，以色列十二支派圍繞會幕四面安營，並規定起行（行軍）的次序。四面各由一支派為首領，率領三支派組成一營：東邊以猶大為首（含以薩迦、西布倫），共186,400人，作第一隊往前行（民2:3-9）；南邊以流便為首（含西緬、迦得），共151,450人，作第二隊往前行（民2:10-16）；會幕與利未營在諸營中間，安營怎樣、起行也怎樣（民2:17）；西邊以以法蓮為首（含瑪拿西、便雅憫），共108,100人，作第三隊往前行（民2:18-24）；北邊以但為首（含亞設、拿弗他利），共157,600人，作末隊往前行（民2:25-31）。總計被數點的以色列人（不含利未人）為603,550名（民2:32-33）。以色列人照耶和華吩咐摩西的安營起行（民2:34）。此編制彰顯神是秩序的神，以會幕（神的同在）為中心，各支派各按本位、本纛，前鋒、中軍、後衛井然有序。

### 民數記

#### [[04 民數記/第2章|第2章]]
- 本章重點：詳細記載十二支派四面安營與起行次序、各營首領、人數統計，並強調以會幕為中心、利未人居中、百姓順服神命。
- 與本章關聯：本章核心事件，建立以色列曠野行軍的組織架構與秩序，為後續行軍（民10章）奠定基礎。

## 主題發展

本章先記神吩咐安營次序（2-17節），再記起行次序（9,16,17,24,31節），最後總結順服執行（32-34節）。東、南、西、北四營分別對應猶大、流便、以法蓮、但四大支派為首，形成十字形陣容，會幕居中。利未人不列軍籍，專司會幕事奉，行軍時在中間抬運聖物。各營人數統計與民數記第1章一致，顯示神的命令精確一致。
