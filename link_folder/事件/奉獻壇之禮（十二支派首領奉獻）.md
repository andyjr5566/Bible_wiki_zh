# 奉獻壇之禮（十二支派首領奉獻）

## 定義

「用膏抹壇的日子，首領都來行奉獻壇的禮，眾首領就在壇前獻供物」（v10）。當會幕與祭壇用膏抹成聖之後，以色列十二支派的首領受神感動，代表各自支派前來行奉獻壇之禮（v10-88；CT、GT）。祭壇是救恩與神人關係調整的唯一關鍵（CT），首領們行奉獻壇之禮，表明全體屬神子民對神住在他們中間的熱誠感恩、效忠與敬拜（GT、KC）。每位首領獻上的供物內容完全相同，包括裝滿調油細麵的銀盤與銀碗、盛滿香的金盂，以及燔祭、贖罪祭與平安祭牲（v13-17）。這說明十二支派在敬拜神、維持聖事與支持會幕事奉上擁有同等的資格、責任與熱忱（GT、KC）。

- 本章重點：本章第10-88節詳細記載十二支派首領一連十二天輪流奉獻極其豐盛的器皿、香料與四類祭物，舉行奉獻壇之禮。
- 與本章關聯：完備摩西律法中會幕祭壇啟用時全民首領敬拜奉獻的神聖禮儀與歷史紀錄。
