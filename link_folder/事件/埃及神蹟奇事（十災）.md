# 埃及神蹟奇事（十災）

## 定義

本章第 3-4 節記載耶和華在埃及向法老和全地所行的神蹟奇事，並使紅海水淹沒追趕以色列的埃及軍兵、車馬，彰顯神的大能與拯救。
