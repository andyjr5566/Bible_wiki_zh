# 曠野生活（曠野漂流）

## 定義

指以色列人出埃及後，在進入迦南美地之前，於曠野漂流四十年的歷史時期與生活經驗。申命記第8章詳細回顧此階段：神引導百姓經過「大而可怕的曠野」（有火蛇、蠍子、乾旱無水），藉著飢餓賜嗎哪、衣服不穿破、腳不腫、磐石出水等神蹟供應，目的是「苦煉」、「試驗」百姓，認識其「心內如何」，教導「人活著不是單靠食物，乃是靠耶和華口裡所出的一切話」，並如父管教兒子般管教他們，使他們學習敬畏神、遵行神道，最終得以進入美地享福。

- 本章重點：摩西吩咐以色列人紀念曠野四十年生活：神引導、苦煉、試驗、賜嗎哪供應、衣履不敝、磐石出水、父式管教，目的是使百姓認識神話語為生命糧、學習倚靠神、敬畏遵行，並以此為鑑戒，防止進入美地飽足後忘記神、驕傲自滿而滅亡。
- 與本章關聯：本章核心經文，直接定義「曠野生活」的神學意涵（試煉、供應、管教、教導）與功能（認識神、認識自己、為進地預備、作警戒依據）。

## 主題發展

本章將曠野生活呈現為神主權安排的教練場：前段（1-6節）強調回顧過去四十年的供應與管教，目的是認識神與認識自己；中段（7-10節）對比即將進入的美地豐盛，警示飽足後的屬靈危機；後段（11-20節）再次提及曠野經歷（引導、磐石出水、賜嗎哪）作為警戒根基，防止百姓在富足時心高氣傲、忘記神、歸功己力，最終走向滅亡。曠野生活因此不僅是地理移動，更是立約群體屬靈塑造的關鍵歷程。
