# 列祖七十人下埃及、神使你如天上星那樣多

## 定義

申命記10:22回顧列祖（雅各全家）七十人下埃及，如今耶和華使以色列人如天上星那樣多。這見證神對亞伯拉罕之約（創15:5、22:17）的應驗，顯示神的信實與祝福。

## 主題發展

此節作為本章結語，呼應前文神的揀選恩典（10:15）與大能作為（10:21），以具體歷史事實證明神應許的成就，勉勵百姓敬畏、事奉神。
