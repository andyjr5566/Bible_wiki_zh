# 鹽海

## 定義／基本資料

鹽海（Salt Sea），又稱死海（Dead Sea）、亞拉巴海。位於以色列與約旦之間的內陸湖，為地球最低點。聖經中[[西訂谷]]即為鹽海所在之地。
