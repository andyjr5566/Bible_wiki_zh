# 西珥山

## 定義

西珥山（Mount Seir），位於死海以南的山區，後為以東人的领地。「西珥」意為「多毛」或「粗獷」，以掃即以東，住在西珥山（創三十六8-9）。在創世記十四章時，[[何利人]]住在西珥山。
