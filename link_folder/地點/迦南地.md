# 迦南地

## 定義／基本資料

神應許賜給亞伯蘭及其後裔的土地。位於亞細亞、歐洲和非洲三洲當中。「迦南」意為「紫紅色之地」，因盛產紫紅色染料。

## 與目前整理書卷的關聯

### 創世記第十二章

- 亞伯蘭出哈蘭時七十五歲，帶著撒萊和羅得前往迦南地
- 亞伯蘭在示劍摩利橡樹，神首次向他顯現於迦南
- 漸漸遷往南地，下埃及逃荒

### 創世記第13章

- 亞伯蘭與羅得分離後，住在迦南地（13:12）
- 羅得住在平原的城邑，漸漸挪移帳棚直到所多瑪
- 來源：CT, GT

### 地理範圍

- 東至阿拉伯曠野，西至地中海
- 南至西乃半島，北至黎巴嫩山
- 東西寬約270里，南北長約600里

### 四大地理區域

1. 沿海平原：非利士平原+沙侖平原
2. 中央高山：黎巴嫩山→以法蓮山→猶太山地→南地
3. 約旦河谷：米倫水→加利利海→死海
4. 河東山地：含亞摩河、雅博河、亞嫩河

### 戰略價值

- 在文化、商業、政治、軍事、交通上均居樞紐地位
- 世界的「肚臍」（結卅八12原文）

### 屬靈意義

- 迦南地豫表基督徒憑信心所要承受的屬靈產業
- 對以色列：應許之地，需要信心和屬靈爭戰
- 對信徒：基督裏的「天上」（弗一3）
