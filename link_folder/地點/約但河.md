# 約但河

## 定義

原文意為「約旦」。羅得選擇約但河全平原，因那地「如同耶和華的園子，也像埃及地」。

#### 第13章
- [[羅得]]舉目看見[[約但河]]全平原，直到瑣珥，都是滋潤的
- 那地在耶和華滅所多瑪、蛾摩拉以前，如同[[耶和華的園子]]，也像[[埃及]]地
- 羅得選擇約但河全平原，往東遷移——這是[[羅得的墮落]]的第一步
- 來源：CT, GT

#### 第33章
- 本章重點：雅各從雅博河北（約但河東岸）渡過約但河，來到迦南地西岸的示劍（18節）。渡河標誌從寄居地進入應許之地核心區。
- 與本章關聯：本章記載雅各生平第二次過約但河（首次創32:10「只拿杖過約但河」），此次帶著兩隊人馬、大群牲畜、完整家庭回歸。
