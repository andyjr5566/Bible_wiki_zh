# 巴珊

## 定義

以色列人轉回向巴珊去的地域。巴珊王噩率眾出戰，被以色列擊敗，以色列得了他的地（民21:33-35）。

<!-- accumulation:申命記:3:start -->
#### [[05 申命記/第3章|第3章]]
- 本章重點：本章詳述巴珊全地的攻取過程：六十座堅固城盡被奪取並毀滅（3:1-7），巴珊全地直到黑門山（3:8-10），巴珊全地（亞珥歌伯）分給瑪拿西半支派（3:13）。
