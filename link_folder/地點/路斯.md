# 路斯

## 定義

伯特利在雅各命名以前的地名。CT 列出的名字含義包括「避難所」與「杏樹」；GT 指出路斯可能原是城邑名，而伯特利起初指鄰近、與神顯現相關的地點。

創世記第二十八章以「路斯」與「伯特利」並列，保存地名改變的記憶：一個原有地方因雅各遇見神、立石澆油而獲得「神的殿」的新名稱。
