# 歌珊地

## 定義

「歌珊地」是古埃及尼羅河三角洲東部地區，約瑟安排雅各全家遷居之地（創45:10, 46:28-34, 47:1-6, 27）。該地水草豐美、適合牧羊，且因埃及人視牧羊為可憎（創46:34），希伯來人得以相對隔離、保持族群認同。

## 主題發展

歌珊地在出埃及記繼續扮演關鍵角色：出8:22神分別歌珊地以色列人不受災、出9:26歌珊地無冰雹、出10:23歌珊地有光。歌珊地成為「神保護、分別、供應」的地理符號，預表福音時代教會在世界中被分別為聖（約17:15-16）。
