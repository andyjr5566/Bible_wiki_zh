# 希實本

## 定義

亞摩利王西宏的京城。西宏曾從摩押先王手中奪取此城及全地直到亞嫩河（民21:26）。以色列人奪取此城及其鄉村居住（民21:25）。詩歌說「願西宏的城被修造，被建立」（民21:27）。有火從希實本發出燒盡摩押亞珥（民21:28）。
