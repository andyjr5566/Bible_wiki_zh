# 希伯崙

## 定義／基本資料

原文意為「同盟、聯合、交通、友誼」。位於耶路撒冷以南約32公里，原名基列亞巴。亞伯蘭與羅得分離後搬到希伯崙幔利的橡樹居住並築壇。

## 參考文獻

- 希伯崙為世界最古城市之一，主前十七世紀已建造
- 海拔約3,300呎，為交通要衝
- 後來成為亞伯拉罕家族的祖墳地（創廿三、四九、五十）
