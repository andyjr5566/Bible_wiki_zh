# 希伯崙

## 定義

原文意為「同盟、聯合、交通、友誼」。位於耶路撒冷以南約32公里，原名基列亞巴。亞伯蘭與羅得分離後搬到希伯崙幔利的橡樹居住並築壇。

#### 第35章
- 本章重點：創世記第35章中，基列亞巴就是希伯崙，亞伯拉罕和以撒曾在此寄居，雅各也來到父親以撒那裡。來源指出希伯崙在族長敘事中常與交通、寄居和家族墓地記憶相連。
- 與本章關聯：本章將希伯崙作為雅各回歸父家的終點，也標誌以撒生命的終結。
