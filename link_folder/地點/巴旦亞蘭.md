# 巴旦亞蘭

## 定義

位於米所波大米（美索不達米亞）北部的亞蘭人地區，又譯「亞蘭的荒野」。是利百加、拉班、彼土利的家鄉（創25:20）。

#### 第28章
- 本章重點：以撒差雅各前往巴旦亞蘭，到彼土利與拉班的家族中娶妻；這也是雅各離家寄居旅程的目的地。
