# 巴旦亞蘭

## 定義／基本資料

位於米所波大米（美索不達米亞）北部的亞蘭人地區，又譯「亞蘭的荒野」。是利百加、拉班、彼土利的家鄉（創25:20）。

## 觸發來源

- [[創世記/第25章|創世記 第25章]]：20節
- [[創世記/第24章|創世記 第24章]]：10節（僕人前往之处）

## 聖經出現

- [[創世記/第24章|創世記 24:10]]：僕人帶十匹駱駝往巴旦亞蘭去
- [[創世記/第25章|創世記 25:20]]：利百加是巴旦亞蘭地的亞蘭人彼土利的女兒
- [[創世記/第28章|創世記 28:2, 5-7]]：雅各往巴旦亞蘭去娶妻，以撒囑咐他不要娶迦南女子，往巴旦亞蘭去
- [[創世記/第31章|創世記 31:18]]：雅各帶著巴旦亞蘭所得的財物回迦南
