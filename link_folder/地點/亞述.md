# 亞述

## 定義／基本資料

亞述（*Assyria*，אַשּׁוּר）。底格里斯河（希底結）流在亞述的東邊。

## 觸發來源

- [[創世記/第2章|創世記 第2章]]：14 節

## 聖經出現

- [[創世記/第2章|創世記 第2章]]：14 節「第三道河名叫底格里斯，流在亞述的東邊」

- 神的恩典像箭一樣臨到敵人（賽十九23-25）
- 伯拉河（幼發拉底河）是以色列在千年國度的東北方邊界（創十五18）
