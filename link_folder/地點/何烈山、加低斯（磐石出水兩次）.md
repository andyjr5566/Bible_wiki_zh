# 何烈山、加低斯（磐石出水兩次）

## 定義

何烈山（Horeb，חֹרֵב）與加低斯（Kadesh，קָדֵשׁ）是以色列在曠野兩次從磐石得水的地點（出埃及記 17、民數記 20），申命記 8:15 回顧神曾為你使水從堅硬的磐石中流出。
