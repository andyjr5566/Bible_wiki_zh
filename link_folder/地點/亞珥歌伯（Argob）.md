# 亞珥歌伯（Argob）

## 定義

巴珊地的一個區域，即噩王國內的亞珥歌伯全境（申3:4,13），共有六十座堅固城邑（申3:4-5）。瑪拿西子孫睚珥佔領全境，並命名為哈倭特睚珥（申3:14）。本章稱此地為「利乏音人之地」（申3:13）。

## 主題發展

亞珥歌伯作為巴珊核心區，其六十座堅固城的攻取顯示耶和華大能的拯救，後成為瑪拿西半支派的核心產業。
