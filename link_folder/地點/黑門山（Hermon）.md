# 黑門山（Hermon）

## 定義

位於巴珊地北界的高山，是以色列從亞摩利王手中奪取領土的北界標誌（申3:8）。西頓人稱西連，亞摩利人稱示尼珥（申3:9）。海拔高聳，後為以色列北境地標。

- 與本章關聯：本章以黑門山為北界，確認以色列在約但河東征服的地理極限。

## 主題發展

黑門山作為北界地標，見證以色列征服巴珊地的地理範圍，其多重名稱反映當地不同族群的文化記憶。
