# 戈印

## 定義／基本資料

戈印（Goiim），意為「列國」或「外邦」。「戈印王提達」可能指多國聯盟的領袖，而非單一國家的王。確切地理位置不確定，可能指小亞細亞一帶的赫人勢力。
