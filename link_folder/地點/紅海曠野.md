# 紅海曠野

## 定義
出埃及記 第13章 18節「所以神領百姓繞道而行，走紅海曠野的路。」紅海曠野（midbar yam suf，מִדְבַּר יַם-סוּף）指紅海（蘆葦海）附近的曠野地帶，位於埃及東部邊境與西乃半島之間。神選擇繞道紅海曠野而非經非利士地沿海大道——此選擇一方面出於護理（百姓未預備打戰，v17），另一方面為後續紅海分開（第14章）和西乃山立約（第19章）鋪路。紅海曠野是出埃及旅程的第一段曠野地。

#### 第13章
- 本章重點：神領百姓不走非利士地沿海大道捷徑，而走紅海曠野的路——護理選擇，為後續紅海與西乃山事件鋪路。
- 與本章關聯：v18 直接提及，與繞道而行、以色列人帶兵器構成關聯敘事。
