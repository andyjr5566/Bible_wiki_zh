# 基列亞巴

## 類型

地點

## 觸發來源

- [[創世記/第23章|創世記 第23章]]：2節

## 目前資料

- 撒拉死在「迦南地的基列亞巴，就是希伯崙」（創廿三2）
- 基列亞巴是希伯崙的古稱，意為「四之城」或「同盟之城」（CT/BH）
- 亞伯拉罕在此與赫人進行麥比拉洞交易
