# 伊勒巴蘭

## 定義／基本資料

伊勒巴蘭（El-paran），位於曠野附近，是四王追擊何利人後返回的地點（創十四6-7）。「伊勒」意為「橡樹」或「大樹」，「巴蘭」可能意為「洞穴」或「曠野」。
