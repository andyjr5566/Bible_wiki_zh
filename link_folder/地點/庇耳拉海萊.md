# 庇耳拉海萊

## 定義／基本資料

庇耳拉海萊（Beer Lahai Roi），原文意為「那看顧我的永活者之井」。夏甲在此井旁遇見耶和華使者，稱神為「看顧人的神」。位於加低斯和巴列中間。
