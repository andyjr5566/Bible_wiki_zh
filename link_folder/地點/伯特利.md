# 伯特利

## 定義／基本資料

原文意為「神的家、神的殿、神的居所」。亞伯蘭在伯特利東邊的山支搭帳棚並築壇求告耶和華的名。

## 與目前整理書卷的關聯

### 創十二8

- 亞伯蘭從示劍遷到伯特利東邊的山
- 西邊是伯特利（神的家），東邊是艾（廢墟）
- 他在那裏支搭帳棚並築壇求告耶和華的名

### 屬靈意義

- 伯特利=神的家（教會），艾=天然生命/世界（廢墟）
- 亞伯蘭在伯特利與艾之間：**必須對付天然己生命（背向艾），才能進入基督身體的生命（面向伯特利）**
