# 埃色

## 定義／基本資料

埃色（Esek），意為「相爭」。以撒在基拉耳谷挖的第一口井，因基拉耳的牧人爭競而得名。

### 創世記 第26章

- 以撒的僕人在谷中挖井，得了一口活水井
- 基拉耳的牧人爭競：「這水是以撒的」
- 以撒給井起名「埃色」（相爭）
