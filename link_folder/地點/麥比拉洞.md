# 麥比拉洞

## 定義／基本資料

麥比拉洞（Cave of Machpelah），位於幔利前（希伯崙附近），是亞伯拉罕在迦南地購置的第一塊永久性產業——作為妻子撒拉的墳地。名字「麥比拉」意為「雙倍」（可能指雙重入口或雙重容量）。

## 與目前整理書卷的關聯

### 創世記 第23章

- 亞伯拉罕指定購買麥比拉洞，堅持按足價交易（9節）
- 以弗崙最終以400舍客勒將整塊田連洞賣出（15-16節）
- 撒拉死後埋葬於此（19節）
- 麥比拉洞成為**希伯來人祖墳**
- 亞伯拉罕在迦南唯一永久性產業竟是一塊墳地——應許地的初熟果子

## 神學或研讀意義

- **復活盼望**：埋在此处的六代族長皆存信心而死，等候與新約信徒同得更美復活（來十一13、39-40）
- **墳地而非產業**：神應許迦南全地，亞伯拉罕所得的初代應竟是一塊安葬死人的地方——指向天上更美的家鄉（來十一16）
- **今日位置**：麥比拉洞上面有亞伯拉罕清真寺（Ibrahimi Mosque）建築，至今仍在希伯崙
