# 所多瑪

## 定義／基本資料

原文意為「焚燒」。所多瑪人在耶和華面前罪大惡極。羅得漸漸挪移帳棚直到所多瑪。

## 與目前整理書卷的關聯

### 創世記 第13章

- [[羅得]]住在平原的城邑，漸漸挪移帳棚，直到[[所多瑪]]
- 所多瑪人在耶和華面前罪大惡極
- 羅得選擇靠近所多瑪是[[羅得的墮落]]的第四步
- 來源：CT, GT

### 創世記 第14章

- 所多瑪王名叫[[比拉（所多瑪王）|比拉]]，參與五王（所多瑪、蛾摩拉、押瑪、洗扁、比拉/瑣珥）聯盟
- 四王與五王在[[西訂谷]]（鹽谷）交戰，五王敗逃
- 四王奪走所多瑪和蛾摩拉一切財物，並擄走當時住在所多瑪的[[羅得]]及其財物
- 亞伯蘭得知姪兒被擄後率家丁追擊，擊敗四王，救出羅得及一切被擄財物
- 所多瑪王提出財物給亞伯蘭、把人留給自己；亞伯蘭斷然拒絕所多瑪王的一切財物，不願從外邦王得利
- 亞伯蘭只取回自己的姪兒與同人被奪之物，並分財物給盟友幔利弟兄
- 來源：CT, GT
