# 伯毘珥（Beth-peor）

## 定義

位於約但河東、毘斯迦山附近的地名（申3:29），意為「毘珥廟/殿」。以色列人在擊敗巴珊王噩後，住在伯毘珥對面的谷中，作為摩西最後講論與交棒約書亞的地理背景。

## 主題發展

伯毘珥作為摩西生命最後階段的駐紮地，見證了以色列從征服約但河東到準備進迦南的過渡時刻。
