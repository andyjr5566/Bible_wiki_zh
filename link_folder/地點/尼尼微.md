# 尼尼微

## 定義／基本資料

寧錄在亞述所建造的大城。古時極大的城，有三日的路程（拿三3），城內不能分辨左右的孩子有十二萬多（拿四11）。

## 與目前整理書卷的關聯

### 寧錄建造尼尼微

- 寧錄從[[示拿地]]出來往亞述去，建造[[尼尼微]]、利河伯、迦拉
- 和尼尼微、迦拉中間的利鮮，這就是那大城
- 有人以為後來的尼尼微大城是四座城聯合而成
