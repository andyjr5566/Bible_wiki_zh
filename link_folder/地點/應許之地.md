# 應許之地

## 定義

申命記 8:1,7-9 描述神向列祖起誓應許之地，有河、泉、源，出產七樣農產，石頭是鐵、山內可挖銅，一無所缺。

## 主題發展

應許之地是神信實應許的實現，象徵屬靈產業與永恆福分的預表。
