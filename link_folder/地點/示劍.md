# 示劍

## 定義

原文意為「肩膀」，項背（有能力負重）。亞伯蘭進迦南後第一個停留的地方，神在此首次向他顯現。

### 與目前整理書卷的關聯

### 創十二6

- 亞伯蘭到了示劍地方摩利橡樹那裏
- 「摩利」意為「教師、教訓、山寨」
- 「橡樹」意為「高大，可供遮蔭休息」
- 神在此首次向亞伯蘭顯現於迦南
- 當時迦南人住在那地——在應許之地有居民，試煉亞伯蘭的信心

### 屬靈意義

- 「到了示劍摩利橡樹」＝真實對神的認識含有生命能力，叫天路旅客重新得力（賽四十29-31）
