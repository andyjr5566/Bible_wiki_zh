# 埃及地

## 定義

埃及地是古代以色列人曾為奴之地。在《民數記》第8章17節中提及，耶和華在埃及地擊殺一切頭生的那天，將以色列人一切頭生的分別為聖歸給神。此事發生在出埃及前夕的逾越節，神擊殺埃及的長子，卻越過並拯救了以色列人的長子。因此，神要求將以色列人一切頭生的（包含人與牲畜）歸屬祂，後來神揀選利未人代替以色列人一切頭生的。
