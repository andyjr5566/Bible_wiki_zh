# 比遜

## 定義

比遜（*Pishon*，פִּישׁוֹן）意思是「自由流淌」或「滿溢」。第一道河，環繞哈腓拉全地。

## 核心摘要

比遜（*Pishon*，פִּישׁוֹן）意思是「自由流淌」或「滿溢」。第一道河，環繞哈腓拉全地。

### 與目前整理書卷的關聯

比遜是四河之一，環繞哈腓拉全地。那裡有金子、珍珠和紅瑪瑙。
