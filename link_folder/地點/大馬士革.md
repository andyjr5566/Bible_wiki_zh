# 大馬士革

## 定義／基本資料

大馬士革（Damascus），敘利亞古城，是亞伯蘭追擊四王聯盟到達北方的地點（創十四15）。亞伯蘭從幔利直追到但，再追到大馬士革左邊的何把，大獲全勝。
