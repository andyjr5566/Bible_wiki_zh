# 聖所

## 定義

聖所是帳幕內部的空間，由內幔與至聖所隔開。本章指出陳設餅桌子安在聖所的北面，金燈臺安在聖所的南面，兩者彼此相對，作為祭司日常事奉與點燈供餅之處。
