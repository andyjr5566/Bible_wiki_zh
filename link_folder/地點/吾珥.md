# 吾珥

## 定義／基本資料

迦勒底的吾珥，他拉和亞伯蘭的故鄉。當時中東最先進的文明中心，位於幼發拉底河下游。

## 與目前整理書卷的關聯

### 吾珥的背景

- 當時中東最先進的文明中心
- 他拉在吾珥事奉別神（書廿四2）
- 神在吾珥呼召亞伯蘭（徒七2-4）

### 屬靈意義

- 吾珥豫表**世界**和**偶像崇拜**
- 亞伯蘭蒙召從吾珥出來，豫表信徒蒙召從世界中分別出來
