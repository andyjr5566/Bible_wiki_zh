# 古實

## 定義

古實（*Cush*，כּוּשׁ）意思是「黑色」或「黑暗」。基訓河環繞古實全地。

## 核心摘要

古實（*Cush*，כּוּשׁ）意思是「黑色」或「黑暗」。基訓河環繞古實全地。

### 與目前整理書卷的關聯

古實是基訓河環繞之地，代表黑暗之地。聖靈的工作臨到所有人，不分種族。
