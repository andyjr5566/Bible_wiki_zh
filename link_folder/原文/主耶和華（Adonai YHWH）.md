# 主耶和華（Adonai YHWH）

## 定義

希伯來文「Adonai YHWH」（אֲדֹנָי יְהוִה），意為「主耶和華」或「主─永恆者」。摩西在申3:24禱告時稱呼神為「主耶和華啊」，承認神的大能主權，並懇求容許過約但河。此稱謂結合「主」（Adonai，主權者）與「耶和華」（YHWH，立約之神），表達僕人對主的絕對順服與信靠。

## 主題發展

此稱謂在摩西被拒進迦南的禱告中出現，凸顯摩西雖屬神親密僕人，仍須在神主權下謙卑懇求，體現「主─僕人」關係的神學張力。
