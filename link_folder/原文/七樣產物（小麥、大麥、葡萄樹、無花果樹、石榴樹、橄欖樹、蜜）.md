# 七樣產物（小麥、大麥、葡萄樹、無花果樹、石榴樹、橄欖樹、蜜）

## 定義

申命記 8:8 列舉美地七樣產物：小麥、大麥、葡萄樹、無花果樹、石榴樹、橄欖樹、蜜，象徵神賜福之地的豐富與農業基礎。

## 主題發展

這七樣產物代表應許之地的物質豐盛，也是以色列後來經濟生活的核心。
