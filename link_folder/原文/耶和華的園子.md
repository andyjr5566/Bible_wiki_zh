# 耶和華的園子

## 定義

創十三10描述約但河全平原「如同耶和華的園子，也像埃及地」。羅得用屬世眼光看這地，卻忽略其中屬靈的危險。

#### 第13章
- 羅得舉目看見約旦河全平原，直到瑣珥，都是滋潤的
- 那地在耶和華滅所多瑪、蛾摩拉以前，「如同耶和華的園子，也像埃及地」
- 羅得選擇這地是[[羅得的墮落]]的第一步
- 「耶和華的園子」對比[[伊甸]]——伊甸是神所造的園子，約但平原只是「如同」
- 來源：CT, GT
