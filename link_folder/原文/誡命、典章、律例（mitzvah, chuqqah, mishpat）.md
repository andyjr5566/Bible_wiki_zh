# 誡命、典章、律例（mitzvah, chuqqah, mishpat）

## 定義

希伯來文分別為 mitzvah（מִצְוָה，誡命）、chuqqah（חֻקָּה，典章）、mishpat（מִשְׁפָּט，律例），申命記 8:11 提到要謹守這三者。
