# 「割禮」原文意義

## 定義／基本資料

「割禮」原文為 milah（מילה），意指「切割」。在創世記十七章，割禮是神與亞伯拉罕及其後裔立約的記號/證據。所有男子出生第八日必須受割禮，作為永遠之約在肉體上的印記。
