# 賠償（shalem）

## 定義

賠償、回報，加倍賠還的概念（出22:1,3-5,7,9,12,14）。

### 利未記

<!-- accumulation:利未記:5:start -->
#### [[03 利未記/第5章|第5章]]
- 本章重點：利未記第5章14-19節引入贖愆祭（asham）制度，核心特徵為「賠償加五分之一」：人若干犯耶和華的聖物（如誤吃聖物、不獻初熟之物、不納十分之一等），須按祭司估價（按聖所舍客勒）如數償還虧欠額，另加五分之一作為罰金給祭司，再獻無殘疾公綿羊為贖愆祭，祭司為他贖罪才蒙赦免（v15-16）。若誤犯禁令（行了耶和華所吩咐不可行的事），雖不知道仍有罪，同樣須獻公綿羊為贖愆祭（v17-19）。CT 指出贖愆祭與贖罪祭的基本不同在於「犯罪的人有償還的機會」；KC 強調賠償後須以更大熱心重新奉獻自己（加五分之一預表更深的委身）；BH 說明 restitution 是補償損失或損害，加五分之一既補償合法物主損失、也作為犯罪者的刑罰。
- 與本章關聯：本章為贖愆祭的賠償原則提供首次完整立法（v14-19），確立「先賠償、後獻祭、才赦免」的次序（太5:23-24 同此原則）。加五分之一的罰金制度（利5:16；6:5；民5:7）貫穿摩西律法，預表基督不僅替人死（贖罪祭）、更償還人因罪對神與人的一切虧欠（賽53:10 稱彌賽亞為贖愆祭）。
