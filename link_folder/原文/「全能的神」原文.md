# 「全能的神」原文

## 定義／基本資料

「全能的神」原文為 El Shaddai（伊勒沙代）。Shaddai 的字根可能源於「山」或「乳房」，描繪神為站在山上、無法抵抗的全能者，或為有母親胸奶的神——全豐、全足、全有的神。
