# 海狗皮（塔哈皮）

## 定義

「海狗皮」在呂振中譯本譯為「塔哈皮」，是會幕罩棚的最外層頂蓋材料（出26:14）。這層皮革外表粗糙、無佳形美容，但質地堅韌耐苦，能抵擋曠野的風吹、日曬、雨淋，保護內層的幕幔不受侵蝕。靈意上，海狗皮表徵耶穌基督在地上做人時，並無佳形美容，卻堅固耐苦，祂的人性足能保護信徒不受撒但和世界的侵害。

#### 第26章
- 本章重點：記載用海狗皮（塔哈皮）做罩棚上的一層頂蓋，作為會幕最外層的保護。
- 與本章關聯：海狗皮堅韌卻外表無華，表徵基督在地上無佳形美容卻堅固耐苦的人性。
