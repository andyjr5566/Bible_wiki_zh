# 為奴之家（埃及）

## 定義

申命記 8:14 提到神將以色列從埃及地「為奴之家」領出來，作為救贖歷史的起點，提醒百姓不可忘記神的拯救。

## 按書卷累積

### 申命記

- 與本章關聯：作為神救贖行動的歷史基點，提醒百姓不可忘記神
