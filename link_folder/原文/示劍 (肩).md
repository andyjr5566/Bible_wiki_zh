# 示劍 (肩)

## 定義

「示劍」（Shechem）意為「肩」、「脊背」、「上背」。地名源於地處以巴路山與基利心山之間的山口，形如人肩。人名示劍（哈抹之子）與城名同源。雅各買地、築壇、住帳棚皆在此（創33:18-20）；後為約瑟葬地（書24:32）；耶穌與撒瑪利亞婦對話處（約4:5）。

#### 第33章
- 本章重點：經文記雅各來到「示劍城」（18節）、向「示劍的父親、哈抹的子孫」買地（19節），雙關地名與人名。地理上示劍為中央山區交通要衝，控制南北大道。
- 與本章關聯：名字「肩」預表此地承擔約史重擔——亞伯拉罕首築壇、雅各買地立約、約書亞立約、約瑟歸葬、耶穌傳福音。
