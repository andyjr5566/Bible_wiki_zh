# 「以撒」名字含義

## 定義／基本資料

「以撒」（Yitzchak）希伯來文意為「他笑」或「喜笑」。這名字記念亞伯拉罕聽見應許時的喜笑。神的應許決不落空，能完成人認為絕無可能之事。
