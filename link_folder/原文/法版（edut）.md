# 法版（edut）

## 定義

希伯來文 edut，意為「見證」，特指神賜給摩西的兩塊石版，上面刻有十誡，作為神與以色列人立約的見證。出25:16,21中，神吩咐將 edut 放入法櫃內，因此法櫃又稱為「見證的櫃」。
