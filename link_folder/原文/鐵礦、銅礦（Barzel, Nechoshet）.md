# 鐵礦、銅礦（Barzel, Nechoshet）

## 定義

申命記 8:9 提到「那地的石頭是鐵，山內可以挖銅」。希伯來文「ברזל」（Barzel，鐵）與「נחושת」（Nechoshet，銅），顯示迦南地具備冶金資源，預示後來以色列國的工業發展。

## 按書卷累積

- 與本章關聯：鐵銅資源為日後所羅門開採銅鐵、建殿造器提供物質基礎。
