# 角蝰（火蛇）

## 定義

申命記 8:15 記載曠野中有「火蛇」（角蝰）、蠍子、乾旱無水之地，顯示環境險惡，神卻保守百姓並供應水源。

火蛇是神在曠野管教百姓的工具，後來銅蛇預表基督被舉起拯救。
