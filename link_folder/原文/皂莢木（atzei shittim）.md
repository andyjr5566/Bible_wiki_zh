# 皂莢木（atzei shittim）

## 定義

希伯來文 atzei shittim，指產於西乃曠野的金合歡木，質地堅硬、耐久且不易朽壞。本章中皂莢木被指定為製作約櫃、陳設餅桌子、抬杠等聖所器具的主要木材，外包精金。皂莢木象徵基督堅固、不朽壞的人性，如根出於乾地，經過精金包裹顯出神性與人性的聯合。
