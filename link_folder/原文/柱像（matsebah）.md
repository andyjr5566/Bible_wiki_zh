# 柱像（matsebah）

## 定義

石柱、紀念柱，打碎他們的柱像（出23:24）

### 05 申命記

<!-- accumulation:05 申命記:7:start -->
#### [[05 申命記/第7章|第7章]]
- 本章重點：迦南地的石柱神像，代表巴力崇拜。CCB CT 解釋柱像與巴力崇拜有關、原文為石柱製神像；KingComments 指出柱像代表神明臨在或權能；BibleHub 連結出23:24、34:13。
- 與本章關聯：本章第5、25節命令打碎柱像，作為聖戰毀滅原則的一部分，拆毀祭壇、打碎柱像、砍下木偶、焚燒偶像。
