# 辦事（melakah）

## 定義

「辦事」（melakah / melakhah）在民數記4:3中專指利未人到達三十歲 mature 年齡後，正式進入會幕所擔負的聖職、管理與至聖物搬運事工（GT）。與一般性的體力勞作不同，melakah 強調具有制度化、責任重大且直接關乎神聖所運作的最高等級職務（GT難解經文）。此字在民數記4章中多次與軍事服役（zaba）並用，表明在會幕裡「辦事」本質上是一場嚴肅的屬靈爭戰與神聖事奉（CT、GT）。
