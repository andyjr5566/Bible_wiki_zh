# 非因你的義（得地非因義）

## 定義

根據申命記 9:4-6，以色列得迦南地並非因自身的義或心裡正直，乃是因迦南國民的惡與神向列祖起誓的應許。

## 主題發展

摩西提醒百姓不可自誇，強調神的恩典與公義是得地的根基。
