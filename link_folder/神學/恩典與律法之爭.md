# 恩典與律法之爭

## 定義／基本資料

恩典與律法的對比是創世記第十六章的核心神學主題。撒萊代表恩典之約，夏甲代表律法之約。保羅在加拉太書四章以此為喻：律法必須服在恩典之下。
