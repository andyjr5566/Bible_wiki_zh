# 軍營聖潔（遠避諸惡）

## 定義

申命記 23:9-14 規定出兵時軍營須保持聖潔：(1) 夢遺者須出營至日落洗澡才能回營；(2) 指定營外便所並攜帶鍬鏟土掩蓋糞便。理由是耶和華在營中行走，救護百姓、交付仇敵，營中不可有污穢，免得神轉離開。將軍事行動納入聖潔律典，強調神臨在是得勝關鍵。

## 主題發展

此段將禮儀潔淨（利 15:16-18 夢遺規定）延伸至軍事語境，並加入衛生設施條例（鍬、掩蓋），兼具神學與公共衛生雙重功能。約書亞記 7 亞干犯罪導致艾城戰敗，印證營中污穢招致神離棄。
