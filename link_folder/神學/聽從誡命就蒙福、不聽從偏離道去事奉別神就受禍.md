# 聽從誡命就蒙福、不聽從偏離道去事奉別神就受禍

## 定義

本章第 13-17、26-28 節陳明若留心聽從誡命、愛神盡心事奉，必按時降秋雨春雨、五穀新酒油豐收、飽足且牲畜有草；若心中受迷惑偏離正路事奉別神，耶和華必發怒，天閉塞不下雨、地不出產、使百姓速速滅亡。
