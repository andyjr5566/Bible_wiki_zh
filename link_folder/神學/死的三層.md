# 死的三層

## 定義

創世記 2:17 中神吩咐「你吃的日子必定死」。這裡的「死」包含三層意義。

## 主題發展

### 神學意義／研讀意義

死的三層：
1. **靈性之死**：與神隔絕（創三8-10，躲避神的面）
2. **肉身之死**：身體歸於塵土（創三19，「你本是塵土，仍要歸於塵土」）
3. **第二次的死**：永遠與神分離（啟廿一8）
