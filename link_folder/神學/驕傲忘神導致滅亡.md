# 驕傲忘神導致滅亡

## 定義

申命記 8:12-14、17-19 警告百姓若因富足「心高氣傲，忘記耶和華─你的神」，並自誇「這貨財是我力量、我能力得來的」，必致滅亡，正如列國被滅作鑑戒。

## 按書卷累積

#### [[05 申命記/第8章|第8章]]
- 本章重點：經文詳細描述驕傲忘神的過程與後果，並引列國滅亡為警戒。
- 與本章關聯：本條目為神學主題，揭示驕傲與忘神的屬靈危機。
