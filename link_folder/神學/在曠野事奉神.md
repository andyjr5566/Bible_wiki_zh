# 在曠野事奉神

## 定義

出七16「容我的百姓去，好在曠野事奉我」——出埃及的核心目的不是政治解放，而是敬拜事奉神。埃及不能提供神的同在，必須走三天路程進入曠野。

## 主題發展

KC指出神現在對祂的百姓有權利聲明——祂要他們事奉祂而不是法老。法老要相反——他要百姓事奉他而不是神。這場衝突是兩個主權的爭奪。

CT強調「事奉」與「守節、祭祀」同義（五1,3）——出埃及的目的不是享受自由而是敬拜真神。信徒蒙拯救也是為事奉神（羅十二1）。

與「在曠野向神守節」（五1）的關聯：GT指出「守節」原文是「朝聖」——離開埃及的目的就是從世界分別出來敬拜真神。KC深論曠野象徵與世界（埃及）的分離。「三天路程」象徵基督的死與復活——完全的分離。
