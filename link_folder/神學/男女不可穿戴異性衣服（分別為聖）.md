# 男女不可穿戴異性衣服（分別為聖）

## 定義

申命記 22:5 禁止婦女穿戴男子所穿戴的，男子也不可穿婦女的衣服，稱這樣行都是耶和華神所憎惡的。此條例維護神創造秩序中性別的分別（創 1:27），防止模糊性別界限導致的混淆與悖逆，體現「分別為聖」原則在日常生活細節上的應用。

## 主題發展

此禁令不僅關乎外在裝扮，更關乎身分認同與創造秩序的尊重，違背者被視為「可憎」（to'evah），與偶像崇拜、性悖逆同列（參 申 7:25-26；利 18:22）。
