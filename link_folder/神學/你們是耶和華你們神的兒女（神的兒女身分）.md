# 你們是耶和華你們神的兒女（神的兒女身分）

## 定義

申命記 14:1-2 宣告以色列人是「耶和華─你們神的兒女」，這身分建立在神的揀選與聖潔呼召之上（v.2：「耶和華從地上的萬民中揀選你特作自己的子民」）。因著這兒女身分，以色列人不可效法外邦人為死人用刀劃身、剃額（v.1），也當在飲食、十一奉獻、社會關懷等生活細節上活出聖潔、敬畏與分別為聖的樣式（v.3-29）。此身分乃恩典主權的賜予，而非人功勞所得，並成為後續律例誡命的神學基礎。

## 主題發展

本章以「你們是耶和華你們神的兒女」為開篇宣告（v.1），隨即連結「聖潔的民」與「揀選」兩大主題（v.2），作為禁止哀悼習俗（v.1）、飲食律例（v.3-21）、十一奉獻與敬畏神（v.22-26）、照顧利未人與弱勢（v.27-29）的動機根基。兒女身分意味著屬於神、被神分別、在神面前生活，並蒙神應許賜福（v.29）。
