# 信心的危機

## 定義／基本資料

創世記第十六章記載亞伯蘭面臨的信心危機：神的應許遲遲未應驗（撒萊不育、在迦南已住十年），人想用肉體力量幫助神成就應許，結果造成家庭衝突。
