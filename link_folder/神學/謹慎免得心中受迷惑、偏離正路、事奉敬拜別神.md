# 謹慎免得心中受迷惑、偏離正路、事奉敬拜別神

## 定義

申命記 11:16 警告百姓「要謹慎，免得心中受迷惑，就偏離正路，去事奉敬拜別神」。這揭示屬靈墮落的過程：心受迷惑 → 偏離正路（離棄誡命） → 事奉別神（屬靈姦淫），並直接引出神忿怒的審判（v.17：天閉塞不下雨、地不出產、速速滅亡）。

## 按書卷累積

### 申命記

<!-- accumulation:申命記:11:start -->
#### [[05 申命記/第11章|第11章]]
- 本章重點：摩西在應許順服蒙福（v.13-15）後，立即發出警告：謹慎免得心受迷惑、偏離、事奉別神（v.16），否則必遭天災地滅（v.17）。
- 與本章關聯：此警告與「愛耶和華、常守誡命」（v.1,13,22）形成正反對照，並為後續「將話存在心內」（v.18）與「教訓兒女」（v.19）提供預防性動機。
