# 全能的神（El Shaddai）

## 定義／基本資料

「全能的神」（El Shaddai）是神在創世記十七章對亞伯拉罕的自稱。原文可能意為「全有」、「全豐」、「站在山上的全能者」。舊約出現48次，在約伯記使用最頻繁（31次）。

### 創世記 第17章

- 亞伯蘭九十九歲時，耶和華向他顯現說：「我是全能的神，你當在我面前作完全人。」
- 神用此名激發亞伯拉罕的信心
- 來源：CT, GT
