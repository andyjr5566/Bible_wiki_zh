# 不可混種、混耕、混織（分別為聖原則）

## 定義

申命記 22:9-11 連續禁止三種混雜：葡萄園不可種兩樣種子（v.9），不可並用牛驢耕地（v.10），不可穿羊毛細麻攙雜料做的衣服（v.11）。這些條例延續利未記 19:19 的「分別為聖」原則，要求神的子民在農業、勞作、衣著上維持種類界限，象徵以色列作為聖潔子民不可與列國混雜（參 申 7:6；林後 6:14-17），違背者產物「充公」（v.9）或被視為悖逆創造秩序。

## 主題發展

三個禁令分別對應植物界、動物界、材質界的混雜，形成完整的「反混雜」神學體系；v.12 緊接著命令在外衣四角做繸子，以視覺提醒子民記得誡命（民 15:38-40），形成「禁止混雜—以繸子為界」的對比結構。
