# 屬靈磐石是基督

## 定義

申命記 8:15 記載神「為你使水從堅硬的磐石中流出來」；保羅在哥林多前書 10:4 指出「那磐石就是基督」，表明磐石為基督的預表，供應屬靈生命水。
