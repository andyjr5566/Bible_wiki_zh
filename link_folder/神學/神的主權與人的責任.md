# 神的主權與人的責任

## 定義

聖經可在同一事件中同時肯定人的道德責任與神掌管歷史的旨意。神使用人的行動成就祂的目的，不等於祂認可惡行，也不取消行動者的責任。

#### 第45章
- 本章重點：約瑟既說自己是兄弟們「所賣到埃及的」，也三次說神差他先來保存生命。
- 與本章關聯：「不是你們，乃是神」說明事件的最終主權，不是否認兄弟犯罪。
