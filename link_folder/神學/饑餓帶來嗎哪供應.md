# 饑餓帶來嗎哪供應

## 定義

申命記 8:3 記載「他苦煉你，任你飢餓，將你和你列祖所不認識的嗎哪賜給你吃」。神主權允許飢餓的苦境，為要在人無力自救時彰顯祂超自然的供應。嗎哪非天然產物，乃「列祖所不認識」的新事物，強調供應完全出於神的主動恩典，而非人類努力或自然規律。此神學模式預表新約中基督為「生命的糧」（約 6:31-35），在屬靈飢渴中賜給永生。

### 申命記

#### [[05 申命記/第8章|第8章]]
- 本章重點：經文描述神任憑百姓飢餓，隨後賜下嗎哪，使他們認識人活著不單靠食物，乃靠神口裡所出的話。
- 與本章關聯：本條目展示「苦煉—飢餓—供應」的神學序列，是曠野恩典經濟學的核心範例。
