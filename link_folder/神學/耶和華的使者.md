# 耶和華的使者

## 定義／基本資料

「耶和華的使者」是舊約中一個特殊的顯現，以神自己第一人稱口氣說話，接受惟有神才能賜予的祝福。一般認為這是指第二位格神，即成為肉身之前的基督。
