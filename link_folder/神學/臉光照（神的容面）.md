# 臉光照（神的容面）

## 定義

「使他的臉光照你」（v25）是民數記6:25亞倫祝福文中的核心神學意象。在希伯來思想與近東文化中，「神的臉（panim）」代表神的親密臨在、看顧與關注；而「使臉光照（panim 'or）」則為表達高度喜悅、悅納、恩寵與讚許的隱喻（詩31:16, 44:3, 80:3；CT、BH）。當神轉臉面向屬祂的子民並以光照耀時，象徵神撕破罪惡的阻隔，將祂的憐恤、救恩與慈愛無遮蔽地傾倒出來（CT、GT）。在救贖史上，這預表了世人藉著基督榮耀福音的光照得著救恩（林後4:4, 6），因為基督乃是神榮耀的光輝與神本體的真像（來1:3），使信徒能在祂的面光中蒙恩與享受生命（CT、KC）。

#### [[04 民數記/第6章|第6章]]
- 本章重點：本章第25節宣告大祭司祝福「願耶和華使他的臉光照你，賜恩給你」，展現神向屬祂子民顯明悅納與福音恩典的神聖容面。
- 與本章關聯：奠定聖經詩篇與新約書信中關於神的面光、福音光照與基督榮耀顯現的神學意象。
