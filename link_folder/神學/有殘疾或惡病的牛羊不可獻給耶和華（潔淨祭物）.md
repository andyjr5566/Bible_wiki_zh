# 有殘疾或惡病的牛羊不可獻給耶和華（潔淨祭物）

## 定義

根據申命記 17:1，凡有殘疾或惡病的牛羊都不可獻給耶和華，因為這是神所憎惡的；祭物必須完全無瑕，反映神對聖潔的要求。
