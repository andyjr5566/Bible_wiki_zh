# 安息

## 定義／基本資料

神在第七日歇了祂創造的工作而安息。原文 *shabbat*（שָׁבַת），是「安息日」的字根。

## 觸發來源

- [[創世記/第1章|創世記 第1章]]：1-31 節（六日創造的完成）
- [[創世記/第2章|創世記 第2章]]：1-3 節

## 聖經出現

- [[創世記/第2章|創世記 第2章]]：2 節「到第七日，神造物的工已經完毕，就在第七日歇了他一切的工，安息了」
- 3 節「神賜福給第七日，定為聖日」

## 與目前整理書卷的關聯

六日創造完成後，第七日神安息了。這不是因為神疲倦（賽四十28），而是工作完成。安息日的設立是神給人的模型——人尚未作工就與神同享安息。

## 神學意義／研讀意義

- 神歇下的是**創造之工**，非管理宇宙的工作（約五17）
- 安息日豫表基督的福音——惟有主耶穌十字架的工作，給人帶來真正的安息
- 人接受主耶穌救恩 = 滿意神的工作，安息在神的安息裡
- 千年國度時，安息日將恢復為全受造物的實際（來四9）
