# 苦煉

## 定義

申命記 8:2、3、16 記載神「苦煉」以色列，讓他們飢餓、賜嗎哪，使他們知道人活著不單靠食物，乃靠神口中所出的話。苦煉是神為了塑造屬靈生命的管教手段。
