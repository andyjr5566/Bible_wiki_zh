# 拜偶像者必被治死（清除惡人）

## 定義

根據申命記 17:2-7，若有人在神所賜的城中事奉別神、拜日月星宿，經查證屬實，須將該男人或女人拉到城門外，用石頭打死；需兩三見證人作見證，見證人先下手，隨後眾民執行，以除掉中間的惡。

## 按書卷累積

### 申命記

<!-- accumulation:申命記:17:start -->
#### [[05 申命記/第17章|第17章]]
- 本章重點：規定拜偶像者經查證後必被治死，需兩三見證人，見證人先下手，眾民隨後執行。
- 與本章關聯：顯示以色列社群必須清除拜偶像的惡，維護約的純潔。
