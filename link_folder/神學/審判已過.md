# 審判已過

## 定義／基本資料

創八2-5中四個「了」字（閉塞了、止住了、過了、現出來了）象徵審判已過，新局面開始。

## 觸發來源

- [[創世記/第8章|創世記 第8 章]]：2-5節

## 與目前整理書卷的關聯

### 創八2-5的四個「了」

1. 「淵源和天上的窗戶都**閉塞了**」
2. 「天下的大雨也**止住了**」
3. 「**過了**一百五十天」
4. 「山頂都**現出來了**」

### 屬靈意義

- 審判已過，信主耶穌者已得蒙赦罪（徒十43）
- 苦難並非立時消除，管教後來必結出平安的果子（來十二11）
- 山頂現出→得救盼望有了確實證據
