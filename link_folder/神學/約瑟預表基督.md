# 約瑟預表基督

## 定義

約瑟預表基督指創世記37章及後續約瑟敘事中，約瑟被父所愛、被弟兄恨惡、奉父差遣、被同族棄絕、被剝衣、被賣、受苦後高升並拯救眾人的模式，預先映照基督的受苦與榮耀。第37章主要呈現預表的受苦開端。

## 按書卷累積

### 創世記
<!-- accumulation:創世記:37:start -->
#### 第37章
- 本章重點：約瑟被父愛、被兄弟恨惡，因夢中權柄被拒絕；他奉父命尋找哥哥，卻被謀害、剝彩衣、丟坑、被賣，最後下埃及。
- 與本章關聯：來源多次把這些情節與基督被父喜悅、被自己的人拒絕、被出賣、受苦後得榮耀相連；本章是此預表最集中的起點之一。
<!-- accumulation:創世記:37:end -->

## 主題發展

此預表不取消約瑟自身歷史處境，而是在歷史敘事中看見神救贖模式：先被棄絕與降卑，後被高舉並成為拯救媒介。
