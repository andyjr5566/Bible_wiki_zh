# 嗎哪每天供應

## 定義

雖申命記 8 章未明說「每日」頻率，但結合出埃及記 16 章背景（本章 3、16 節提及「列祖所不認識的嗎哪」），神在曠野四十年（8:2、4）持續賜下嗎哪，暗示每日定時供應的規律。這教導百姓日日倚靠神，不可囤積（出 16:19-20），也預表主禱文「我們日用的飲食，今日賜給我們」（太 6:11）。每日供應彰顯神信實的慈愛（哀 3:22-23），並考驗百姓是否遵行神的指示（出 16:4）。
