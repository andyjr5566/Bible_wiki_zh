# 不可在壇旁栽樹木作木偶（禁止亞舍拉）

## 定義

耶和華禁止在為祂築壇的旁邊栽種任何樹木作為木偶（亞舍拉），因這屬於偶像崇拜，違背獨一真神的敬拜。
