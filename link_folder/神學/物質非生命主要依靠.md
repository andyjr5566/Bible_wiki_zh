# 物質非生命主要依靠

## 定義

申命記 8:3 核心宣告：「使你知道，人活著不是單靠食物，乃是靠耶和華口裡所出的一切話。」此句被主耶穌引用以抵擋試探（太 4:4、路 4:4），確立屬靈生命優於物質生存的神學秩序。曠野飢餓與嗎哪供應的歷史事件，成為這真理的教具：物質食物（嗎哪）本身無法賜予真生命，唯有神的話語（命令、應許、啟示）才是生命的真正來源與維持力。這真理貫穿聖經（約 6:63、彼前 1:23-25）。
