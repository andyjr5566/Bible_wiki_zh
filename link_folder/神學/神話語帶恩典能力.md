# 神話語帶恩典能力

## 定義

申命記 8:3 指出神藉著飢餓與嗎哪，要教導百姓「人活著……乃是靠耶和華口裡所出的一切話」。這「話」不僅指誡命律例（v.1、6、11），更涵蓋神創造性的命令、立約的應許、供應的話語（如「要有光」、「嗎哪降下」）。神的話語帶有恩典能力：它創造供應（嗎哪）、保守生命（衣服不穿破、腳不腫 v.4）、賜福應許之地（v.7-9）、並管教建立兒子身分（v.5）。這話語最終成就於基督——道成肉身者（約 1:14），祂是神終極的話語，帶來豐盛生命的恩典（約 1:16-17、來 1:1-3）。
