# 歸神

## 定義

指利未人被神揀選代替以色列一切頭生的，成為「歸神」的產業（民 3:12-13, 41, 45）。強調神對頭生的主權與利未人的聖別身分。

## 按書卷累積

- 本章重點：耶和華揀選利未人代替以色列頭生的，宣告利未人歸祂。
- 與本章關聯：本章神學核心，奠定利未人歸神的依據。
