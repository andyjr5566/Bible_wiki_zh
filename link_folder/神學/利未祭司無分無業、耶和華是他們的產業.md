# 利未祭司無分無業、耶和華是他們的產業

## 定義

申命記 18:1-2 指出利未支派（包括祭司）在以色列諸支派中不得地土產業（nachalah），他們的產業是耶和華自己。這神學宣告將祭司職任與土地經濟脫鉤，確立「神自己」作為事奉者的供應根基，並預表新約信徒以神為福分（參 彼前 2:9）。

## 主題發展

從出埃及記 19:6「祭司的國度」到此處具體化為利未支派的制度性安排；後在以西結 44:28 重申「我就是他們的產業」，新約延伸至全體信徒的屬靈產業。
