# 寄居身分

## 定義

寄居表示暫住在不屬於自己永久產業的地方。族長雖生活在不同地區，仍以寄居者自稱，因他們等候神所應許的產業。

## 按書卷累積

### 創世記

<!-- accumulation:創世記:47:start -->
#### 第47章
- 本章重點：雅各的兒子稱到埃及是寄居，雅各也稱一百三十年為在世寄居的年日。
- 與本章關聯：定居歌珊與歸葬迦南的要求共同表明埃及不是最終家鄉。
- 來源：CT、GT、KC、BH
