# 打橄欖樹枝上剩下的留給窮人

## 定義

申命記 24:20 規定打橄欖樹時，枝上剩下的不可再打，要留給寄居者、孤兒、寡婦。古代採收橄欖以棍棒搖打樹枝（賽 17:6；24:13），此命令要求農夫不可盡行採收，須為弱勢族群保留餘蘊。這與收割遺落捆（19 節）、摘葡萄剩餘（21 節）並列，構成完整的農業拾穗保障體系。

## 主題發展

此條例體現「安息年與禧年」精神在年度收成中的微觀實踐。猶太傳統（Pe'ah 5:6-7）規定橄欖樹拾穗的具體操作。路得記雖未提橄欖，但波阿斯吩咐少年人「從捆中抽出些來，留給她拾取」（得 2:15-16）展現超越律法最低要求的慷慨。
