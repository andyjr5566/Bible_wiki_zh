# 不可拿磨石作當頭

## 定義

申命記 24:6 禁止「拿人的全盤磨石或是上磨石作當頭」，因為磨石是維持生計的工具，拿去作當頭等同「拿人的命作當頭」。此法保障貧窮人基本生存資源不被剝奪。
