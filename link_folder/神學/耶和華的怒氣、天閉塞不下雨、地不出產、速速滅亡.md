# 耶和華的怒氣、天閉塞不下雨、地不出產、速速滅亡

## 定義

申命記 11:17 警告以色列人，若心中受迷惑、偏離正路去事奉別神，耶和華的怒氣必向他們發作，導致天閉塞不下雨、地不出產，使他們在耶和華所賜的美地上速速滅亡。這顯示約的咒詛層面：違背約法將失去應許之地的生存基礎（雨水與出產）。
