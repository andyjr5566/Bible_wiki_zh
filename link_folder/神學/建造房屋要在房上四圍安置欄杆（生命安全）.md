# 建造房屋要在房上四圍安置欄杆（生命安全）

## 定義

申命記 22:8 規定建造房屋必須在房頂四圍安置欄杆，免得有人從房上掉下來，流血的罪就歸於房主家。古代以色列房頂平坦供人行走、睡覺、曬曬糧食（參 書 2:6；撒下 11:2），此條例建立房主對使用者生命安全的法律責任，體現「守護生命」高於產權自由的原則，違者須負流血罪責。

## 按書卷累積

### 申命記

<!-- accumulation:申命記:22:start -->
#### [[05 申命記/第22章|第22章]]
- 本章重點：建房須在房頂四圍安欄杆，防人跌落，否則流血罪歸房主。
- 與本章關聯：本章「生命安全」段落，確立財產所有者對他人生命負預防保護責任。

## 主題發展

此條例將「不可殺人」（出 20:13）延伸為預防性的建築規範，成為後世建築安全法規的濫觴；拉比解經將「欄杆」引申為一切預防危險的防護措施（Bava Kamma 15b）。
