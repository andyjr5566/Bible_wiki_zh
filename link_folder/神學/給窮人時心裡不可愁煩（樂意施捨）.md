# 給窮人時心裡不可愁煩（樂意施捨）

## 定義

申命記 15:10 命令「你總要給他，給他的時候心裡不可愁煩」。施捨不僅是行為，更是心志：不因損失而憂愁，反因順服而喜樂。應許「耶和華─你的神必在你這一切所行的，並你手裡所辦的事上，賜福與你」。這將經濟行為提升為敬拜行為。

## 主題發展

保羅在哥林多後書 9:7「樂意施捨的人，是神所喜愛的」直接引用此原則。舊約祭司制度中「甘心樂意獻祭」（利 1:3, 22:19）同樣重心態超過儀式。拉比文獻（Mishnah Pe'ah 1:1）列「施捨」為無定量誡命之一，強調內在動機。
