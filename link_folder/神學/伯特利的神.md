# 伯特利的神

## 定義
創31:13 神在夢中自稱：「我是伯特利的神；你在那裡用油澆過柱子，向我許過願。現今你起來，離開這地，回你本地去吧！」直接連結創28:13-22 雅各在伯特利見天梯、立柱澆油、許願的經歷。神以「伯特利的神」自稱，提醒雅各當初立約，呼喚他履行許願、踏上歸回應許之地的旅程。

## 主題發展
「伯特利的神」自稱揭示神信實守約：從亞伯拉罕（創12:7）、以撒（創26:24）到雅各（創28:13）、摩西（出3:6）至大衛（代上17:23），神以「列祖之神」自稱，在基督裡成就「我是亞伯拉罕的神、以撒的神、雅各的神」（太22:32）。
