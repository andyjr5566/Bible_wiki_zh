# 無條件之約

## 定義／基本資料

創八21-22與創九10-17記載神與挪亞立的約，是聖經中五個無條件之約的第二個。神單方面應許不再用洪水滅世，不附帶任何條件。

## 觸發來源

- [[創世記/第8章|創世記 第8 章]]：21-22節

## 與目前整理書卷的關聯

### 五個無條件之約

1. 與亞當之約
2. 與挪亞之約（創八21-22；創九10-17）
3. **與亞伯蘭之約**（創十二1-3；十五1-21；十七1-8）— 參[[亞伯拉罕之約]]
4. 與大衛之約
5. 與以色列人之約

### 創八21-22的特點

- 神不再因人的緣故咒詛地
- 稼穡、寒暑、冬夏、晝夜永不停息
- 不要求挪亞或任何人履行條件
- 基於神的恩典和馨香之祭（基督的功績）

## 神學意義

「地還存留的時候」暗示這地將被新地取代（來一11；彼後三10、13；啟廿一1）。
