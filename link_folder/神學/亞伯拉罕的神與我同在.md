# 亞伯拉罕的神與我同在

## 定義
雅各在創31:42的屬靈告白：「若不是我父親以撒所敬畏的神，就是亞伯拉罕的神與我同在」。強調神與亞伯拉罕、以撒立約的同在，延續至雅各身上，成為雅各在拉班欺壓下仍得保護、財產增長的根本原因。

#### 第31章
- 本章重點：v.5「我父親的神向來與我同在」、v.42「亞伯拉罕的神與我同在」、v.3 神應許「我必與你同在」
- 與本章關聯：作為雅各敢於對抗拉班、帶領全家逃離的屬靈底氣；神的同在不僅保護也主動轉移產業（v.9「神把你們父親的牲畜奪來賜給我了」）
<!-- accumulation:創世記:31:end -->

## 主題發展
神與亞伯拉罕、以撒、雅各三代同在的連續性，構成族長史神學骨幹；出3:6「我是亞伯拉罕的神、以撒的神、雅各的神」成爲神自我啟示的標準公式，最終在基督裡成就「以馬內利——神與我們同在」（太1:23）。
