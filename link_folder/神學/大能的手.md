# 大能的手

## 定義
出埃及記 第13章 四次重複「耶和華用大能的手將你們從埃及領出來」（v3, v9, v14, v16），希伯來原文 yad chazaqah（יָד חֲזָקָה）是出埃及記的核心描述詞組，描繪神救贖行動的權能彰顯。此詞組在出埃及記中反覆出現（6:1「大能的手」逼法老容以色列人去、32:11「大能的手」大施權能），與「伸出來的膀臂」（zeroa netuyah）並用。本章 v3,9,14,16 四次重複形成循環結構，強調出埃及不是以色列人的自救，而是耶和華主權權能的救贖行動。
