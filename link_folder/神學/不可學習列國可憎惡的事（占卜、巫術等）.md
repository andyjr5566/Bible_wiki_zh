# 不可學習列國可憎惡的事（占卜、巫術等）

## 定義

申命記 18:9-14 嚴禁以色列人進入迦南後學習列國可憎惡的事，列舉九種禁術：使兒女經火、占卜、觀兆、法術、邪術、迷術、交鬼、巫術、過陰。經文指出這些事為耶和華所憎惡，正是因列國行這些事被趕出。以色列要在耶和華面前作「完全人」（tamim），單單倚靠神啟示，不求靈媒巫術。

## 主題發展

利未記 19:31、20:6 同樣禁令；掃羅求交鬼婦（撒上 28）違此命獲判；新約加拉太書 5:20 將巫術列為肉體之事。
