# 摘葡萄園剩下的留給窮人

## 定義

申命記 24:21 規定摘葡萄園的葡萄，所剩下的不可再摘，要留給寄居者、孤兒、寡婦。這是拾穗律法的第三項，針對葡萄採收（通常用手摘或小刀剪），要求農夫不可窮盡採摘，須為無產階級保留餘蘊。三項拾穗條例（禾捆、橄欖、葡萄）涵蓋以色列主要農作物，構建全方位的社會安全網。

## 主題發展

利未記 19:10；23:22 平行規定「不可將葡萄園的葡萄盡行摘取，也不可拾取葡萄園掉落的葡萄」。以賽亞書 17:6；24:13 用「搖橄欖樹、摘葡萄剩下的」比喻餘民得保留。新約馬太福音 20:1-16 葡萄園工人比喻雖非直接引用，但反映同一農業社會背景下的公義與恩典張力。
