# 不守誡命典章律例

## 定義

申命記8:11「免得忘記耶和華─你的神，不守他的誡命、典章、律例」。此為「忘記神」在行為層面的必然後果：誡命（米茲瓦，道德律）、典章（米什帕特，審判/公義律）、律例（胡卡，禮儀/律典）三重律法體系全盤廢棄。揭示遺忘神不僅是認知偏差，更導致倫理崩壞與約關係破裂。

## 主題發展

申命記將三詞並列（4:45, 5:31, 6:1, 7:11, 8:11, 11:1），強調律法整體性；歷代先知責備以色列「廢棄耶和華的律法」（王下17:15）；新約基督「沒有來廢掉，乃是來成全」（太5:17），信徒靠聖靈「律法的義就在我們身上成全了」（羅8:4）。
