# 嗎哪象徵基督

## 定義

申命記 8:3、16 提到神在曠野賜嗎哪，使百姓「知道人活著不是單靠食物，乃是靠耶和華口裡所出的一切話」；新約將嗎哪視為基督的預表（約 6:31-35），象徵屬靈生命的糧食。
