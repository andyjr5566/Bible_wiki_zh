# 兒子地位帶來安息

## 定義

申命記 8:5 確立以色列為神兒子（「耶和華─你神管教你，好像人管教兒子一樣」），這身分是曠野生存與進入迦南安息的根基。兒子地位意味著：① 父神負責供應（嗎哪、水、衣服不壞 v.3-4, 15）；② 父神管教塑造（v.5, 16）；③ 父神賜福產業（美地、七樣產物、鐵銅礦 v.7-9）；④ 父神立約堅定（v.18）。安息不在於環境舒適，而在於確知「我是神兒子，父必看顧」。這預表信徒在基督裡得兒子名分（加 4:4-7、羅 8:15-17），在今生曠野中享有屬靈安息（來 4:1-11），最終進入永恆安息。
