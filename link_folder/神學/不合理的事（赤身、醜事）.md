# 不合理的事（赤身、醜事）

## 定義

申命記 24:1 提到丈夫見妻子有「不合理的事」（希伯來文 ervat davar，意指「赤身的事」或「醜陋的事」），便可寫休書離婚。此詞語在猶太傳統中引發對離婚理由的討論，後來拉比學派對其解釋不一（如沙买學派寬鬆，希列學派嚴格）。
