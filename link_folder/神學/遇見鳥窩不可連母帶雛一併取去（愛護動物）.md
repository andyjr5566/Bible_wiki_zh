# 遇見鳥窩不可連母帶雛一併取去（愛護動物）

## 定義

申命記 22:6-7 規定在路上遇見鳥窩，母鳥伏在雛或蛋上時，不可連母帶雛一併取去，必須放母，只可取雛，應許「可以享福，日子得以長久」。此條例體現對受造秩序的敬畏，保護母鳥的繁衍本能，彰顯神對生命延續的眷顧，並將仁慈對待動物與個人福祉掛鉤。

## 主題發展

此律法被猶太傳統列為「輕微誡命」的典型（參 太 23:23），卻附帶長壽應許，顯示神看重內心憐憫超過儀式外表；拉比文獻將其視為培養憐憫品格的訓練（Mishnah Chullin 12:5）。
