# 獻祭只能在神揀選的地方（集中敬拜）

## 定義

申命記12章嚴禁在各處隨意獻燔祭（v.13），規定一切獻祭——燔祭、平安祭、十分取一之物、舉祭、還願祭、甘心祭、牛群羊群頭生的——必須奉到耶和華揀選的地方（v.5-7, 11, 14, 17-18, 26-27）。平安祭的血要倒在壇上，肉可在神面前吃；燔祭連肉帶血全獻於壇上。此法確保敬拜純正、防止混合主義、維持國家宗教統一。

## 按書卷累積

### 申命記

<!-- accumulation:申命記:12:start -->
#### [[05 申命記/第12章|第12章]]
- 本章重點：禁止在各處隨意獻燔祭，規定一切獻祭必須帶到耶和華揀選的地方，並詳細列舉祭物類別與處理方式。
- 與本章關聯：核心神學命令，貫串本章v.5-28，與「在各城可宰殺吃肉」形成祭物與非祭物的界線。

## 主題發展

歷史上希西家、約西亞改革即本於此法（王下18, 23）；新約在基督裡實現屬靈集中敬拜（約4:21-24, 來13:10-15）。
