# 山上的樣式

## 定義

神在出埃及記25章兩次強調（25:9, 25:40），摩西製造帳幕和其中一切器具，都必須「照著在山上指示你的樣式」。這表明會幕的設計不是出於人的構想，而是源於神在西乃山上向摩西啟示的屬天藍圖。地上的會幕是「天上事的形狀和影像」（來8:5），因此每一細節都具有神學意義，必須嚴格遵照神的指示，不可擅自更改。這原則教導：神的事工必須按照神的啟示和方法執行，順服是執行神計畫的關鍵。
