# 山上的樣式

## 定義

神吩咐摩西建造帳幕與一切器具時，必須嚴格照著在山上指示他的樣式。這表明屬地的敬拜與事奉必須完全根據屬天的啟示，而非人的意思或創意。會幕的樣式是天上真帳幕的影兒，預表基督的位格與工作。

## 主題發展

此原則在希伯來書中被進一步解明，指出地上的帳幕是天上事的形狀和影像。這教導我們，教會的建造與屬靈的事奉也必須完全順服神話語的啟示與聖靈的引導，不可擅自更改神的定規。
