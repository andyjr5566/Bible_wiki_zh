# 衣服沒穿破、腳沒腫為神保守證據

## 定義

申命記 8:4 記載「這四十年，你的衣服沒有穿破，你的腳也沒有腫」，顯示神在曠野中超自然保守百姓身體與生活需要，作為神信實保守的見證。

## 按書卷累積
