# 逃奴不可交還

## 定義

申命記 23:15-16 規定逃到以色列的奴僕不可交還主人，准許其在以色列城邑中自由選擇居住地，且不可欺負。這在古代近東法典（如漢摩拉比法典 §15-20）中極為獨特，體現出埃及救贖經驗對弱勢者的倫理投射：曾為奴的以色列當憐恤逃奴。

## 按書卷累積

### 申命記

- 與本章關聯：承接軍營聖潔後，轉入社會弱勢保護條例，展現聖潔的社會倫理維度。

## 主題發展

保羅在腓利門書處理阿尼西母逃奴事件時，雖未直接引用此條，但「不再作奴僕，乃是作親愛的弟兄」（門 1:16）呼應此精神。猶太傳統將此限定為逃入迦南地的外邦奴僕（Mishnah Gittin 4:5）。
