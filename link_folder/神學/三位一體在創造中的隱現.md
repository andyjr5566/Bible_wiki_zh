# 三位一體在創造中的隱現

## 定義／基本資料

創世記第1章已隱含三位一體的真理：神（父）、神的靈、神說／神的話（子）。

## 觸發來源

- [[創世記/第1章|創世記 第1章]]：1-3、26-27節

## 聖經出現

- [[創世記/第1章|創世記 第1章]]：1節「神」原文複數、2節「神的靈」、3節「神說」
- 26節「我們要照著我們的形像」— 三一神格會議

## 與目前整理書卷的關聯

1:1 的「神」原文為複數（*Elohim*），1:26 的「我們」為複數，暗示三位一體參與創造。子神是創造的媒介（約一3；西一16；來一1-2），聖靈運行其上。

## 神學意義／研讀意義

- 創造是三一神共同的工作
- 子神是創造的「媒介」或「工具」（約一3；西一16）
- 聖靈運行在水面上，預備新生命的誕生
- 26節的「神格會議」態度極其慎重，顯示造人是創造的高峰
