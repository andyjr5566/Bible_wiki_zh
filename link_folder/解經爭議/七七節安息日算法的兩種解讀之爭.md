# 七七節安息日算法的兩種解讀之爭

## 定義

利未記23章11節與15節提及「安息日的次日」獻禾捆為搖祭，並從該日算起滿七個安息日。對於此處「安息日」的解讀，歷史上存在兩種主要爭議：撒都該人主張這是指無酵節期間的尋常安息日（一週的第七日），因此五旬節的日期每年會落在不同的星期幾；而法利賽人及七十士譯本則主張此「安息日」是指無酵節的第一個聖會日（正月十五日），因此「安息日的次日」固定為正月十六日，五旬節也固定在特定日期。猶太人至今仍按法利賽人的解法，從正月十六日算計四十九天到七七節。
