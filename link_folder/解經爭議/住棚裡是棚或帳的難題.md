# 住棚裡是棚或帳的難題

## 定義

利未記23章42至43節指示以色列人要住在「棚」裡七日，紀念出埃及時曾使他們住在棚裡。然而出埃及記16章16節卻說百姓在曠野是住在「帳」裡，且聖經其他地方論及曠野住處皆用「帳幕」而非「棚」字，這引發了解經難題。解答在於利未記此處指的並非百姓曠野四十年長期的居所，而是針對在耶路撒冷守住棚節時的指示。因為為期只有一週，他們被指示要住在暫時搭起的「棚」裡。帳幕和棚都是人暫時居住的所在，用住棚節的棚作為曠野帳幕的標記是適當的，旨在教導百姓紀念神在曠野的保護與引導。

#### [[03 利未記/第23章|第23章]]
- 本章重點：本章42至43節要求百姓住棚七日，與出埃及記中百姓住帳篷的記載產生用語與歷史情境的探討。
- 與本章關聯：透過暫時性「棚」的居住體驗，預表信徒在地上是客旅，盼望那永存的帳幕。
