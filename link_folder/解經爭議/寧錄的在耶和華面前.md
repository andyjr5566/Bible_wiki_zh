# 寧錄的在耶和華面前

## 定義

創十9「他在耶和華面前是個英勇的獵戶」中的「在耶和華面前」是褒義還是貶義？

## 按書卷累積

### 觸發來源

- [[創世記/第10章|創世記 第10 章]]：9節

## 主題發展

### 貶義解釋（主流）

- 「在耶和華面前」= **目中無神**、狂傲
- 他的英勇是在神面前的**挑釁和對抗**
- 這與他作為「背叛者」（名字含義）的形象一致
- 帖後二4：抵擋主、高抬自己超過一切稱為神的

### 褒義解釋（少數）

- 「在耶和華面前」= 他的英勇連神都注目
- 俗語說「像寧錄在耶和華面前是個英勇的獵戶」是正面描述
- 但這與上下文（巴別的建立、對抗神的國度）不一致

### 結論

主流解經認為「在耶和華面前」是貶義，表示寧錄**公然對抗神**。
