# 贖罪日禁食起始日的難題（第九日或第十日）

## 定義

利未記23章32節記載：「從這月初九日晚上到次日晚上，要守為安息日。」這引發了贖罪日禁食究竟是在第九日還是第十日開始的解經難題。因為27節明確指出「七月初十是贖罪日」，而32節卻說從初九日晚上開始守。解答在於猶太人的曆法計算方式是從日落到日落算為一天。因此，初九日的日落實際上已是初十日的起頭。禁食開始在第九日傍晚而延續到第十日傍晚，這使得在任何兩天的交接處開始守這聖日都是合宜的，確保了初十日全日的完全禁食與安息。
