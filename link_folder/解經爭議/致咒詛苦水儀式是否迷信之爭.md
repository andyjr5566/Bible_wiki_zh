# 致咒詛苦水儀式是否迷信之爭

## 定義

民數記5:11-31記錄婦人喝致咒詛苦水以試驗貞潔的條例，引發此儀式「是否為無科學根據之迷信或巫術」的解經與神學爭議（GT難題篇）。主要存在兩種權威解經立場：(1) 神聖干涉與仁愛法規立場（李道生《舊約聖經問題總解》）：認為此法絕非迷信行為。古代野蠻民族的神明裁判（如手拿熱鐵、手入滾油、喝毒水）乃惡毒苦刑；而神所吩咐的水（聖水加塵土）本無毒無害，不傷身體。此儀式是神在舊約時代所用的超自然方法，將夫婦疑難交託神來表白，既救了許多無辜婦女免受虐待與冤屈，又免除許多不必要的離婚與疑恨，在當時實為一種仁愛而合理的制度；(2) 屬靈與心理學/身心反應立場（賈斯樂郝威《聖經難解經文詮釋手冊》）：指出苦水本身無化學毒性或生理定罪機制，腹脹消瘦並非化學反應，而是身心反應（psychosomatic）或屬靈心理作用。婦人在神面前發嚴肅誓言，若確有行淫罪惡感，心理壓力與恐懼會引發假性懷孕（false pregnancy）或腹脹等生理症狀（類似醫學安慰劑效應）；若無辜清白則心安理得、毫無症狀。苦水扮演「屬靈測謊器」功能，顯示神的審判貫穿人人心思意念（BH）。兩者皆強調：使儀式有效的不是苦水本身的魔術力量，而是全知全能的神（GT《串珠》）。新約後聖經未再提起此法（GT《啟導本》）。
