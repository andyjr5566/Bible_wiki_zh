# 耶和華的節期年曆總覽（利未記23章七大節期架構）

## 定義

利未記23章詳列了耶和華神親自設立並吩咐以色列人遵守的七大節期，分別為逾越節、無酵節、初熟節、七七節（五旬節）、吹角節、贖罪日和住棚節。這些節期依宗教曆計算，均在頭七個月內舉行，標明了農曆年的各個階段並慶賀收成。節期中「七」是最要緊的數目，每逢第七日為安息日，每年七月為聖月。這七大節期不僅是以色列人紀念神救贖與祝福的時刻，在救贖史的意義上，也預表了神歷代所施行的救恩，從基督作神羔羊被殺（逾越節）、信徒過無罪生活（無酵節）、基督復活（初熟節）、聖靈降臨設立教會（五旬節），直到以色列人悔改歸主（吹角節與贖罪日）以及進入國度安息（住棚節）。
