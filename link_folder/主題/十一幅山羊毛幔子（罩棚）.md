# 十一幅山羊毛幔子（罩棚）

## 定義

帳幕的第一層覆蓋物，由十一幅山羊毛織成的幔子組成，分為五幅與六幅相連，以銅鉤相連成一片，作為帳幕的罩棚。

## 按書卷累積

### 出埃及記

<!-- accumulation:出埃及記:26:start -->
#### 第26章
- 本章重點：用山羊毛織十一幅幔子作為帳幕以上的罩棚，以五十個銅鉤相連。
- 與本章關聯：罩棚垂下的餘幅遮蓋帳幕後頭與兩旁，形成保護層。
