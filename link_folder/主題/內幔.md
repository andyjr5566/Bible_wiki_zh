# 內幔

## 定義

內幔是指隔開聖所與至聖所的幔子。出埃及記26章31節記載，這幔子要用藍色、紫色、朱紅色線和撚的細麻織成，並以巧匠的手工繡上基路伯。它掛在四根包金的皂莢木柱子上，柱子安在四個帶卯的銀座上（出26:32）。其功用是將聖所和至聖所隔開，法櫃和施恩座被抬進幔子內的至聖所（出26:33-34）。內幔表徵基督的身體（來10:20），當耶穌在十字架上斷氣時，殿裡的幔子從上到下裂為兩半，表明祂為我們開了一條又新又活的路，使信徒能因祂的救贖直接來到神面前。

#### 第26章
- 本章重點：詳細記載內幔的材料、繡上基路伯的工法，以及掛在四根包金皂莢木柱子上的規格。
- 與本章關聯：內幔隔開聖所與至聖所，表徵基督的身體，預表藉著基督的救贖開啟通往神面前的道路。
