# 立約

## 定義／基本資料

在創世記中，立約是神與人、人與人之間建立關係的正式行動。以撒與亞比米勒在別是巴立約互不侵犯。

## 觸發來源

- 多處創世記經文

## 與目前整理書卷的關聯

### 創世記 第26章

- 亞比米勒承認「耶和華與你同在」，要求立約互不侵犯
- 以撒設擺筵席，清早彼此起誓
- 起名示巴（起誓），城名別是巴（盟誓的井）
- 來源：CT, GT, KC, BH
