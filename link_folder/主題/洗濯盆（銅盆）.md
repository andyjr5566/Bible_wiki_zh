# 洗濯盆（銅盆）

## 定義

用銅製作的盆和盆座，放置於會幕和祭壇的中間，盆內盛水。亞倫和他的兒子（祭司）進會幕或就近壇前供職獻火祭時，必須在此洗手洗腳，免得死亡。此為祭司世世代代永遠的定例。銅表徵審判，洗濯盆表徵藉著神的話中之水洗淨生活行為中的污穢，使事奉者保持聖潔以親近神。
