# 不可與經期婦人親近

## 定義

禁止在女人行經不潔淨的時候露她的下體與她親近；行經期在禮儀上被視為不潔淨（參利十五19），此禁令不僅關乎衛生，也因宗教儀式上的不潔，違者必從民中剪除（利二十18）。

## 按書卷累積

### 利未記

<!-- accumulation:利未記:18:start -->
#### [[03 利未記/第18章|第18章]]
- 本章重點：第19節禁止在女人行經不潔淨的時候與她發生性關係，將禮儀不潔與道德禁令結合；犯者將從民中剪除（參二十18），強調婚姻內性生活也須尊重神所定的潔淨條例。
- 與本章關聯：本條目為本章性道德禁令的一部分，將禮儀潔淨律與道德律連結，表明即使在婚姻關係內，也須按神的聖潔標準行事。
