# 陳設餅桌子

## 定義

陳設餅桌子是用皂莢木製作、包上精金的一張桌子，長二肘、寬一肘、高一肘半，四圍鑲金牙邊，並有一掌寬的橫梁與金牙邊。桌子的四角安有金環，可穿皂莢木包金的杠以便抬運。桌上常擺陳設餅，共十二個，代表以色列十二支派，在神面前作為立約子民蒙悅納的記號。桌子與其上的盤子、調羹、爵、瓶等器具皆用精金製作，表明基督是生命的糧，供應神子民並使他們在神面前得享交通。
