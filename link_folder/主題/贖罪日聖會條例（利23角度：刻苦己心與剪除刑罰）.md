# 贖罪日聖會條例（利23角度：刻苦己心與剪除刑罰）

## 定義

利未記23章27至32節從百姓任務的角度記載了七月初十贖罪日的條例。百姓當日要守為聖會，並「刻苦己心」（通常指禁食、懺悔及禱告，表達為罪憂傷痛悔），同時獻火祭給耶和華，且什麼工都不可做。神設立了嚴厲的刑罰：凡不刻苦己心的，必從民中剪除；凡這日做什麼工的，神必將他從民中除滅。這條例強調人必須放下自己的行為與努力，單單接受神的救恩。預表上，贖罪日表徵以色列全家未來的悔改與得救，唯有經過憂傷痛悔與認罪，才能進入後續住棚節的歡樂。
