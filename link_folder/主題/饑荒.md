# 饑荒

## 定義／基本資料

在創世記中，飑荒是神用來訓練祂僕人的工具和信心的試題。亞伯拉罕和以撒都曾經歷飢荒。

## 觸發來源

- [[創世記/第12章|創世記 第12章]]：10節（亞伯拉罕時期）
- [[創世記/第26章|創世記 第26章]]：1節（以撒時期）

## 與目前整理書卷的關聯

### 創世記 第12章

- 亞伯蘭因飑荒下埃及暫居
- 來源：CT, GT, KC

### 創世記 第26章

- 「在亞伯拉罕的日子，那地有一次饑荒；這時又有饑荒」——再次發生（兩次時隔約80餘年）
- 以撒遷至基拉耳
- 神安排飑荒的目的：試驗信心、堅固信心、顯明神是可靠的
- 來源：CT, GT, KC, BH
