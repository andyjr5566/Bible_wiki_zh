# 銅

## 定義

銅在本章中主要用於製作會幕外層結構的連接件與支撐座。具體應用包括： 製作五十個銅鉤，鉤在鈕扣中使山羊毛罩棚連成一個；以及為帳幕門簾的五根柱子， 用銅鑄造五個帶卯的座，作為門簾柱子的根基。
