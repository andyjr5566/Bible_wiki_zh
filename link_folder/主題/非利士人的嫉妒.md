# 非利士人的嫉妒

## 定義／基本資料

以撒在基拉耳昌盛後，非利士人因嫉妒而塞住水井、要求以撒離開。這是神祝福引來世界敵意的典型模式。

## 觸發來源

- [[創世記/第26章|創世記 第26章]]：14-16節

## 與目前整理書卷的關聯

### 創世記 第26章

- 以撒有羊群牛群，又有許多僕人，非利士人就嫉妒他
- 塞住亞伯拉罕時期所挖的水井（切斷生存倚賴）
- 亞比米勒要求以撒離開：「你比我們強盛得多」
- 來源：CT, GT, KC, BH
