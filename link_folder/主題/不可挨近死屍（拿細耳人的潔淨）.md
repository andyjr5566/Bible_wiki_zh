# 不可挨近死屍（拿細耳人的潔淨）

## 定義

拿細耳人三大禁令之三為「不可挨近死屍。他的父母或是弟兄姊妹死了的時候，他不可因他們使自己不潔淨」（v6-7）。在舊約律法中，死亡乃罪的結果與工價（羅6:23），生命的主神極其厭惡死亡玷污。普通祭司在父母或至親死亡時仍許觸摸死屍居喪（利21:1-3），惟獨大祭司與拿細耳人被嚴格禁止（利21:11, 民6:7；GT, KC, BH）。這說明拿細耳人所蒙的要求比一般祭司更高——在神的聖潔與權柄面前，絕不可因人間至親的情感或天然關係而違背對神的絕對忠誠，神必須在人心裏居首位（CT、KC；太10:37）。若無意中沾染死屍，即算沾染了離俗的頭，必須經歷嚴肅的潔淨與獻祭程序（v9-12）。

#### [[04 民數記/第6章|第6章]]
- 本章重點：本章第6-7節嚴禁拿細耳人在離俗期間觸摸任何死屍，甚至至親父母手足死亡亦不可因親情玷污離俗的頭。
- 與本章關聯：展現舊約聖潔律法對絕不沾染死亡、至高神權勝於親情人情的最高規範。
