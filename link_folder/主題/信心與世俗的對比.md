# 信心與世俗的對比

## 定義／基本資料

創十三記載亞伯蘭與羅得的對比：亞伯蘭代表信心、謙讓、順服神帶領；羅得代表屬世眼光、自私選擇、逐步妥協。

## 與目前整理書卷的關聯

### 創世記 第13章

| 亞伯蘭 | 羅得 |
|--------|------|
| 讓羅得先選，信靠神 | 隨自己眼光選擇 |
| 放棄選擇權 | 選擇眼前利益 |
| 獲得神重申應許 | 與罪惡之城為鄰 |
| 住在迦南地（應许之地） | 住在所多玛（罪恶之地） |
| 築壇敬拜神 | 漸漸挪移直到罪惡之源 |

- 亞伯蘭的謙讓不是軟弱，而是信心的表現
- 羅得的選擇四重錯誤：年長者是跟隨者、產業來自亞伯蘭、地是應許給亞伯蘭的
- 來源：CT, GT, KC, BH
