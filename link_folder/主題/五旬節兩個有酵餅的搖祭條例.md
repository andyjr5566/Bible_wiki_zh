# 五旬節兩個有酵餅的搖祭條例

## 定義

利未記23章17節規定，在五旬節（七七節）當天，以色列人要從住處取出細麵伊法十分之二，加酵烤成兩個餅作為搖祭獻給耶和華。這是摩西律法中極為特殊的條例，因為通常獻給神的素祭不可有酵（酵表徵罪惡）。此處加酵的餅預表新約時代的教會，教會由猶太人與外邦人（兩個餅）組成，雖然在人性裡仍有罪的成分（酵），但藉著基督贖罪的火祭（同獻的贖罪祭），教會仍可在復活的基督裡向神奉獻為神而活。這兩個餅只被搖在耶和華面前，而不燒在壇上。

#### [[03 利未記/第23章|第23章]]
- 本章重點：本章17至20節詳述五旬節獻上兩個加酵細麵餅作搖祭的條例，並需同獻贖罪祭與平安祭。
- 與本章關聯：此條例預表聖靈降臨後教會的建立，彰顯蒙恩罪人在基督裡成為初熟之物獻給神。
