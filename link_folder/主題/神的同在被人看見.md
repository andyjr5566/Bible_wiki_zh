# 神的同在被人看見

## 定義／基本資料

以撒的生命見證讓外邦人也能看出「耶和華與他同在」。這是創世記中反覆出現的主題——神的同在不僅是主觀經歷，也是客觀可見的事實。

## 觸發來源

- [[創世記/第26章|創世記 第26章]]：28節

## 與目前整理書卷的關聯

### 創世記 第26章

- 亞比米勒對以撒說：「我們明明地看見耶和華與你同在」
- 以撒昌盛、挖井得水、蒙神保守——都是神同在的證據
- 來源：CT, GT, KC, BH
