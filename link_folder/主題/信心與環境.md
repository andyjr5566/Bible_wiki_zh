# 信心與環境

## 定義／基本資料

信心與環境的對比是創世記第十五章的主題：亞伯蘭面對無子的事實（環境），仍信靠神的話語（信心）。

## 與目前整理書卷的關聯

### 創世記 第15章

- 亞伯蘭提出疑問：「我既無子，你還賜我甚麼呢？」
- 神否定以利以謝為後裔，應許亞伯蘭本身所生的
- 亞伯蘭信耶和華，耶和華就以此為他的義
- 來源：CT, GT, KC
