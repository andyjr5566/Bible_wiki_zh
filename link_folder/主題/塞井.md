# 塞井

## 定義／基本資料

非利士人將亞伯拉罕時期所挖的水井塞住、填滿土，等於切斷以撒一家的生存倚賴和居住權。

## 觸發來源

- [[創世記/第26章|創世記 第26章]]：15節

## 與目前整理書卷的關聯

### 創世記 第26章

- 「當他父親亞伯拉罕在世的日子，他父親的僕人所挖的井，非利士人全都塞住，填滿了土。」
- 水井在巴勒斯坦是生存與生計的關鍵，塞井等於宣戰書和驅逐令
- 來源：CT, GT, KC
