# 瓦器打碎銅器擦淨

## 定義

煮贖罪祭肉的器皿處理條例：瓦器因多孔吸收祭肉汁液無法淨化，須打碎；銅器不吸收汁液，只需擦磨、在水中涮淨即可再用（利 6:28）。預表天然生命（瓦器）須十字架破碎，重生生命（銅器）只需話語水洗淨（弗 5:26）。

## 主題發展

利未記 6:28 專論贖罪祭肉煮熟後的器皿處理。CT 逐節註解解釋：當時瓦器無磁釉，汁液滲入必打碎；銅器不滲透，擦洗即淨。靈意上：「瓦器表徵人天然生命，須十字架破碎；銅器表徵審判，基督已滿足公義審判，信徒只要接受話語光照與水洗」。GT 拾穗引丁良才同樣指出瓦器吸收油味須打碎，銅器擦洗即淨；《舊約背景註釋》補充陶器多孔吸收不潔，青銅器易清洗。KingComments 引用林後 4:7「瓦器」喻身體，須破碎；銅器喻稱義後狀態，需話語洗淨。BibleHub 同樣解經：陶器多孔須毀壞，金屬器可擦洗重用。
