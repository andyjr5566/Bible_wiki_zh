# 打死人的必被治死

## 定義

根據利未記 24:17、21，規定故意殺人者必須被處死，強調人命的聖潔與公義。

## 按書卷累積

### 利未記

<!-- accumulation:利未記:24:start -->
#### [[03 利未記/第24章|第24章]]
- 本章重點：本章明文規定打死人者必被治死，與牲畜賠償條文並列，顯示法律對人命的最高保護。
- 與本章關聯：與「打死牲畜的必賠上牲畜」共同構成以命償命的法律體系，並延伸至寄居者同歸一例。
