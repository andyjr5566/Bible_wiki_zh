# 挖井

## 定義／基本資料

以撒重新挖出父親亞伯拉罕在世之日所挖的水井，並繼續挖新井。挖井豫表尋求屬靈供應。

## 觸發來源

- [[創世記/第26章|創世記 第26章]]：18-22節

## 與目前整理書卷的關聯

### 創世記 第26章

- 以撒重新挖出父親的井（非利士人曾塞住），仍照父親所叫的名字命名
- 繼續挖新井：埃色（相爭）、西提拿（為敵）、利河伯（寬闊）
- 以撒的品格：平和沉靜、堅忍持久，不願相爭，靜靜繼續挖井
- 來源：CT, GT, KC, BH
