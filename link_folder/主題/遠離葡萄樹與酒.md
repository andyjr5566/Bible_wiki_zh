# 遠離葡萄樹與酒

## 定義

拿細耳人三大禁令之首為「遠離清酒濃酒，也不可喝葡萄汁，不可吃鮮葡萄和乾葡萄，凡葡萄樹上結的自核至皮都不可吃」（v3-4）。酒在舊約中常象徵地上的喜樂與享受（士9:13, 詩104:15），拿細耳人遠離一切葡萄製品，表徵自願放棄屬地自然界的合法享受，防微杜漸，杜絕任何可能發酵成酒、影響心思清明或引致失控與罪惡的導火線（CT、GT、BH）。此禁令並非否定自然界的良善（1提4:4-5），而是強調更高層次的神聖奉獻：不以世俗喜樂為樂，單單以神自己為至寶與喜樂（CT、KC）。主耶穌在最後晚餐宣示「不再喝這葡萄汁，直到在父國裡喝新的」（太26:29），亦展現了完美拿細耳人離俗專一的屬靈意涵（KC）。
