# 除罪水

## 定義

除罪水在《民數記》第8章7節中被提及，是潔淨利未人儀式的一部分。摩西被吩咐將除罪水彈在利未人身上，以洗淨他們的罪污。在聖經的預表意義中，除罪水象徵基督流血的救贖功效與聖靈的洗淨，用以對付人外在的罪污，使事奉神的人能成為聖潔。

#### [[04 民數記/第8章|第8章]]
- 本章重點：利未人被選出後，摩西用除罪水彈在他們身上以潔淨他們。
- 與本章關聯：作為潔淨利未人以備事奉的第一步儀式，象徵取用基督贖罪的功效。
