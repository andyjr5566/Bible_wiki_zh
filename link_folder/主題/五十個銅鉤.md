# 五十個銅鉤

## 定義

用來連接山羊毛罩棚兩組幔子（五幅連成一幅與六幅連成一幅）的鉤子，使罩棚連成一個整體。

## 按書卷累積

### 出埃及記

<!-- accumulation:出埃及記:26:start -->
#### 第26章
- 本章重點：在罩棚相連的幔子末幅邊上做五十個鈕扣，並做五十個銅鉤鉤在鈕扣中。
- 與本章關聯：銅鉤使罩棚連成一個，象徵外層覆蓋物的完整與堅固連結。
