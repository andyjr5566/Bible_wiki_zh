# 會幕（帳幕整體）

## 定義

本章詳細記載了會幕（帳幕整體）的結構與建造指示。帳幕由內層的十幅細麻幔子、外層的山羊毛罩棚、染紅公羊皮蓋子與海狗皮頂蓋組成，並以皂莢木豎板為骨架，透過金鉤與銅鉤將各部件相連，成為一個可移動的神聖空間，作為神與人相會之處。
