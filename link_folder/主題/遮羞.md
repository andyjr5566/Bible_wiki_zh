# 遮羞

## 類型
主題

## 觸發來源

- [[創世記/第20章|創世記 第20 章]]：16節

## 目前資料

- 亞比米勒給撒拉一千銀子作「遮羞」（16節）
- 原文為「眼」，意為遮蓋、證明無辜
- 一千舍客勒約25磅銀子，遠超奴僕身價30舍客勒
- 證明撒拉清白，也承認亞比米勒家族的過犯
- 靈意：信徒犯罪後需付出代價對付，取用主救贖的功績（來十22）
