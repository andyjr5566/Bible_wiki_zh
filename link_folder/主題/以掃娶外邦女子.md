# 以掃娶外邦女子

## 定義／基本資料

以掃四十歲（以撒一百歲）娶了兩個赫人女子為妻，表明他不看重神的約，輕看長子名分。她們常使以撒和利百加愁煩。

## 觸發來源

- [[創世記/第26章|創世記 第26章]]：34-35節

## 與目前整理書卷的關聯

### 創世記 第26章

- 以掃娶赫人比利的女兒猶滴和赫人以倫的女兒巴實抹
- 「她們常使以撒和利百加心裡愁煩」——極可能因敬拜外邦偶像
- 以掃此舉違背了亞伯拉罕堅持不娶迦南女子的原則（創24:3-4）
- 來源：CT, GT, KC, BH
