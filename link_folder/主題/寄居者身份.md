# 寄居者身份

## 定義／基本資料

亞伯拉罕在創世記第23章中對赫人說：「我在你們中間是外人，是寄居的。」這不是一句客套話，而是對自己地位的現實承認——儘管神已應許迦南全地歸他，他此時連一塊葬身之地都沒有。

## 與目前整理書卷的關聯

### 創世記 第23章

- 亞伯拉罕對赫人說：「我在你們中間是外人，是寄居的」（4節）
- 表面上看似軟弱，實則信心宣告：神的應許不靠自己的勢力成就
- 承認寄居者身份 → 不靠自己的方法奪地，而是按公義程序購買
- 對比：神已應許迦南全地（創十三14-15；十五18），但亞伯拉罕在此選擇合法交易而非武力奪取

## 神學或研讀意義

- **信徒在世是寄居者**：我們是天上的國民（腓三20），在地上是外人、寄居的（來十一13；彼前二11）
- **不靠勢力、不憑血統**：亞伯拉罕沒有因神的應許而就地佔有，而是以合法程序取得
- **信心的順服不著急**：應許的成就有時需要過程和等待
