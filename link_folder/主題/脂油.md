# 脂油

## 定義

在利未記第1章的燔祭條例中，脂油被視為祭牲身上最精華、最美好的部分。經文指出，在剝皮切塊之後，祭司要把肉塊和頭並脂油擺在壇上火的柴上焚燒。脂油與頭、肉塊一同被獻在壇上全然燒毀，歸給耶和華，成為獻與神為馨香的火祭之重要組成部分。
