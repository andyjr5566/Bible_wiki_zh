# 金子（包金）

## 定義

金子在本章廣泛用於帳幕的建造，包括使幔子相連的五十個金鉤、包裹皂莢木豎板與閂、製作板上的金環、內幔四根柱子的包金與金鉤，以及門簾柱子的包金。金子象徵著神聖的尊貴與純潔。
