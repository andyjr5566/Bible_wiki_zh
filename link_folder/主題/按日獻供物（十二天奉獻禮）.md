# 按日獻供物（十二天奉獻禮）

## 定義

耶和華對摩西說：「眾首領為行奉獻壇的禮，要每天一個首領來獻供物」（v11）。民數記第7章用了聖經中最長的篇幅之一（共89節），不厭其煩地連續12次逐字詳細記錄了12位首領按日獻上一模一樣供物的過程（v12-83）。CT與KC深刻剖析，這種似乎重複的「流水帳」記載具有極高貴的神學價值：(1) **神的個人關注**：神不只看重整體，更極其珍惜並詳細記錄每一位屬神子民個人發自愛心的獻上，絕不敷衍與輕忽（CT、KC）；(2) **純潔不爭競**：各首領所獻完全一致，說明奉獻是向神表達敬愛而非與人攀比爭勝（CT）；(3) **完全的見證**：「十二」代表神揀選與事奉的完全，十二天的按日奉獻宣告以色列全體子民同心合意地活在蒙揀選與奉獻的神聖見證中（CT、GT、KC）。
