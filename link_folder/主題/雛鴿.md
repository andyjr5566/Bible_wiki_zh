# 雛鴿

## 定義

利未記5:7、11與斑鳩並列，作為貧窮人獻贖愆祭的替代祭物。若力量不夠獻羊羔，可帶兩隻斑鳩或兩隻雛鴿；若連兩隻鳥也買不起，則改獻細麵。雛鴿同樣一隻作贖罪祭，一隻作燔祭，獻祭儀式同斑鳩。

## 按書卷累積

### 利未記

<!-- accumulation:利未記:5:start -->
#### [[03 利未記/第5章|第5章]]
- 本章重點：第7、11節將雛鴿與斑鳩同等視為第二級贖愆祭祭物，反映神對經濟弱勢者的體恤。
- 與本章關聯：雛鴿是本章分級獻祭制度的一部分，確保人人都能依照能力獻祭贖罪。
