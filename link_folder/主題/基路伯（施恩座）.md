# 基路伯（施恩座）

## 定義

基路伯是天使的一種，常與神的榮耀和臨在相關。在施恩座上，兩個基路伯由同一塊精金錘出，安在施恩座兩頭，彼此接連一塊。二基路伯高張翅膀遮掩施恩座，臉對臉、朝著施恩座。他們象徵守護神聖空間、彰顯神的榮耀，並在贖罪日彈血之處見證遮蓋之恩。神從二基路伯中間向摩西說話，表明神臨在於恩典與公義相會之處。
