# 約櫃

## 定義

約櫃（aron ha'edut）是會幕至聖所中最核心的聖物，用皂莢木製造、裡外包精金，內存兩塊法版（十誡），上有精金施恩座與兩個基路伯。約櫃象徵神的寶座與同在，是神與以色列人立約的見證。其尺寸（長二肘半、寬一肘半、高一肘半）由「五」與「三」的半數構成，豫表達到完滿見證的地步。皂莢木包金豫表基督神人二性；施恩座（蔽罪蓋）豫表基督為挽回祭，遮蓋律法的定罪，使神能施恩與人相會。約櫃四腳安金環、穿杠、杠不可抽出，表徵隨時預備行動、高舉基督、儆醒事奉。
