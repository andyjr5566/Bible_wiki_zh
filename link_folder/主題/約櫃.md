# 約櫃

## 定義

約櫃是會幕中最核心的器具，以皂莢木製成，內外包精金，內藏法版。它預表基督的道成肉身（木代表人性，金代表神性），並見證神與人立約。約櫃之上安放施恩座，成為神與人相會、施恩與啟示的中心。

## 主題發展

約櫃在舊約歷史中代表神的同在與引導，後來被安置於所羅門聖殿的至聖所中。其屬靈意義在基督身上得著應驗，基督成為神與人相會的實體，祂完全滿足了律法的要求，並成為神榮耀同在的彰顯。
