# 聖衣（祭司聖服）

## 定義

聖衣是為大祭司亞倫及其兒子專門設計的服飾，象徵聖潔與分別為聖的事奉。出埃及記29章記載，亞倫需穿上內袍、以弗得的外袍、以弗得、胸牌，束上巧工織的帶子，戴上冠冕與聖冠，並受膏油澆灌。聖衣表徵基督的公義與榮耀，信徒事奉神必須「披戴基督」，有好的生活行為。亞倫的聖衣需留給子孫，後代繼承大祭司職分時要穿著七天，表明祭司事奉的傳承與完全的分別為聖。

## 按書卷累積

### 出埃及記

<!-- accumulation:出埃及記:29:start -->
#### 第29章
- 本章重點：亞倫就職時需穿戴內袍、外袍、以弗得、胸牌等聖衣，並受膏；其聖衣需留給子孫世代傳承。
- 與本章關聯：聖衣預表基督的公義與榮耀，表明事奉神者必須披戴基督，並在生活行為上完全分別為聖。
