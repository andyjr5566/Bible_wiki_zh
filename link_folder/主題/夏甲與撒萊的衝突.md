# 夏甲與撒萊的衝突

## 定義／基本資料

夏甲與撒萊的衝突是本章的核心事件：夏甲懷孕後小看撒萊，撒萊苦待夏甲，夏甲逃走。這衝突反映恩典與律法、屬靈與屬肉之間的張力。
