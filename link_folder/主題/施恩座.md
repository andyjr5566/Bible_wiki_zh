# 施恩座

## 定義

施恩座是安在約櫃上邊的一塊精金蓋板，希伯來文 kapporet，字根有「遮蓋」之意，故又稱蔽罪座。長二肘半、寬一肘半，與約櫃尺寸相同。其上兩端各有一個金子錘出的基路伯，高張翅膀遮掩施恩座。神在此與摩西相會，並從二基路伯中間向以色列人說話。施恩座表明神的公義有多大，恩典也有多大；律法被恩典遮蓋，神在此不是審判，乃是施恩。

#### 第25章

- 與本章關聯：本章詳述施恩座的尺寸、材質與功用，顯明神在約櫃上設立恩典的寶座，使律法下的百姓可以藉遮蓋之恩與神相交。
