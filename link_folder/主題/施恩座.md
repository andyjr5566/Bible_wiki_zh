# 施恩座

## 定義

施恩座是約櫃的精金蓋子，希伯來文字根具有「遮蓋」或「贖罪」之意。其尺寸與約櫃完全相同，表徵神的救恩完滿無缺。在贖罪日大祭司需將祭牲的血彈在施恩座上，表明神公義的要求已因血得著滿足，使律法的定罪被恩典遮蔽，神便在此向人施恩。

## 按書卷累積

### 出埃及記

<!-- accumulation:出埃及記:25:start -->
#### 第25章
- 本章重點：神指示要用精金做施恩座，安在約櫃上邊，並作為神與摩西相會、說話的所在。
- 與本章關聯：施恩座蓋住櫃內的法版，表徵神的憐憫與恩典建立在祂的公義與律法之上，成為神人相會的唯一根基。
