# 出埃及記 第28章

1. 你要從以色列人中，使你的哥哥[[亞倫]]和他的兒子[[拿答]]、[[亞比戶]]、[[以利亞撒]]、[[以他瑪]]一同就近你，給我供[[祭司職分]]。
2. 你要給你哥哥[[亞倫]]做[[聖衣]]為[[榮耀]]，為[[華美]]。
3. 又要吩咐一切心中有[[智慧]]的，就是我用[[智慧]]的靈所充滿的，給[[亞倫]]做衣服，使他[[分別為聖]]，可以給我供[[祭司職分]]。
4. 所要做的就是[[胸牌]]、[[以弗得]]、[[外袍]]、[[內袍]]、[[冠冕]]、[[腰帶]]，使你哥哥[[亞倫]]和他兒子穿這[[聖衣]]，可以給我供[[祭司職分]]。
5. 要用金線和藍色、紫色、朱紅色線，並細麻去做。
6. 他們要拿金線和藍色、紫色、朱紅色線，並撚的細麻，用巧匠的手工做[[以弗得]]。
43. 亞倫和他兒子進入會幕，或就近壇，在聖所供職的時候必穿上，免得擔罪而死。這要為[[亞倫]]和他的後裔作[[永遠的定例]]。

## 本章整理


**重點摘要**
1. **亞倫被選為祭司（1-5節）**：神揀選亞倫和他的四個兒子（拿答、亞比戶、以利亞撒、以他瑪）供祭司職分，吩咐摩西為他們做聖衣，為榮耀、為華美。
2. **製作以弗得（6-14節）**：以弗得用金線、藍色、紫色、朱紅色線和撚的細麻作成，兩條肩帶上各安一塊紅瑪瑙，刻以色列十二個兒子的名字，作為紀念石，亞倫在耶和華面前擔當他們的名字。
3. **製作胸牌（15-30節）**：胸牌四方的，疊為兩層，鑲十二顆寶石，按以色列十二支派的名字刻上。胸牌裡放烏陵和土明，亞倫進聖所時帶在胸前，在耶和華面前常作紀念。
4. **製作外袍（31-35節）**：外袍全是藍色的，底邊有石榴和金鈴鐺相間，亞倫供職時穿著，聲音被聽見，使他不至於死亡。
5. **製作冠冕、內袍、腰帶、褲子（36-43節）**：冠冕上有金牌，刻「歸耶和華為聖」；內袍用細麻織成；腰帶用繡花手工；褲子遮掩下體。6. **膏立祭司（40-43節）**：為亞倫和他兒子穿上聖衣，膏立他們，分別為聖，供祭司職分。7. **永遠的定例（43節）**：這要為亞倫和他的後裔作永遠的定例。

**神學意義**
- 祭司聖衣預表基督：以弗得預表基督作大祭司（來5:1-5, 7:26-28），胸牌預表基督將信徒名字刻在心上（來7:25），烏陵土明預表基督的智慧與引導（來7:25）。
- 祭司職分預表基督：亞倫家族祭司預表基督作大祭司（來5:1-5, 7:26-28），信徒作王室祭司（彼前2:9, 啟1:6, 5:10）。
- 聖衣預表基督的義：聖衣為榮耀、為華美，預表基督的義歸給信徒（來7:26-28, 彼前2:9）。
- 永遠的定例：祭司職分是永遠的定例，指向基督永遠的祭司職分（來7:24-25）。
