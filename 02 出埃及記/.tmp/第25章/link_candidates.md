# Link Candidates — 出埃及記 第25章

## 人物
- 摩西 → 人物

## 地點
- 西乃山 → 地點 — 摩西仍在山上四十晝夜中領受建會幕的指示，承接出24:15-18（出25:9,40）

## 神學
- 甘心樂意的奉獻 → 神學 — 凡甘心樂意的人可送禮物歸神作建聖所之用；奉獻動機須出於甘心非勉強（出25:2-3）
- 神住在人間 → 神學 — 又當為我造聖所，使我可以住在他們中間，是建造會幕的核心理由（出25:8）
- 山上的樣式 → 神學 — 製造帳幕和其中一切器具都要照所指示的樣式，全書多次重複強調（出25:9,40）
- 會幕材料的豫表 → 神學 — 金、銀、銅、藍紫朱紅線、細麻、山羊毛、染紅公羊皮、海狗皮、皂莢木、油、香料、寶石，各按其性質豫表基督與救恩（出25:3-7）

## 主題
- 約櫃 → 主題 — 皂莢木包精金所造，長二肘半寬高一肘半，內存法版，是立約見證（出25:10-16,21-22）
- 施恩座 → 主題 — 精金所造約櫃蓋，神在此與摩西相會說話，是遮罪、施恩之處（出25:17,21-22）
- 基路伯（施恩座） → 主題 — 兩個金錘出的基路伯，安在施恩座兩頭，高張翅膀相對遮掩施恩座（出25:18-20）
- 陳設餅桌子 → 主題 — 皂莢木包精金所造，桌上常擺陳設餅在耶和華面前（出25:23-30）
- 金燈臺 → 主題 — 精金一塊錘出，六枝七盞，杏花杯球花裝飾，是聖所唯一光源（出25:31-39）

## 原文
- 皂莢木（atzei shittim） → 原文 — 產於西乃曠野的堅硬木材，用以製造約櫃、桌子、燈臺骨架與槓（出25:5,10,13,23,28）
- 禮物（terumah） → 原文 — 甘心奉獻歸神的供物，非強索之物（出25:2-3）
- 法版（edut） → 原文 — 刻十誡的兩塊石版，是神與人立約的見證（出25:16,21）

## 互文
- 出24:15-18 → 互文 — 摩西進入雲中四十晝夜，與本章「耶和華曉諭摩西」承接
- 來8:5 → 互文 — 摩西謹慎照山上指示的樣式建造帳幕，對照出25:9,40
- 來9:8-9 → 互文 — 頭一層帳幕仍存的時候，進入至聖所的路還未顯明，對照出25:8-9
- 來9:11-12 → 互文 — 基督作大祭司經過更大更全備的帳幕，成就永遠的救贖，對照出25:9
- 來9:23-24 → 互文 — 地上帳幕是天上事物的複本，基督親自進入天上真聖所，對照出25:8-9
- 羅3:25 → 互文 — 神設立耶穌作挽回祭，原文與施恩座同字，對照出25:17
- 約1:14 → 互文 — 道成肉身，支搭帳棚住在我們中間，對照出25:8
- 約2:19-21 → 互文 — 耶穌以自己的身體為殿，對照出25:8-9
- 約6:35 → 互文 — 我就是生命的糧，對照出25:30陳設餅
- 約8:12 → 互文 — 我是世界的光，跟從我的必得著生命的光，對照出25:31-37金燈臺
- 啟1:20 → 互文 — 七個金燈臺就是七個教會，對照出25:31-39
- 啟21:18 → 互文 — 新耶路撒冷是精金所造，對照出25:11,24,31等處精金
- 創3:24 → 互文 — 基路伯把守生命樹的道路，對照出25:18-20基路伯的守護職分
- 結1:5-11 → 互文 — 以西結所見基路伯異象，人面獸身有翅膀，對照出25:18-20
- 亞4:1-6 → 互文 — 撒迦利亞所見金燈臺異象，七盞燈是神的靈，對照出25:31-37
- 撒下6:6-7 → 互文 — 烏撒摸約櫃遭擊殺，顯明約櫃的神聖不可直接觸碰，對照出25:14-15槓不可抽出
- 林後9:7 → 互文 — 各人隨本心所酌定的，捐得樂意的人是神所喜愛的，對照出25:2甘心樂意的奉獻
- 王上6:23-28 → 互文 — 所羅門聖殿至聖所內兩個包金的基路伯，對照出25:18-20
