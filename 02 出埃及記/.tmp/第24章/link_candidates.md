## 神學
- 西乃之約 → 神學 — 立約儀式、灑血、約書、百姓應許遵行（出24:3-8）
- 立約之血 → 神學 — 摩西將血灑在壇上與百姓身上，立約的憑據（出24:6-8）
- 神的榮耀 → 神學 — 雲彩遮蓋山、耶和華的榮耀如烈火、摩西進入雲中（出24:15-18）
- 中保 → 神學 — 摩西作中保，獨自親近神，預表基督（出24:2, 12-18）
- 遠遠下拜 → 神學 — 祭司長老只能遠遠下拜，不可親近（出24:1-2, 9-11）
- 看見神 → 神學 — 長老們看見以色列的神，腳下如藍寶石，又吃又喝（出24:9-11）
- 四十晝夜 → 神學 — 摩西在山上四十晝夜，領受石版律法誡命（出24:18）

## 事件
- 立約儀式 → 事件 — 摩西述說命令、百姓應許、寫約書、築壇、立柱、獻燔祭平安祭、灑血、念約書、百姓再應許、灑血在百姓身上（出24:3-8）
- 長老上山見神 → 事件 — 摩西、亞倫、拿答、亞比戶、七十長老上山，看見神，吃喝（出24:9-11）
- 摩西上山領石版 → 事件 — 神召摩西上山，摩西帶約書亞，囑咐長老等候，雲彩遮山六天，第七天召摩西，摩西進雲中四十晝夜（出24:12-18）
- 十二根柱子 → 事件 — 按十二支派立柱子，作立約見證（出24:4）

## 互文
- 出19:8 → 互文 — 百姓齊聲說「我們都必遵行」，與出24:3,7平行
- 出19:12-13 → 互文 — 界限、不可摸山，與出24:1-2親近等級平行
- 出20:21 → 互文 — 摩西挨近濃雲，與出24:2,15-18平行
- 出24:12 → 互文 — 神要將石版律法誡命賜給摩西，與出31:18, 34:1,28平行
- 來9:18-22 → 互文 — 不流血立約不能成立，解釋出24:6-8灑血儀式
- 來9:19 → 互文 — 摩西將血灑在約書和百姓身上
- 來12:24 → 互文 — 新約中保耶穌，灑血說話比亞伯的血更美，對照出24:8
- 彼前1:2 → 互文 — 靈使人成聖，以致順服耶穌基督，又蒙祂血灑在身上
- 來3:1-6 → 互文 — 摩西為僕人，基督為兒子，掌管神的家，對照出24:2中保預表
- 來10:19-20 → 互文 — 藉著耶穌的血，我們坦然無懼進入至聖所，對照出24:1-2遠遠下拜
- 太26:28 → 互文 — 這是我立約的血，為多人流出來，使罪得赦，對照出24:8
- 路22:20 → 互文 — 這杯是用你們的血所立的新約，對照出24:8
- 林前11:25 → 互文 — 這杯是用我的血所立的新約，對照出24:8
- 結1:26-27 → 互文 — 以西結見神榮耀，腳下如藍寶石，人形，對照出24:10
- 申9:9-11 → 互文 — 摩西在山四十晝夜領受石版，對照出24:18
- 申10:1-5 → 互文 — 重造石版，對照出34:1-4
- 王上18:31 → 互文 — 以利亞按十二支派立十二石，對照出24:4
