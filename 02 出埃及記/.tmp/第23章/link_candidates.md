# Link Candidates — 出埃及記 第23章

## 神學
- 司法公義 → 神學 — 不可隨眾行惡、不可偏護窮人、不可受賄賂，體現神的公義屬性（出23:1-8）
- 安息年 → 神學 — 第七年地歇息，供給窮人與野獸，彰顯神的供應與眷顧（出23:10-11）
- 安息日 → 神學 — 第七日安息，使牲畜、僕婢、寄居者都得舒暢，體現神的憐憫（出23:12）
- 三大節期 → 神學 — 除酵節、收割節、收藏節，一年三次朝見神，紀念救贖與供應（出23:14-19）
- 天使引導 → 神學 — 神差遣使者在前引導，奉神名而來，不可惹他（出23:20-23）
- 專一事奉 → 神學 — 不可跪拜別神、不可效法他們行為、要拆毀神像（出23:24-25）
- 神賜福應許 → 神學 — 事奉神必賜福糧水、除去疾病、無墜胎不生產、滿足年日（出23:25-26）
- 逐漸趕出 → 神學 — 神漸漸趕出迦南人，免得地荒涼野獸增多（出23:29-30）
- 立約界限 → 神學 — 不可與迦南人及其神立約，恐成為網羅（出23:32-33）

## 原文
- 謠言（shema`） → 原文 — 虛妄的報告、空洞的傳聞，不可隨夥佈散（出23:1）
- 聯手（shathaph） → 原文 — 放置手、聯手，不可與惡人聯手妄作見證（出23:1）
- 隨眾（rab） → 原文 — 許多、大多數，不可隨眾行惡、隨眾偏行（出23:2）
- 屈枉（natah） → 原文 — 折彎、傾斜，作見證屈枉正直、不可屈枉窮人（出23:2,6）
- 寄居的（ger） → 原文 — 旅居者、外僑，不可欺壓寄居的（出23:9）
- 歇息（shamat） → 原文 — 讓落下、鬆開，第七年要叫地歇息（出23:11）
- 安息（shabath） → 原文 — 中止、休息，第七日要安息（出23:12）
- 守節（chagag） → 原文 — 守節期、朝聖，一年三次向神守節（出23:14）
- 除酵（matsah） → 原文 — 無酵餅，守除酵節吃無酵餅七天（出23:15）
- 初熟（bikkurim） → 原文 — 初熟之物，收割節獻初熟之物（出23:16）
- 收藏（asaph） → 原文 — 收集、聚集，年底收藏節（出23:16）
- 使者（mal'akh） → 原文 — 天使、差遣者，神差遣使者在前面行（出23:20）
- 惹（marah） → 原文 — 違背、悖逆，不可惹使者（出23:21）
- 剪除（kareth） → 原文 — 消除、抹去，神必將迦南人剪除（出23:23）
- 柱像（matsebah） → 原文 — 石柱、紀念柱，打碎他們的柱像（出23:24）
- 黃蜂（tsirah） → 原文 — 大黃蜂，神打發黃蜂趕出迦南人（出23:28）
- 漸漸（me'at me'at） → 原文 — 一點點、少量，神漸漸將他們趕出去（出23:30）
- 網羅（moqesh） → 原文 — 陷阱、圈套，事奉他們的神必成為網羅（出23:33）

- 古代近東三大節期 → 背景 — 農業社會節期與自然週期相關、神將救贖史連結其上
- 古代近東天使/使者 → 背景 — 神差遣使者引導、保護、審判，奉神名而來
- 古代近東偶像崇拜 → 背景 — 迦南宗教有神像、柱像、廟妓、獻兒女給摩洛

- 古代近東立約 → 背景 — 不可與迦南人立約、恐成為網羅、宗教混合主義危險
- 古代近東寄居者保護 → 背景 — 外僑無親族保護、易被欺壓、以色列親身經歷埃及寄居
- 古代近東安息日 → 背景 — 七日週期、人道理由讓牲畜僕婢歇息、非單純宗教儀式
- 古代近東賄賂 → 背景 — 政府官員非法收入、影響判決、漢摩拉比法典剝奪職位

## 主題
- 公義與真理 → 主題 — 司法公正、不可作假見證、不可隨眾、不可受賄賂
- 愛鄰舍與仇敵 → 主題 — 歸還仇敵牛驢、幫助恨你人抬重擔、不可欺壓寄居者
- 關懷弱勢 → 主題 — 安息年供給窮人野獸、安息日使僕婢寄居者歇息、司法保護窮人
- 敬畏神與專一事奉 → 主題 — 不可提別神名、不可跪拜偶像、拆毀神像、單單事奉耶和華
- 神的引導與保護 → 主題 — 差遣使者在前引導、向仇敵作仇敵、使敵人驚駭逃跑
- 漸進性征服 → 主題 — 神漸漸趕出迦南人、免得地荒涼野獸增多、直到人數加多承受地業
- 節期與紀念 → 主題 — 一年三次朝見神、除酵節收割節收藏節、紀念出埃及與神供應
