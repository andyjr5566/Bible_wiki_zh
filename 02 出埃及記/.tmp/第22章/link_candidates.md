## 神學

## 原文
- 偷（ganab） → 原文 — 偷竊、拐帶、拿走（出22:1）
- 賠償（shalem） → 原文 — 賠償、回報，加倍賠還的概念（出22:1,3-5,7,9,12,14）
- 寄居的（ger） → 原文 — 臨時居民、外國人、外僑（出22:21）
- 當頭（`abot） → 原文 — 典當、抵押品，窮人外袍作當頭須日落前歸還（出22:26）
- 取利（neshek） → 原文 — 利息、高利，借錢給窮人不可取利（出22:25）
- 毀謗（qalal） → 原文 — 藐視、詛咒，不可毀謗神與官長（出22:28）
- 行邪術的女人（mekhashephah） → 原文 — 女巫、行邪術者，不可容她存活（出22:18）
- 滅絕（charam） → 原文 — 滅絕、毀滅、歸神當滅之物（出22:20）
- 聖潔（qodesh） → 原文 — 分別、神聖，你們要為聖潔的人（出22:31）
- 聘禮（mohar） → 原文 — 聘金、新娘價，引誘處女須交聘禮（出22:16-17）
- 審判官（elohim） → 原文 — 神與審判官同字，受託者須就近審判官起誓（出22:8-9）

## 背景
- 古代近東偷竊賠償法 → 背景 — 漢摩拉比法典偷竊廟宇宮殿判死刑，出22從輕賣賊為奴
- 古代近東自衛權 → 背景 — 吾珥南模法典夜間殺賊正當，出22:2-3區分日夜
- 古代近東受託制度 → 背景 — 無銀行保險制度，受託保管須起誓證明清白
- 古代近東聘禮制度 → 背景 — 聘禮為結婚協議一部分，保障女方未來生活
- 古代近東行邪術 → 背景 — 迦南宗教盛行巫術占卜，以色列嚴禁以保信仰純潔
- 古代近東人獸性行為 → 背景 — 赫人法律亦禁止，與迦南生育崇拜有關
- 古代近東放債取利 → 背景 — 古代社會放債常致人賣身為奴，以色列禁向同胞取利
- 古代近東當頭制度 → 背景 — 外袍為窮人唯一禦寒衣物，日落前須歸還
- 古代近東初熟果子奉獻 → 背景 — 農業社會以初熟農產獻祭承認神的主權
- 古代近東頭生奉獻 → 背景 — 頭生牲畜第八天歸神，與逾越節長子救贖關聯
- 古代近東飲食潔淨 → 背景 — 被野獸撕裂的肉不潔淨，丟給狗吃
- 古代近東社會弱勢群體 → 背景 — 寄居者、寡婦、孤兒為社會最無保障階層

## 主題
- 賠償與公義 → 主題 — 偷竊與損壞須加倍賠償，彰顯神的公義原則
- 聖潔生活 → 主題 — 以色列人要在神面前為聖潔的人，從飲食到行為分別為聖
- 保護弱勢 → 主題 — 神特別保護寄居者、寡婦、孤兒、窮人
- 敬畏神 → 主題 — 不可毀謗神、不可祭祀別神、初熟果子歸神
- 財產權 → 主題 — 偷竊賠償、受託責任、借物賠償保障私有財產
- 聖潔飲食 → 主題 — 不可吃被野獸撕裂的肉，聖潔的人不吃不潔之物
