# 不誠心的

## 定義

指信徒（或整體群體）在祈求或敬拜表面上參與，但內心缺乏真實悔改與信心。[[何西阿]]特別以此詞描述北國百姓的宗教行為：為物質祝福而聚集，卻非真正歸向神。

## 與目前整理書卷的關聯

== 背景說明 ==

何西阿書7章14節顯示百姓的「宗教行為」（在床上呼求、聚集）並非出於真心悔改，而是為物質利益（五穀新酒）。他們把神當作可以操縱的工具來求福，而非以盟約的心回應神。這與整卷何西阿的核心信息一致：神要的是「良善（憐�）和認識」，不是祭祀（6:6）。

## 神學意義

- 敬拜不等於信仰：表面的聚集和呼號不等於真正的回轉
- 神是否接受祭祀／敬拜取決於內心動機，非表面的儀式
- 「仍然悖逆」說明持續的悖逆使宗教行為變成一種徒勞
