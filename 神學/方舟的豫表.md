# 方舟的豫表

## 定義／基本資料

創世記六章方舟豫表基督——在神的審判中，惟有在方舟（基督）裡的人得保全生命。方舟的每個細節都指向基督的某方面。

## 方舟豫表基督八方面

| 項目 | 說明 | 靈意豫表 |
|---|---|---|
| 歌斐木 | 富樹脂柏木，耐水浸蝕 | 基督堅固完美的人性 |
| 分一間一間造 | 內部隔間 | 基督是各種人的安息 |
| 抹松香 | 柏油/瀝青防水 | 基督寶血的救贖功效 |
| 尺寸 | 300×50×30 肘 | 基督的完備豐滿 |
| 透光處 | 上邊高一肘 | 基督是世界的光 |
| 門 | 門開旁邊 | 基督可親可近，是通神之門路 |
| 三層 | 上、中、下 | 對基督經歷的不同程度 |
| 松香原文 | 與「贖罪」同字 | 基督的救贖 |

## 神學意義／研讀意義

- 在方舟裡＝在基督裡，得保全生命
- 方舟內外的分別：惟有進入方舟者得救——「除祂以外，別無拯救」（徒四12）
- 洪水豫表審判（太廿四39）和受浸的水（彼前三20-21）
- 方舟也豫表教會——在基督裡一同經過審判
