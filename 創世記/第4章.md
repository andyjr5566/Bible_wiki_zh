# 創世記 第4章

1. 有一日，那人和他妻子[[夏娃]]同房，夏娃就懷孕，生了[[該隱]]（就是得的意思），便說：「耶和華使我得了一個男子。」
2. 又生了[[該隱]]的兄弟[[亞伯]]。[[亞伯]]是牧羊的；[[該隱]]是種地的。
3. 有一日，[[該隱]]拿地裡的出產為供物獻給耶和華。
4. [[亞伯]]也將他羊群中頭生的和羊的脂油獻上。耶和華看中了[[亞伯]]和他的供物，
5. 只是看不中[[該隱]]和他的供物。[[該隱]]就大大的發怒，變了臉色。
6. 耶和華對[[該隱]]說：「你為什麼發怒呢？你為什麼變了臉色呢？
7. 你若行得好，豈不蒙悅納？你若行得不好，[[罪伏在門前|罪就伏在門前]]；他必戀慕你，你卻要制伏他。」
8. [[該隱]]與他兄弟[[亞伯]]說話；二人正在田間。[[該隱]]起來打他兄弟[[亞伯]]，把他殺了。
9. 耶和華對[[該隱]]說：「你兄弟[[亞伯]]在那裡？」他說：「我不知道！[[我豈是看守我兄弟的麼]]？」
10. 耶和華說：「你作了什麼事呢？你兄弟[[亞伯的血哀告|亞伯的血]]有聲音從地裡向我哀告。
11. 地開了口，從你手裡接受你兄弟的血。現在你必從這地受咒詛。
12. 你種地，地不再給你效力；你必流離飄蕩在地上。」
13. [[該隱]]對耶和華說：「我的刑罰太重，過於我所能當的。
14. 你如今趕逐我離開這地，以致不見你面；我必流離飄蕩在地上，凡遇見我的必殺我。」
15. 耶和華對他說：「凡殺[[該隱]]的，必遭報七倍。」耶和華就給[[該隱的記號|該隱立一個記號]]，免得人遇見他就殺他。
16. 於是[[該隱]]離開耶和華的面，去住在[[伊甸]]東邊[[挪得]]之地。
17. [[該隱]]與妻子同房，他妻子就懷孕，生了[[以諾（該隱之子）|以諾]]。[[該隱]]建造了一座城，就按著他兒子的名將那城叫作以諾。
18. [[以諾（該隱之子）|以諾]]生以拿；以拿生米戶雅利；米戶雅利生瑪土撒利；瑪土撒利生[[拉麥]]。
19. [[拉麥]]娶了兩個妻：一個名叫亞大，一個名叫洗拉。
20. 亞大生雅八；雅八就是住帳棚、牧養牲畜之人的祖師。
21. 雅八的兄弟名叫猶八；他是一切彈琴吹簫之人的祖師。
22. 洗拉又生了土八該隱；他是打造各樣銅鐵利器的（或作：是銅匠鐵匠的祖師）。土八該隱的妹子是拿瑪。
23. [[拉麥]]對他兩個妻子說：「亞大、洗拉，聽我的聲音；拉麥的妻子，細聽我的話語：壯年人傷我，我把她殺了；少年人損我，我把她害了（或作我殺壯士卻傷自己，我害幼童卻損本身）。
24. 若殺[[該隱]]，遭報七倍，殺[[拉麥]]，必遭報七十七倍。」
25. [[亞當]]又與妻子同房，他就生了一個兒子，起名叫[[塞特]]，意思說：「神另給我立了一個兒子代替[[亞伯]]，因為[[該隱]]殺了他。」
26. [[塞特]]也生了一個兒子，起名叫[[以挪士]]。那時候，人才[[求告耶和華的名]]。

## 本章整理

### 一、該隱與亞伯的出生（1-2節）

- **該隱**（原意「得著」）：夏娃誤以為他是創三15所應許的「女人的後裔」
- **亞伯**（原意「氣息、虛空」）：夏娃失望後以此命名，反映她體認到離開神的人生是「虛空的虛空」
- 行業對比：亞伯牧羊（為神而活）、該隱種地（為肚腹而活）

### 二、獻祭的對比（3-7節）

| 該隱的祭 | 亞伯的祭 |
|---|---|
| 地裡出產（莊稼） | 羊群頭生的 + 脂油 |
| 憑己意，非出於啟示 | 因著信，遵行神旨意（來十一4） |
| **被神拒絕** | **被神看中** |
| 象徵：人意宗教，靠行為稱義 | 象徵：啟示宗教，藉流血犧牲蒙悅納 |

- 神不只悅納祭物，也悅納**獻祭的人**
- **聽命勝於獻祭**（撒上十五22）
- 神對該隱的警戒（7節）：罪被描寫為**人格化的活物**，蹲伏門前隨時撲來（羅七17）
- 宗教勸人為善，但**不能給人能力行善**（羅七18）

### 三、殺害與刑罰（8-16節）

**殺害（8節）：**
- 屬血氣的逼迫屬靈的（加四29）
- 宗教特徵之一：兇殺；宗教徒殺人還以為是事奉神（約十六2）

**刑罰（11-12節）：**
- 地不再給他效力
- 流離飄蕩（外面流離失所 + 裡面飄蕩無定向）

**該隱的回應（9節）：**
- 「我不知道。我豈是看守我兄弟的嗎？」——說謊 + 頂撞神

**該隱的自憐（13節）：**
- 「我的刑罰太重，過於我所能當的」——**自憐的怨歎，非真正認罪**

**神的憐憫（15節）：**
- 立記號，殺該隱者遭報七倍

**離開神面（16節）：**
- 人第一次墮落：離開伊甸園；第二次墮落：**離開耶和華的面**
- 住在伊甸東邊「挪得」（意為「飄蕩」）之地

### 四、該隱的後代——無神文化（17-24節）

**人名意義與文化發展**：

| 人名 | 原意 | 文化意義 |
|---|---|---|
| 以諾 | 開始、學習 | 人開始學習無神生活技能 |
| 拉麥 | 健壯、有能力 | 強力推展無神文化建設 |
| 雅八 | 流動、領導 | 發明遊牧生活（經濟） |
| 猶八 | 音樂 | 發明音樂（藝術文化） |
| 土八該隱 | 該隱的擴展 | 發明銅鐵利器（戰爭文化） |
| 拿瑪 | 悅心的 | 殺人取樂，強橫兇暴 |

**拉麥的狂妄宣言（23-24節）：**
- 「壯年人傷我，我把他殺了」— 自我膨脹，以暴制暴
- 「殺拉麥，必遭報七十七倍」— 自以為比神照顧該隱還強十倍
- 該隱路線的三種文化：**經濟（雅八）→ 藝術（猶八）→ 戰爭（土八該隱）**

### 五、塞特的出生——神的恢復（25-26節）

- **塞特**（原意「設立、代替」）：神另立後裔代替亞伯
- 豫表**新造的族類**，歸入基督，脫離舊造敗壞（羅六3-7）
- **以挪士**（原意「脆弱必死的人」）：開始求告耶和華的名
- 人唯有認識自己的**軟弱無能**，才會轉而呼求神

### 六、兩條路線的對比

| 不要神的人（該隱路線） | 要神的人（塞特路線） |
|---|---|
| 發展文化：遊牧→音樂→武器→殺戮 | 認識自己軟弱必死（以挪士） |
| 越走越遠，離開神面 | 轉而求告耶和華的名 |
| 豫表舊造的族類 | 豫表新造的族類 |
