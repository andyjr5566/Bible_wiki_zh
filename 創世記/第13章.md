# 創世記 第13章

1. [[亞伯蘭]]帶著他的妻子[[撒萊]]，並他姪兒[[羅得]]，和一切所有的，從[[埃及]]上[[南地]]去。
2. [[亞伯蘭]]的牲畜和金銀極多。
3. 他從[[南地]]漸漸往[[伯特利]]去，到了伯特利和艾的中間，就是從前支搭帳棚的地方，
4. 也是他起先[[帳棚與祭壇|築壇]]的地方；他又在那裏[[信心與世俗的對比|求告耶和華的名]]。
5. [[羅得]]也有牛群、羊群、帳棚。
6. 那地容不下他們，因為他們的財物甚多，使他們不能同居。
7. [[亞伯蘭]]的牧人和[[羅得]]的牧人相爭。
8. [[亞伯蘭]]對[[羅得]]說：「你我不可相爭，你的牧人和我的牧人也不可相爭，因為我們是弟兄。
9. 遍地不都在你眼前嗎？請你離開我。你向左，我就向右；你向右，我就向左。」
10. [[羅得]]舉目看見[[約但河]]全平原，直到瑣珥，都是滋潤的。那地在耶和華滅所多瑪、蛾摩拉以前，如同[[耶和華的園子]]，也像[[埃及]]地。
11. 於是[[羅得]]選擇[[約但河]]全平原，往東遷移；他們就分離了。
12. [[亞伯蘭]]住在[[迦南地]]，[[羅得]]住在平原的城邑，漸漸挪移帳棚，直到[[所多瑪]]。
13. [[所多瑪]]人在耶和華面前罪大惡極。
14. [[羅得]]離別之後，耶和華對[[亞伯蘭]]說：「從你所在的地方，你舉目向東西南北觀看；
15. 凡你所看見的一切地，我都要賜給你和你的後裔，直到永遠。
16. 我也要使你的後裔如同地上的塵沙那樣多，人若能數算地上的塵沙，才能數算你的後裔。
17. 你起來，縱橫走遍這地，因為我必把這地賜給你。」
18. [[亞伯蘭]]就搬了帳棚，來到[[希伯崙]][[幔利]]的橡樹那裏居住，在那裏為耶和華[[帳棚與祭壇|築了一座壇]]。

## 本章整理
### 一、亞伯蘭的回歸與信心恢復（1-4節）

- 亞伯蘭從[[埃及]]上[[南地]]去，金、銀、牲畜極多
- 原文「極多」意為「很沉重」——**財物比窮乏是更深的危機**；人勝過饑餓，但不一定能勝過富足
- 亞伯蘭回到[[伯特利]]和艾的中間，就是從前支搭帳棚、起先築壇的地方
- 恢復「求告耶和華的名」——恢復與神相交的信心生活
- 靈命的復甦是「漸漸」的

### 二、牧人相爭與[[謙讓]]（5-9節）

- [[羅得]]也有牛群、羊群、帳棚——他跟隨亞伯蘭卻缺乏獨立的屬靈信念
- 「那地容不下他們，因為他們的財物甚多」——**教會中的難處往往肇因於財物**
- 亞伯蘭以長輩身份主動提出分手，強調「我們是弟兄」
- 他讓羅得優先選擇，自己接受任何結果
- **放棄「選擇權」乃最佳和平解決辦法**；有信心的人讓別人先選，因神選的總比人好

### 三、[[羅得]]的選擇與[[羅得的墮落]]（10-13節）

#### 羅得選擇的四重錯誤

1. 亞伯蘭比他年長
2. 羅得是跟隨亞伯蘭的
3. 羅得的產業多因亞伯蘭得來
4. 那地是應許給亞伯蘭的

#### 墮落步驟

| 步驟 | 內容 |
|------|------|
| 1. 眼錯 | 舉目看見約旦河全平原（10節） |
| 2. 心錯 | 選擇約旦河全平原（11節） |
| 3. 腳錯 | 往東遷移（11節） |
| 4. 漸近罪惡 | 漸漸挪移帳棚，直到[[所多瑪]]（12節） |

- 羅得選擇的比喻：「如同[[耶和華的園子]]，也像[[埃及]]地」——他像回了埃及
- 原文「瑣珥」意為「細小、降低」——名如其地，逐步走下坡
- 所多瑪人在耶和華面前罪大惡極
- 撒但往往先引動人「眼目的情慾」，再引誘犯罪墮落

### 四、神對亞伯蘭的再次應許（14-17節）

- 神在羅得離去後立即重申並擴展應許
- 「凡你所看見的一切地，我都要賜給你和你的後裔，直到永遠」
- 「你的後裔如同地上的塵沙那樣多」——地上塵沙喻屬地子孫，天上眾星喻屬天後裔（創十五5）
- 「你起來，縱橫走遍這地」——**信心有多大，便能取用神的應許有多大**
- 東西加南北構成十字架；縱橫是一豎一橫——**經過十字架處理的，遍地都成屬靈產業**
- 凡為神而蒙受的損失，神必親自補償

### 五、亞伯蘭搬到[[希伯崙]]（18節）

- 希伯崙=同盟、聯合、交通、友誼
- 幔利=能力、剛強、肥沃
- 亞伯蘭在希伯崙幔利的橡樹那裏居住，為耶和華築了一座壇
- 亞伯蘭的築壇之旅：示劍（生命的能力）→ 伯特利（神的家）→ 希伯崙（身體的交通）

### 六、[[信心與世俗的對比]]

| 亞伯蘭 | 羅得 |
|--------|------|
| 讓羅得先選，信靠神 | 隨自己眼光選擇 |
| 放棄選擇權 | 選擇眼前利益 |
| 獲得神重申應許 | 與罪惡之城為鄰 |
| 住在迦南地（應许之地） | 住在所多玛（罪恶之地） |
| 築坛敬拜神 | 漸漸挪移直到罪惡之源 |

### 七、亞伯蘭下埃及與出埃及的再次預表

- 亞伯蘭再次因饑荒下埃及（10-13節的失敗在埃及發生）
- 再次因撒萊的美貌面臨危機
- 再次被法老責遣返
- 神的信實保守到底
