# 創世記 第1章

1. 起初神創造天地。
2. 地是[[空虛混沌的原意|空虛混沌]]，淵面黑暗；[[神的靈的角色|神的靈]]運行在水面上。
3. [[神的話（創造媒介）|神說]]：「要有光」，就有了光。
4. 神看光是好的，就把光暗分開了。
5. 神稱光為晝，稱暗為夜。有晚上，有早晨，這是頭一日。
6. [[神的話（創造媒介）|神說]]：「諸水之間要有空氣，將水分為上下。」
7. 神就造出空氣，將空氣以下的水、空氣以上的水分開了。事就这样成了。
8. 神稱空氣為天。有晚上，有早晨，是第二日。
9. [[神的話（創造媒介）|神說]]：「天下的水要聚在一處，使旱地露出來。」事就这样成了。
10. 神稱旱地為地，稱水的聚處為海。神看著是好的。
11. [[神的話（創造媒介）|神說]]：「地要發生青草和結種子的菜蔬，並結果子的樹木，[[各從其類]]，果子都包著核。」事就这样成了。
12. 於是地發生了青草和結種子的菜蔬，各從其類；並結果子的樹木，各從其類；果子都包著核。神看著是好的。
13. 有晚上，有早晨，是第三日。
14. [[神的話（創造媒介）|神說]]：「天上要有光體，可以分晝夜，作記號，定節令、日子、年歲，
15. 並要發光在天空，普照在地上。」事就这样成了。
16. 於是神造了兩個大光：大的管晝，小的管夜，又造眾星，
17. 就把這些光擺列在天空，普照在地上，
18. 管理晝夜，分別明暗。神看著是好的。
19. 有晚上，有早晨，是第四日。
20. [[神的話（創造媒介）|神說]]：「水要多多滋生有生命的物；要有雀鳥飛在地面以上，天空之中。」
21. 神就造出大魚和水中所滋生各樣有生命的動物，各從其類；又造出各樣飛鳥，各從其類。神看著是好的。
22. 神就賜福給這一切，說：「滋生繁多，充滿海中的水；雀鳥也要多生在地上。」
23. 有晚上，有早晨，是第五日。
24. [[神的話（創造媒介）|神說]]：「地要生出活物來，各從其類；牲畜、昆蟲、地上的野獸，各從其類。」事就这样成了。
25. 於是神造出野獸，各從其類；牲畜，各從其類；地上一切昆蟲，各從其類。神看著是好的。
26. [[神的話（創造媒介）|神說]]：「我們要照著我們的[[形像與樣式|形像]]、按著我們的[[形像與樣式|樣式]]造[[人]]，使他們管理海裡的魚、空中的鳥、地上的牲畜，和全地，並地上所爬的一切昆蟲。」
27. 神就照著自己的形像造人，乃是照著他的形像造男造女。
28. 神就賜福給他們，又對他們說：「要[[生養眾多與治理|生養眾多]]，遍滿地面，[[治理與管理|治理]]這地，也要管理海裡的魚、空中的鳥，和地上各樣行動的活物。」
29. [[神的話（創造媒介）|神說]]：「看哪，我將遍地上一切結種子的菜蔬和一切樹上所結有核的果子全賜給你們作食物。
30. 至於地上的走獸和空中的飛鳥，並各樣爬在地上有生命的物，我將青草賜給它們作食物。」事就这样成了。
31. 神看著一切所造的都[[神看為好（甚好）|甚好]]。有晚上，有早晨，是第六日。

## 本章整理

### 一、創造的三段式結構（1-2節 vs. 3-31節）

1:1-2 記載「起初的創造」與「審判後的光景」；3-31 記載「六日復造」。

- 1:1 的「創造」原文 *bara*，指從無變為有（羅四17；來十一3）
- 1:2 的「地是空虛混沌」原文有「變成了」之意，表示第一、二節之間經過了一段時間，地變荒涼是審判的結果
- 「神的靈運行在水面上」原文含慈愛、覆翼、愛育之意（申卅二11）

### 二、六日復造的次序

| 日 | 經文 | 核心作為 |
|---|---|---|
| 第一日 | 3-5 | 光與暗的分別 |
| 第二日 | 6-8 | 穹蒼與上下的分別 |
| 第三日 | 9-13 | 水與地的分別＋植物 |
| 第四日 | 14-19 | 日、月、星等光體 |
| 第五日 | 20-23 | 水族與飛鳥 |
| 第六日 | 24-31 | 地上動物與人 |

### 三、創造的三個手段

1. **神的靈** — 「神的靈運行」（2節）
2. **神的話** — 「神說」（3、6、9、11、14、20、24、26節）
3. **神的手** — 「神就造出」（7、16、21、25、27節）

### 四、人的創造（26-27節）— 全章核心

- 「我們要照著我們的形像」— 神原文單數、我們為複數，暗示三一神格會議
- **形像** = 裡面的性情（心思、情感、意志）
- **樣式** = 外面的體格
- 神造人的三大目的：彰顯神的形像、執掌神的權柄、對付神的仇敵

### 五、神看為「好」與「甚好」

- 各日皆說「好」，造人之後才說「甚好」— 人是神造物的中心和目的
- 墮落前人類只吃素（29節）；洪水後才准吃肉（創九3）

### 六、六日復造與信徒重生經歷的對應

| 經歷階段 | 屬靈意義 |
|---|---|
| 第一日 | 靈運行＋神的話光照，從黑暗歸向光明 |
| 第二日 | 分別上面的事和下面的事（西三2-3） |
| 第三日 | 與死亡權勢分別，心田適於結生命果子 |
| 第四日 | 守住屬天地位，發光普照黑暗世界 |
| 第五日 | 勝過死亡環境，超脫屬地纏累 |
| 第六日 | 生命成熟，彰顯神形像，代表神掌權 |
