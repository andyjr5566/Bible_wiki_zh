# 希臘（Ἑλλάς）

## 概述

**希臘**在聖經歷史中代表希臘帝國，是但以理書第七章第三獸（豹，有四頭鳥的翅膀，四個頭）的應驗。希臘帝國由**亚历山大大帝**（Alexander the Great, 主前 356-323 年）建立，在短短十餘年間建立了横跨歐、亞、非三洲的龐大帝國。亞歷山大去世後，帝國被他的四將瓜分（四个頭），形成四個王國：**卡山德**（马其顿/希腊）、**利西馬科斯**（色雷斯/小亞细亚）、**塞琉古**（叙利亚/巴比倫/波斯）和**托勒密**（埃及）。希臘時期對猶太人的歷史產生了深遠影響，特別是在宗教、文化和語言方面。

## 聖經出現

### 但以理書中的预言之應驗

- **但以理書 2:39, 40b**：「在你以後另興一國，不及於你的（銀胸臂 = 瑪代波斯）；又有第三國，就是銅腹腰（希臘）。」
- **但以理書 7:6**：「我觀看，又有一獸如豹，背上有鳥的四個翅膀；這獸有四個頭，又得了權柄。」（第三獸 = 希臘帝國及其分裂後的四國）
- **但以理書 8:5-8**：「看見一隻公山羊從西方過來……直撞二千角獸（瑪代波斯）……這山羊極其自高自大，正強盛的時候，那大角折斷了；又在角根上向天的四方長出四個非常的角來。」（亞歷山大及其四將）
- **但以理書 8:21-22**：「那公山羊就是希臘王；兩眼當中的大角就是頭一王（亞歷山大）。至於那折斷了的角，在其根上長出四角，這四角就是四國，只是權勢都不及他（亞歷山大）。」
- **但以理書 11:3-4**：「勇敢的王必興起，執掌大權，隨意而行。他興起的時候，他的國必破裂，向天的四方分開，卻不歸他的後裔，國勢也不及他（亞歷山大）。」

### 馬加比書中的記載（次經）

- **馬加比書上 1:1-10**：詳述亚历山大東征到耶路撒冷的經過，以及希臘文化對猶太人的影響。

### 新約中的希臘

- **約翰福音 7:35**：猶太人議論耶穌是否要往「希臘人」那裡去。
- **使徒行傳 11:20**：門徒首次向外邦人（希臘人）傳福音。
- **使徒行傳 17:16-34**：保羅在雅典（亞略巴古）向希臘人傳福音。
- **使徒行傳 18:1-4, 18-21, 24-28**：保羅在哥林多（希臘重鎮）的事工。
- **使徒行傳 20:1-6**：保羅第三次宣教旅程中在希臘地區的停留。
- **哥林多前書 1:22-24**：「希臘人是求智慧……」
- **加拉太書 3:28**：「你們受洗歸入基督的，並不分猶太人、希臘人……」
- **以弗所書 6:12**：原文「我們並不是與屬血氣的爭戰」涉及希臘化的属灵背景。
- **歌羅西書 3:11**：「在此並不分希臘人、猶太人……」

### 希伯來書中的反思

- **希伯來書 11:34**：「打退了外邦的君王……」——可能指馬加比時期對抗希臘勢力的戰争。

## 神學意義

### 1. 世俗帝國的縮影
希臘帝國代表了世俗文明的巅峰——哲學、藝術、軍事、政治都達到極致。但以理的異象卻指出它不過是世俗帝國中的一個環節，終將過去。

### 2. 文化的污染與保護
希腊化（Hellenization）带來文化的统一，但也威胁了犹太教的独特性。犹太人必須在文化融入和信仰持守之間找到平衡，這是一個历久常新的课题。

### 3. 上帝對萬族的主權
希腊時期，上帝使用希腊文化為預備福音的傳播鋪平了道路（共同語言、法律、道路）。上帝利用異教帝國有益的部分為祂的救贖計劃服務。

### 4. 人的智慧與神的智慧
保羅在林前 1 章直言：「十字架的道理，在那愚拙的為愚拙，在那智慧的卻為無能。」希腊哲學的理性主義（蘇格拉底、柏拉图、亚里士多德）都有其洞見，卻救不了人。

### 5. 逼迫与持守
希腊王安条克四世對猶太人的大逼迫（馬加比書上，下）炼淨了犹太人的信心，也是馬加比革命的導火線。教會歷史中的许多逼迫都是同样的模式——暂時性的苦難带來属灵的复兴。

## 應用與默想

1. **文化與信仰**：我們生活在一個高度全球化的世界，文化多元，信仰面臨同樣的挑戰。如何保持信仰的獨立性，又不過度閉關自守？
2. **上帝在歷史中的掌權**：亞歷山大以為自己征服了世界，但實際上是上帝在用他手中的杖引導歷史。今天世界領袖的每一个决定，不也是在上帝允許之下？
3. **理性與信心**：希臘人求智慧，今天的人也一樣。在知識爆炸的時代，我們更需要的是「十字架的道理」，不是嗎？
4. **逼迫與復興**：安条克四世的逼迫带來馬加比革命。今天在哪些地方，逼迫正在带來教會的复兴？
5. **上帝使用萬事**：上帝使用希腊文化預備了福音传布的土壤（通用希臘語、道路系统、罗马和平）。祂也在使用我生命中看似「不属灵」的事物。
