# 基提

## 概述

基提（Kittim）是聖經中的一個地理名稱，通常指塞浦路斯島及其周邊地區。在[[創世記]]的家譜中，基提是[[雅完]]（Ionia/希臘）的兒子。後來，基提的名稱擴展到泛指希臘和羅馬的西方勢力。

## 聖經出現

- **創世記 10:4** — 「雅完的兒子是以利沙、他施、基提、多單。」
- **民數記 24:24** — 「基提的苦難……」
- **以賽亞書 23:1, 12** — 論推羅的預言中提到基提
- **耶利米書 2:10** — 「你們且過到基提海島察看……」
- **以西結書 27:6** — 用基提的橡樹作船槳
- **但以理書 11:30** — 「基提的戰船必來攻擊他」— 指羅馬勢力

## 地理背景

基提最初指塞浦路斯島（Cyprus），位於地中海東部。塞浦路斯在古代是重要的銅礦產地（「塞浦路斯」的拉丁名 Cyprus 即來自「銅」）。後來，基提的名稱擴展到泛指地中海西部的希臘和羅馬勢力。

## 神學意義

1. **西方勢力的代表**：基提從塞浦路斯擴展到泛指西方列強
2. **預言的應驗**：但以理書11:30的基提與羅馬的興起相關
3. **神的計劃**：基提的興起是神預言計劃的一部分
4. **地理的擴展**：從具體的島嶼到廣泛的西方世界
