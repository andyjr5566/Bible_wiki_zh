# 申命記 第17章

1. [[有殘疾或惡病的牛羊不可獻給耶和華（潔淨祭物）|凡有殘疾，或有什麼惡病的牛羊]]，你都不可獻給耶和華─你的神，因為這是耶和華─你神所憎惡的。
2. 在你們中間，在耶和華─你神所賜你的諸城中，無論哪座城裡，若有人，或男或女，行耶和華─你神眼中看為惡的事，違背了他的約，
3. [[拜偶像者必被治死（清除惡人）|去事奉敬拜別神]]，或拜日頭，或拜月亮，或拜天象，是主不曾吩咐的；
4. 有人告訴你，你也聽見了，就要細細地探聽，果然是真，準有這可憎惡的事行在以色列中，
5. 你就要將行這惡事的男人或女人拉到城門外，[[拜偶像者必被治死（清除惡人）|用石頭將他打死]]。
6. 要[[拜偶像者必被治死（清除惡人）|憑兩三個人的口作見證]]將那當死的人治死；不可憑一個人的口作見證將他治死。
7. [[拜偶像者必被治死（清除惡人）|見證人要先下手]]，然後眾民也下手將他治死。這樣，就[[拜偶像者必被治死（清除惡人）|把那惡從你們中間除掉]]。
8. 你城中若起了爭訟的事，或因流血，或因爭競，或因毆打，是你[[難斷的案件要上到耶和華揀選的地方（中央法庭）|難斷的案件]]，你就當起來，往耶和華─你神所選擇的地方
9. 去見祭司利未人，並當時的審判官，求問他們，他們必將判語指示你。
10. 他們在耶和華所選擇的地方指示你的判語，你必照著他們所指教你的一切話謹守遵行。
11. 要按他們所指教你的律法，照他們所斷定的去行；他們所指示你的判語，你[[難斷的案件要上到耶和華揀選的地方（中央法庭）|不可偏離左右]]。
12. 若有人擅敢不聽從那侍立在耶和華─你神面前的祭司，或不聽從審判官，那人就必治死；這樣，便將那惡從以色列中除掉。
13. 眾百姓都要聽見害怕，不再擅敢行事。
14. 到了耶和華─你神所賜你的地，得了那地居住的時候，若說：[[立王的條例（王的條例）|我要立王治理我，像四圍的國一樣]]。
15. 你總要立耶和華─你神所揀選的人為王。必從你弟兄中立一人；不可立你弟兄以外的人為王。
16. 只是[[立王的條例（王的條例）|王不可為自己加添馬匹]]，也不可使百姓回埃及去，為要加添他的馬匹，因耶和華曾吩咐你們說：不可再回那條路去。
17. 他[[立王的條例（王的條例）|也不可為自己多立妃嬪]]，恐怕他的心偏邪；[[立王的條例（王的條例）|也不可為自己多積金銀]]。
18. [[立王的條例（王的條例）|他登了國位，就要將祭司利未人面前的這律法書，為自己抄錄一本]]，
19. 存在他那裡，要平生誦讀，好學習敬畏耶和華─他的神，謹守遵行這律法書上的一切言語和這些律例，
20. [[立王的條例（王的條例）|免得他向弟兄心高氣傲，偏左偏右，離了這誡命]]。這樣，他和他的子孫便可在以色列中，在國位上年長日久。

## 本章整理

### 潔淨祭物與純全獻祭（v1）
本章開篇規定 **[[有殘疾或惡病的牛羊不可獻給耶和華（潔淨祭物）|有殘疾或惡病的牛羊]]** 不可獻給耶和華，因為這是神所憎惡的。這延續申命記 15:21 關於頭生牲畜不可有殘疾的原則，強調獻祭必須純全，反映獻祭者對神榮耀的重視程度。獻祭不是形式上的交易，而是內心敬畏的外在表達。

### 拜偶像者必被治死：清除內部的惡（v2-7）
經文轉向處理社群內部的屬靈背叛。若有人 **[[拜偶像者必被治死（清除惡人）|事奉敬拜別神]]**（拜日頭、月亮、天象），經過 **細細探聽** 確認屬實，須拉到城門外用石頭打死。此程序嚴謹：
1. **雙重見證**：憑兩三個見證人定案（v6），防止誣告。
2. **見證人先下手**：見證人須親自執行刑罰（v7），增加作見證的嚴肅性與責任感。
3. **眾民同參與**：全體會眾參與執行，象徵社群共同切斷與惡的連結，**「把那惡從你們中間除掉」**（v7, 12, 13 等處重複出現，為申命記核心神學動機）。

### 難斷案件上訴中央法庭（v8-13）
地方法庭無法判決的案件（流血、爭競、毆打），須上呈 **[[難斷的案件要上到耶和華揀選的地方（中央法庭）|耶和華揀選的地方]]**，由 **祭司利未人** 與 **當時的審判官** 判斷。其判語具終局效力，**不可偏離左右**（v11）。抗拒權威者必治死（v12），目的是叫眾民聽見害怕，不再擅敢行事（v13）。這建立了以色列司法體系的 **最高上訴機制**，權威源於神的同在（耶和華所選之處）。

**上訴流程摘要**：
1. 地方城門口審判 → 2. 遇難斷案件 → 3. 上呈耶和華揀選之處 → 4. 祭司利未人與審判官查詢 → 5. 指示判語 → 6. 百姓謹守遵行（不可偏左右） → 7. 擅敢不聽者治死 → 8. 眾民聽見害怕。

### 立王的條例：受約法約束的君主（v14-20）
預見以色列進入迦南後要求立王（如四圍列國），神訂立 **[[立王的條例（王的條例）|王的條例]]**，將王權納入約法框架，區別於古近東絕對君主制：

| 限制項目 | 經文依據 | 神學意涵 |
| :--- | :--- | :--- |
| 人選限制 | 必從弟兄中選，非外邦人 (v15) | 保障約群體純粹性，王是「弟兄」而非主宰 |
| 軍事限制 | 不可多馬匹、不可使民回埃及 (v16) | 禁止依靠軍事力量（馬匹）與回頭尋求埃及幫助，單靠耶和華得勝 |
| 家庭限制 | 不可多立妃嬪 (v17a) | 防止透過聯姻結盟導致心偏邪、拜偶像 |
| 財富限制 | 不可多積金銀 (v17b) | 防止財富自誇，忘記神是賜福的源頭 |
| 屬靈操練 | 抄寫律法、平生誦讀、學習敬畏、謹守遵行 (v18-19) | 王是律法的僕人，而非律法的主人；以神的話語治理內心與國家 |
| 態度限制 | 免得心高氣傲、偏左偏右 (v20) | 確保王權傳承長久（他和子孫在國位上年長日久） |

### 跨章脈絡：祭物、審判、王權的內在連貫
本章三大單元（祭物、審判、王權）共同勾勒約群體聖潔秩序的輪廓。**潔淨祭物** 奠定敬拜純粹性；**中央法庭** 確保公義判斷權威來自神的同在；**王的條例** 將最高統治者置於律法之下，以抄寫誦讀律法為核心職分。三者同指向一個中心：以色列的生存秩序不在於儀式形式、人治權術或軍事財力，而在於全人全社群對耶和華話語的絕對順服與敬畏。這為後續先知書審判君王、祭司、假先知的神學基準，也為新約彌賽亞作真祭物、真審判官、真王的職分預備了類型輪廓。
