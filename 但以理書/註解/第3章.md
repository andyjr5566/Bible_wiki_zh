# 但以理書 第3章 註解

## 一、章節綱要
本章記載尼布甲尼撒王造金像、全民敬拜的命令，以及三位忠信少年沙得拉、米煞、亞伯尼歌因拒絕 idolatrous worship 被投入烈火窖，卻見神奇保全的事件。全章可分為五段：
1. 金像的造立與敬拜命令（1‑7節）
2. 迦底人控告三友不敬拜金像（8‑12節）
3. 王的最後通牒與三友的信心見證（13‑18節）
4. 三友被投入火窖、神的 miraculous 保護（19‑27節）
5. 王的讚美、保護信仰的詔令與三友的晉升（28‑30節）

## 二、逐節重點摘錄

**1‑7節**：王造高六十肘、寬六肘的金像，立於巴比倫省杜拉平原。命令所有官員聽見號角、笛、琵琶、琴、瑟、笙等樂器聲時，必須俯伏敬拜金像；不敬拜者立刻投入烈火窖。這顯示王試圖以統一的宗教儀式來鞏固政治權力，消除宗教多樣性。

**8‑12節**：某些迦底人（即巴比倫的術士、占星家）出來告發猶太人，特別點名王所派管理巴比倫省的沙得拉、米煞、亞伯尼歌不事奉王的神、也不敬拜金像。他們的動機是嫉妒猶太人得升官，也想激發王的怒氣。

**13‑15節**：王大怒，將三人帶到跟前，詰問他們是否故意不敬拜。王給他們一次機會：若聽見樂器聲仍願意敬拜金像，可免罪；若仍不敬拜，必立即被投入火窖。王並質問：「有何神能救你們脫離我的手？」

**16‑18節**：三人回答：第一，他們不必向王解釋此事（因其行為並非違法，只是不願犯偶像拜）；第二，即使神不將他們從火窖中救出，他們也決不事奉王的神或敬拜金像。此乃「即或不然」的信心宣告——忠心不依賴於得救的結果。

**19‑23節**：王怒火中燒，命將火窖加熱七倍，並派精銳士兵將三人捆綁扔入火中。抬他們上去的士兵被火焰燒死，但三人卻在火中行走，髮不焦、衣不變色，甚至沒有火燎氣味。此景象顯出神的超自然保護。

**24‑27節**：王驚訝地站起來，見四人同行（未被捆綁），其中第四個「形狀好像神子」。他隨即呼喚三人出火窖，三人安然 uscire。王見證後，隨即宣告神的偉大。

**28‑30節**：王讚美沙得拉、米煞、亞伯尼歌的神是應當頌讚的，因祂派遣使者救護倚靠祂的僕人，他們違背王命、犧牲自己，仍不事奉別神。王又下詔：凡毀謗此三人之神者必被凌遲，房屋變糞堆；因無其他神能如此施拯救。最後，王在巴比倫省提升三人的官職。

## 三、神學主題

1. **神對王權的最高主權**（但三20‑25）：即使在極端的政治壓力下，神仍能保護祂的僕人，顯示祂超越一切世俗權勢。
2. **信心的堅定與「即或不然」的見證**（17‑18節）：真正的信心不僅在得讚美時顯現，更在可能犧牲時仍不改變對神的敬拜。
3. **神的同在與拯救**（24‑27節）：火中的第四人象徵神與苦難中的信徒同在；祂的拯救不僅是肉體的解脫，更是對祂信實與大能的宣告。
4. **見證帶來的轉化**（28‑30節）：外邦王從強迫拜偶像轉為讚美真神、保護信仰，顯明神的作用能改變甚至最硬硬的心。
5. **神的國度最終勝過一切人權勢**（暗示與第二章的石頭相連）：火中得救的三人預表神所立的永不敗壞之國度，最終要摧毀一切人間金像與帝國。
