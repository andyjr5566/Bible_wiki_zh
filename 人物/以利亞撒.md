# 以利亞撒

## 概述

以利亞撒（Eleazar）是[[亞倫]]的第三個兒子，在亞倫死後繼承了大祭司的職分。以利亞撒在以色列人進入迦南地之前和之後都擔任大祭司，是祭司傳統的重要傳承者。

## 聖經出現

- **出埃及記 6:23** — 以利亞撒被列為亞倫的兒子
- **出埃及記 28:1** — 亞倫和他的兒子們被召
- **民數記 3:2-4** — 以利亞撒繼承大祭司
- **民數記 4:16** — 以利亞撒負責聖所物品的管理
- **民數記 20:25-28** — 亞倫將以利亞撒穿上大祭司的衣袍
- **民數記 26:1-3, 63** — 以利亞撒參與數點百姓
- **民數記 31:21-31** — 以利亞撒處理戰利品
- **約書亞記 14:1** — 以利亞撒參與分配土地
- **約書亞記 19:51; 21:1** — 以利亞撒在分地中的角色

## 神學意義

1. **祭司傳承**：以利亞撒繼承父親的祭司職分，確保祭司傳統的延續
2. **聖潔的管理**：以利亞撒負責管理聖所的神聖物品
3. **信心的傳承**：從亞倫到以利亞撒，信仰的火炬代代相傳
4. **神的信實**：祭司制度的延續彰顯神的信實
