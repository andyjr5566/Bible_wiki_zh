# 基提

## 概述

基提（Kittim）是聖經中的一個民族和地理名稱，通常指塞浦路斯島及其居民，後來也泛指希臘和羅馬的西方勢力。在但以理書中，基提與[[羅馬]]的興起有關（但以理書 11:30）。

## 聖經出現

- **創世記 10:4** — 「雅完的兒子是以利沙、他施、基提、多單。」基提是雅完（希臘）的兒子
- **以賽亞書 23:1, 12** — 論推羅的預言中提到基提
- **耶利米書 2:10** — 「你們且過到基提海島察看……」
- **以西結書 27:6** — 用基提的橡樹作船槳
- **但以理書 11:30** — 「他必回往，惱恨聖約，任意而行。他必回來聯絡背棄聖約的人。」（「基提」在此指羅馬勢力）

## 神學意義

1. **西方勢力的代表**：基提從塞浦路斯擴展到泛指西方列強
2. **預言的應驗**：但以理書11:30的基提與羅馬的興起相關
3. **神的計劃**：基提的興起是神預言計劃的一部分
4. **地理的擴展**：從具體的島嶼到廣泛的西方世界
