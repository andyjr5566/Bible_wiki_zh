# 亞歷山大

## 概述

亞歷山大大帝（Alexander the Great，前356年—前323年），馬其頓國王，古代世界最偉大的軍事征服者之一。他建立了橫跨歐亞非的龐大帝國，將希臘文化傳播到已知世界（希臘化時代）。在但以理書中，亞歷山大的帝國是[[四獸]]異象中第三獸（豹，四個翅膀）和[[公山羊]]的歷史應驗。

## 聖經出現

### 舊約預言中的亞歷山大

- **但以理書 7:6** — 「此後我觀看，又有一獸如豹，背上有鳥的四個翅膀……」
- **但以理書 8:5-8, 21-22** — 「公山羊……遍行全地，腳不沾塵。……那公山羊就是希臘王；兩眼中的大頭就是他的頭一王。至於那折斷了的角，在其根上又長出四角，就是指這國必興起四國……」

### 次經中的亞歷山大

- **瑪加比一書 1:1-7** — 簡述亞歷山大的征服和帝國分裂

## 歷史背景

亞歷山大在前334年開始東征，短短十年內征服了波斯帝國全境，遠達印度。前323年，年僅32歲的亞歷山大在[[巴比倫]]病逝。他的帝國隨後分裂為四個王國，由四位部將統治：[[托勒密]]（埃及）、[[塞琉古]]（敘利亞/波斯）、[[利西馬科斯]]（色雷斯）、[[卡山得]]（馬其頓）。

## 神學意義

1. **預言的精確應驗**：亞歷山大的迅速征服（「腳不沾塵」）精確應驗了但以理的預言
2. **神的主權**：帝國的興起和分裂彰顯神的預定
3. **希臘化的影響**：亞歷山大的征服為新約時代的希臘語言和文化奠定了基礎
4. **帝國的分裂**：四角的預言精確對應四大將的王國
5. **榮美之地**：亞歷山大經過以色列地而沒有毀滅它，應驗了預言

亞歷山大的故事展示了即使是最強大的帝國也在神的手中。人類的野心和榮耀是短暫的，唯有神的國度永遠長存。亞歷山大的早逝（32歲）提醒我們生命的短暫和神的至高主權。
