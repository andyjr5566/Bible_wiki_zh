# 尼希米

## 概述

[[尼希米]]（Nehemiah）是[[波斯]]王[[亞達薛西]]一世的酒政（斟酒官），後來被任命為猶大省長，負責重建[[耶路撒冷]]的城牆。他大約在主前445年帶領第三批以色列百姓歸回耶路撒冷，並在極其困難的條件下，僅用52天就完成了城牆的重建工程（[[尼希米記]] 6:15）。

尼希米記是舊約中少數以第一人稱記述的歷史書卷，記錄了尼希米的回憶錄。他不僅是一位出色的行政領袖，更是一位禱告者、改革者和信心的榜樣。他的事奉與[[以斯拉]]的事奉相輔相成——以斯拉專注於宗教和律法的復興，尼希米則專注於城牆重建和社會改革。

- [[尼希米記]] 1:1-11 — 尼希米得知耶路撒冷城牆被毀、百姓遭難，痛哭禱告

- [[尼希米記]] 2:9-20 — 尼希米夜間巡視城牆，號召百姓重建
- [[尼希米記]] 3:1-32 — 詳細記錄各家族、各群體負責重建的區段
- [[尼希米記]] 4:1-23 — 面對參巴拉等人的嘲笑和威脅，尼希米帶領百姓一手做工、一手拿兵器

- [[尼希米記]] 6:1-14 — 敵人多次設謀陷害尼希米，但他識破詭計
- [[尼希米記]] 6:15-16 — 城牆52天完工，仇敵「便懼怕，因為這工作完成是出乎我們的上帝」
- [[尼希米記]] 7:1-73 — 城牆完工後的人口普查和安置
- [[尼希米記]] 8:1-18 — 以斯拉宣讀律法書，百姓守住棚節
- [[尼希米記]] 9:1-38 — 百姓聚集認罪禱告
- [[尼希米記]] 10:1-39 — 百姓簽約遵守上帝的律法
- [[尼希米記]] 11-12章 — 百姓抽籤住進耶路撒冷，城牆奉獻禮

## 神學意義

1. **禱告與行動的結合**：尼希米是禱告的典範。他在採取任何行動之前，總是先禱告（[[尼希米記]] 1:4-11, 2:4, 4:4-5, 4:9）。他的禱告不是被動的等待，而是積極尋求上帝引導後的勇敢行動。

2. **屬靈領袖的品格**：尼希米展現了多項領袖特質：對百姓苦難的同理心（1:4）、在王面前的勇氣（2:2-5）、面對威脅時的堅定（4:14）、處理內部不公的正直（5:14-15）、識破詭計的智慧（6:2-4）。

3. **一手做工、一手拿兵器**：[[尼希米記]] 4:17記載重建者「一手做工、一手拿兵器」。這提醒我們，上帝的工作不僅需要屬靈的熱情，也需要實際的智慧和預備。

4. **社會公義的實踐**：尼希米不僅重建城牆，也處理了富人剝削窮人的問題（第5章）。他要求省長們免除百姓的債務，自己也不索取省長的俸祿。這表明上帝的國度關注的不僅是宗教儀式，也包括社會公義。

5. **城牆的屬靈意義**：耶路撒冷城牆不僅是物理的防禦，更是上帝百姓身分的象徵。城牆的重建代表上帝百姓的恢復、分別為聖和上帝同在的保護。

## 應用與默想

1. 尼希米在採取行動之前總是先禱告。你的「禱告生活」和「行動生活」是否平衡？你是否在重大決定之前，花時間安靜在上帝面前？

2. 尼希米面對內憂外患（內部經濟不公、外部敵人威脅）仍然堅持完成工作。你是否也曾在「四面受敵」的情況下，仍然堅持上帝託付給你的使命？

3. 尼希米不僅重建城牆，也處理社會不公。你是否也關注身邊的「不義」？你是否願意為弱勢群體發聲？
