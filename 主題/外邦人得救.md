# 外邦人得救

## 概述

外邦人得救是新約神學的核心主題之一，保羅大量引用何西阿書 1:10、2:23 作為神學依據，論證神的救恩計劃超越以色列民族，擴展到萬國萬民。

## 聖經出現

- **何西阿 1:10** — 「將來在那裡必對他們說『你們是永生神的兒子』」
- **何西阿 2:23** — 「本非我民的，我必對他說：你是我的民」
- **羅馬書 9:24-26** — 保羅引用何西阿 1:10、2:23 論述外邦人蒙召
- **羅馬書 10:12-13** — 「猶太人和希利尼人並沒有分別……凡求告主名的，就必得救」
- **羅馬書 11:11-32** — 外邦人得救激發以色列回心轉意
- **以弗所 2:11-22** — 外邦人在基督裡成為「神家裡的人」
- **加拉太 3:28** — 「不再分猶太人、希利尼人……在基督耶穌裡都成為一了」
- **啟示錄 7:9** — 「各國、各族、各方、各民，沒有人能數過來」

## 神學意義

1. **何西阿的原意**：以色列復興的應許
2. **保羅的釋經學**：以色列的經文應用在外邦人，顯明救恩計劃的普世性
3. **救恩歷程**：猶太人 → 外邦人 → 激發猶太人回心 → 以色列全家得救
4. **教會的身分**：外邦信徒與猶太信徒在基督裡合一，同作「神的子民」
4. **終極盼望**：羅 11:26 「這樣，以色列全家都要得救」
