# 耶戶王朝

## 概述

耶戶王朝（主前 842-752）是北國以色列歷史上最長的王朝，始於耶戶推翻亞哈家，終於撒迦利雅被沙龍弒殺。何西阿 1:4 預言神必討耶戶家在耶斯列殺人流血的罪。

## 歷史背景

- **建立者**：耶戶（主前 842-815），以利沙膏立，奉命剷除亞哈家
- **暴力建立**：耶戶在耶斯列殺亞哈七十子、亞哈謝王、亞哈家所有大臣祭司（王下 9-10）
- **王朝傳承**：耶戶 → 約阿施 → 耶羅波安二世 → 撒迦利雅
- **終結**：撒迦利雅在位六個月被沙龍弒殺（王下 15:8-10），王朝終結
- **耶羅波安二世時期**：政治經濟頂峰，但屬靈極度敗壞（何西阿開始事奉）

## 聖經出現

- **何西阿 1:4-5** — 「給他起名叫耶斯列；因為再過片時，我必討耶戶家在耶斯列殺人流血的罪」
- **列王紀下 9-10** — 耶戶篡位與清洗亞哈家
- **列王紀下 13:1-3** — 約阿施事奉耶和華不盡心
- **列王紀下 14:23-29** — 耶羅波安二世復興國境
- **列王紀下 15:8-12** — 撒迦利雅被弒，耶戶王朝終結
- **何西阿 1:4** — 「討耶戶家在耶斯列殺人流血的罪」

## 神學意義

1. **神使用耶戶審判亞哈家**，但耶戶過度殺戮超出授權
2. **暴力建立的政權必以暴力終結**：耶戶王朝以血開始，以血結束
3. **神的審判跨代**：「討耶戶家的罪」延遲幾代才成就
4. **何西阿的見證**：先知在耶戶王朝末期事奉，見證其滅亡
