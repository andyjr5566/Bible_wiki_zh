# 曠野

## 概述

曠野（Wilderness）在聖經中是一個重要的神學概念。它不僅是地理上的沙漠，更是神試煉、更新、與子民建立親密關係的地方。

## 聖經出現

- **何西阿書 2:3** — 「使她如曠野，如乾旱之地，因渴而死」
- **何西阿書 2:14** — 「領她到曠野，對她說安慰的話」
- **何西阿書 9:10** — 「在曠野遇見以色列」
- **何西阿書 13:5** — 「在曠野認識你」
- **何西阿書 13:15** — 在乾旱之地
- **出埃及記 16-17** — 以色列在曠野的經歷
- **申命記 8:2-5** — 曠野的試煉
- **耶利米書 2:2-6** — 曠野中的愛情
- **以西結書 20:10-26** — 曠野中的律法
- **馬太福音 4:1-11** — 耶穌在曠野受試探

## 神學意義

1. **試煉與管教**：神在曠野試煉祂的子民
2. **更新與恢復**：神在曠野重新建立關係
3. **親密關係**：曠野是與神最親密的地方
4. **依靠**：在曠野中，人必須完全依靠神

## 應用與默想

曠野不是咒诅，而是神用來更新我們關係的場所。當我們感到「乾旱」時，可能是神要帶領我們回到祂身邊。
