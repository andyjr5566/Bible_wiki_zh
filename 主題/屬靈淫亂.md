# 屬靈淫亂

## 概述

屬靈淫亂是聖經中用來描述以色列人對神不忠、轉拜偶像的隱喻性語言。神視以色列為妻子，以色列的偶像崇拜如同妻子對丈夫不貞。

## 原文意義

- 希伯來文「淫亂」為複數形式，指反覆不斷、根深蒂固的淫亂
- 可指肉體性行為或**宗教上的不忠**（拜偶像）

## 聖經出現

- **何西阿書 1:2** — 「這地大行淫亂，離棄耶和華」
- **何西阿書 2:2-13** — 神對屬靈淫亂妻子的責備
- **何西阿書 4:13-14** — 祭司和先知行淫
- **以西結書 16:1-43** — 耶路撒冷的不忠
- **以西結書 23:1-49** — 阿荷拉與阿荷利巴的比喻
- **耶利米書 3:1-10** — 以色列的淫亂

## 神學意義

1. **婚姻作為盟約隱喻**：神與以色列的關係如同婚姻，要求絕對的忠貞
2. **拜偶像 = 姦淫**：將神獨得的榮耀歸與被造之物，在神眼中如同妻子背夫別戀
3. **神的聖潔與慈愛張力**：聖潔的神使用不潔的婚姻為象徵，旨在揭示以色列的背約
4. **悔改與復興**：即使以色列行屬靈淫亂，神仍應許挽回

## 新約延伸

- **雅各書 4:4** — 「你們這些淫亂的人哪，豈不知與世俗為友就是與神為敵嗎？」
- **啟示錄 17:1-5** — 「大巴比倫，作世上的淫婦」

## 應用與默想

屬靈淫亂提醒我們，任何在神之外追求的事物（金錢、權力、關係）都可能成為我們的「偶像」。神要求我們專一地愛祂。
