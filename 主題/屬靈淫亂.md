# 屬靈淫亂

## 概述

屬靈淫亂是聖經中用來描述神的子民對神不忠、轉而敬拜偶像的核心隱喻。神視祂的子民為妻子/新婦，子民拜偶像如同妻子行淫。

## 名字字義

- **淫亂**：希伯來文 *zānâ*（זָנָה），指賣淫、行淫，引申為屬靈上的不忠
- **屬靈**：指與神關係層面，非肉體行為

## 聖經出現

- **何西阿書 1:2** — 「這地大行淫亂，離棄耶和華」
- **何西阿書 2:2-13** — 神對屬靈淫亂妻子的責備
- **何西阿書 4:12-13** — 百姓求問木偶、在橡樹下獻祭
- **何西阿書 5:3-4** — 「以法蓮行淫了，以色列被玷污了」「淫心在他們裡面」
- **以西結書 16:1-43** — 耶路撒冷的不忠
- **以西結書 23:1-49** — 阿荷拉與阿荷利巴
- **耶利米書 3:1-10** — 以色列的淫亂
- **啟示錄 17:1-5** — 大巴比倫，作世上的淫婦

## 神學意義

1. **婚姻作為盟約隱喻**：神與子民的關係如同婚姻，要求專一忠貞
2. **拜偶像 = 行淫**：將神獨得的榮耀歸給被造之物，在神眼中如同妻子背夫別戀
3. **神的忌邪**：神是忌邪的神（出 20:5），不可容忍子民分心
4. **審判與復興**：神的管教目的是挽回，非毀滅（何 2:14-23）

## 應用與默想

屬靈淫亂提醒我們，任何在神之外追求的事物（金錢、權力、關係、事業）都可能成為我們的「偶像」。神要求我們專一地愛祂。
