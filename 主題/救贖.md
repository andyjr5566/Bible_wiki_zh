# 救贖

## 概述

「救贖」（Redemption）是聖經中最核心的神學概念之一，指上帝透過付上代價，將人從罪惡、死亡和捆綁中釋放出來。在舊約中，救贖的觀念根植於上帝將以色列民從埃及為奴之家領出來的「出埃及」事件；在新約中，救贖在耶穌基督的十字架上得到最完全的成就。

但以理書 9:24記載了「七十個七」預言的目標之一：「贖盡罪孽」（to atone for iniquity），直接將救贖與彌賽亞的工作聯繫在一起。整本聖經的敘事——從創世記到啟示錄——可以被視為上帝救贖計畫的展開。

## 聖經出現

### 舊約中的救贖

- [[出埃及記]] 6:6 — 上帝對以色列說：「我要用伸出來的膀臂重重地刑罰埃及人，救贖你們脫離他們的重擔。」
- [[出埃及記]] 15:13 — 摩西之歌：「你憑慈愛領了你所贖的百姓。」
- [[利未記]] 25:25-28, 47-49 — 親屬救贖者（Go'el）的制度：至近的親屬有責任贖回弟兄的產業或自由
- [[利未記]] 16章 — 贖罪日的儀式：大祭司為百姓贖罪
- [[民數記]] 18:15-17 — 頭生的救贖
- [[申命記]] 7:8 — 「只因耶和華愛你們……所以用大能的手領你們出來，從為奴之家救贖你們。」
- [[詩篇]] 130:7-8 — 「耶和華有豐盛的慈愛，他有豐盛的救贖。他必救贖以色列脫離一切的罪孽。」
- [[以賽亞書]] 43:1 — 「你不要害怕！因為我救贖了你。我曾提你的名召你，你是屬我的。」
- [[以賽亞書]] 44:22-23 — 「我塗抹了你的過犯，像厚雲消散；我塗抹了你的罪惡，如薄雲滅沒。你當歸向我，因我救贖了你。」
- [[以賽亞書]] 53:10-12 — 受苦的僕人：「他必看見後裔，並且延長年日，耶和華所喜悅的事必在他手中。他必看見自己勞苦的功效，便心滿意足。有許多人因認識我的義僕得稱為義，並且他要擔當他們的罪孽。」
- [[耶利米書]] 31:11 — 「因耶和華救贖了雅各，救贖他脫離比他更強之人的手。」
- [[何西阿書]] 13:14 — 「我必救贖他們脫離陰間，救贖他們脫離死亡。死亡啊，你的災害在哪裡？陰間哪，你的毀滅在哪裡？」
- [[彌迦書]] 4:10 — 「錫安的女子啊，你要疼痛劬勞，彷彿臨盆的婦人，因為你必從城裡出來……必被帶到巴比倫，在那裡蒙救贖。」
- 但以理書 9:24 — 「要止住罪過，除淨罪惡，贖盡罪孽，引進永義」

### 新約中的救贖

- [[路加福音]] 1:68 — 撒迦利亞頌：「主以色列的上帝是應當稱頌的，因他眷顧他的百姓，為他們施行救贖。」
- [[路加福音]] 2:38 — 女先知亞拿將耶穌的事「講論給一切盼望耶路撒冷得救贖的人聽」。
- [[路加福音]] 24:21 — 以馬忤斯的門徒說：「但我們素來所盼望、要贖以色列民的就是他。」
- [[羅馬書]] 3:24 — 「如今卻蒙上帝的恩典，因基督耶穌的救贖，就白白地稱義。」
- [[以弗所書]] 1:7 — 「我們藉這愛子的血得蒙救贖，過犯得以赦免，乃是照他豐富的恩典。」
- [[歌羅西書]] 1:14 — 「我們在愛子裡得蒙救贖，罪過得以赦免。」
- [[提多書]] 2:14 — 「他為我們捨了自己，要贖我們脫離一切罪惡，又潔淨我們，特作自己的子民，熱心為善。」
- [[希伯來書]] 9:12 — 「並且不用山羊和牛犢的血，乃用自己的血，只一次進入聖所，成了永遠贖罪的事。」
- [[彼得前書]] 1:18-19 — 「知道你們得贖……不是憑著能壞的金銀等物，乃是憑著基督的寶血，如同無瑕疵、無玷污的羔羊之血。」
- [[啟示錄]] 5:9 — 「他們唱新歌，說：你配拿書卷，配揭開七印，因為你曾被殺，用自己的血從各族、各方、各民、各國中買了人來，叫他們歸於上帝。」

## 神學意義

1. **代價的付出**：救贖的核心是「付上代價」。在舊約中，上帝用大能的手將以色列從埃及贖出；在新約中，基督用祂的寶血作為贖價（[[馬可福音]] 10:45：「人子來……要捨命，作多人的贖價」）。救贖不是免費的——對上帝而言，代價是祂兒子的生命。

2. **親屬救贖者（Go'el）的預表**：舊約中的「至近親屬救贖者」制度（[[利未記]] 25章，[[路得記]] 4章）預表了基督的工作：祂成為我們的「弟兄」（[[希伯來書]] 2:11），有權利也有能力贖回我們。正如波阿斯贖回路得和拿俄米的產業，基督贖回我們脫離罪的捆綁。

3. **出埃及與十字架**：出舊約的「出埃及」事件是新約「十字架救贖」的預表。正如逾越節的羔羊拯救以色列長子免於死亡（[[出埃及記]] 12章），基督作為「上帝的羔羊」（[[約翰福音]] 1:29）拯救一切信祂的人免於永死。

4. **救贖的完全性**：但以理書 9:24列出救贖的六個目標：止住罪過、除淨罪惡、贖盡罪孽、引進永義、封住異象和預言、膏至聖者。這表明基督的救贖工作不僅是「遮蓋」罪，更是徹底地「除去」罪，並引進永遠的公義。

5. **救贖的末世向度**：救贖在基督的十字架上已經成就，但尚未完全實現。信徒目前「等候得著兒子的名分，乃是我們身體的贖罪」（[[羅馬書]] 8:23）。最終的救贖將在基督再來時完成——新天新地、身體復活、永遠與上帝同在。

## 應用與默想

1. 彼得前書1:18-19告訴我們，我們得贖不是憑著金銀，而是憑著基督的寶血。這對你理解自己的「價值」有什麼影響？你是否真正相信上帝為你付上了最大的代價？

2. 舊約的以色列民每年記念出埃及的救贖。你是否也定期記念和感恩基督在你生命中的救贖？你的「逾越節」是什麼？

3. 救贖不僅是脫離罪的刑罰，更是「特作自己的子民，熱心為善」（[[提多書]] 2:14）。你的救贖是否帶來了生活方式的改變？你是否活出「被贖之人」的生命？
