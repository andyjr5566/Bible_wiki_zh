# 瑪加比

## 概述

「瑪加比」（Maccabee）是[[哈斯蒙尼]]家族起義領袖[[猶大]]的稱號，意為「錘子」。後來泛指整個起義家族和運動。瑪加比起義（前167年—前160年）是猶太人反抗[[塞琉古帝國]]統治和希臘化政策的武裝起義。

## 聖經出現

### 次經中的瑪加比

- **瑪加比一書 2:1-6** — 瑪他提亞和他的五個兒子
- **瑪加比一書 2:42-48** — 猶大·瑪加比接續父親領導起義
- **瑪加比一書 3:1-9** — 猶大·瑪加比的軍事勝利
- **瑪加比一書 4:36-59** — 潔淨聖殿，修殿節的起源
- **瑪加比二書 8-10章** — 起義的詳細記錄

### 歷史關聯

- **但以理書 8:23-25; 11:32-35** — 預言安条克四世和隨後的起義

## 神學意義

1. **信仰的堅持**：瑪加比家族為信仰自由而戰
2. **神的拯救**：神通過少數人拯救了祂的子民
3. **修殿節**：潔淨聖殿後設立的節期（約翰福音 10:22）
4. **預言的應驗**：但以理書的預言在瑪加比時期得到應驗
