# 如野獸生活

## 聖經參考
- 但以理書 4:25, 32-33
- 但以理書 4:34-37

## 概述
但以理書第四章記載尼布甲尼撒王因極度驕傲而受神特別的管教——他被奪去理性，如野獸般生活了七時（被廣泛理解為七年）。期間，「他被趕出離開世人，與野地的獸同居，喫草如牛，被天露滴濕，頭髮長長，好像鷹毛；指甲長長，如同鳥爪」（v.33）。這種生活狀態在醫學上可能稱為「博人癣」（lycanthropy或boanthropy），即人認為自己是獸並表現出相應行為的精神症狀。

這段經歷並非單純的身體疾病，而是神有目的的靈性管教。王失去了所有使人成為人的尊嚴：理性、社交禮節、個人衛生與尊貴生活方式。他被減至僅維持基本生存的狀態，以便學住最基本的功課——承認「至高者在人的國中掌權」（v.25）。

值得注意的是，這樣的待遇並非毫無保護。經文提到神在管教中仍施恩：留下樹墩（暗示復位的可能）、提供天露為飲水、使他得與野獸共處而不受傷害。這顯示即使在最嚴厲的 disciplinary 中，神的愛與慈悲仍然存在。

## 神學意義
1. **人性的真相**：當人失去神所賜的理性與道德約束時，真正的「人性」會如何？這段經歷顯示，無神論的人生最終會墮落為純粹的生存本能，失去使人區別於禽獸的理性與道德。
2. **神的公義與慈愛並存**：雖然管教極其嚴厲（失去王位、人格尊嚴、正常生活），但神仍在其中顧念保護祂的兒子——這正是管教的本質：不是毀滅，而是救贖。
3. **認識神的途徑**：有時人必須經歷極端的處境才能真正看見神。尼布甲尼撒在王宮中的榮華並未帶領他認識神；正是在草地上吃草、如禽獸般生活時，他終於抬眼看見天上的神。
4. **真正的自由**：奇妙的是，當人看似失去一切自由時（被束縛於野獸的生活），其靈魂反而可能得到真正的自由——從罪的捆綁中釋放出來，來到神面前。

## 應用與默想
1. 你有否經歷過「被減至無」的時刻？在那段期間，你對神、對自己和對生命有了什麼新的認識？
2. 這則經文挑戰我們常見的成功觀念：真正的偉大不在於征服多少土地或積累多少財富，而在於能否承認自己一切皆來自於神。
3. 當你看到他人陷入絕境時，你的態度是什麼？像但以理一樣為他們禱告，還是像當時的巴比倫官員一樣高掛免懲牌？
4. 「如野獸生活」的經歷提醒我們：任何離開神而自高自大的嘗試，最終只會帶來更深的羞辱與痛苦。唯有謙卑才能帶來真正的尊貴。
