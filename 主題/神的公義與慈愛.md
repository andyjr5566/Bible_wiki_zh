# 神的公義與慈愛

## 概述

神的公義與慈愛並行不悖，是何西阿書的核心神學張力。神必須審判罪惡（公義），但祂的心意是挽回、復興（慈愛）。

## 聖經出現

- **何西阿 1:2-9** — 三個孩子的名字宣告審判：耶斯列（分散）、羅路哈瑪（不蒙憐憫）、羅阿米（非我民）
- **何西阿 1:10-11** — 然而……如海沙、稱為永生神的兒子
- **何西阿 2:14-23** — 曠野安慰、亞割谷指望、永遠婚約
- **何西阿 3:1-5** — 買贖、管教、末後歸回
- **何西阿 5:15** — 「我要回到原處，等他們自覺有罪，尋求我面」
- **何西阿 11:1-11** — 神的父愛：「我怎能捨棄你……我的心在我裡面翻轉」
- **何西阿 14:4** — 「我必醫治他們背道的病，我必以甘心愛他們」
- **羅馬書 11:22** — 「神的恩慈和嚴厲：向跌倒的人是嚴厲的，向你是恩慈的」

## 神學意義

1. **公義要求審判**：罪必須被處理，神不輕忽罪惡
2. **慈愛提供出路**：審判不是終點，是為了復興
3. **張力的解決**：在基督十字架上完全顯明——公義得滿足，慈愛得彰顯
4. **何西阿的體現**：先知的婚姻生活身歷這張力

神的公義與慈愛不是矛盾，而是完美統一。我們既要敬畏神的聖潔，又要確信神的愛。十字架是這張力的最終解決。
