# 亞伯尼歌

## 概述

亞伯尼歌（Abednego）是[[但以理]]的三個朋友之一，原名[[亞撒利雅]]（Azariah）。在[[巴比倫]]被擳時期，他與[[沙得拉]]和[[米煞]]一同被選入宮廷服務。他們因拒絕跪拜[[尼布甲尼撒]]王的金像而被投入烈火窯，卻因神的同在而毫髮無傷。

## 聖經出現

- **但以理書 1:6-7** — 亞撒利雅被選入宮廷，改名為亞伯尼歌
- **但以理書 3:1-30** — 烈火窯事件
- **但以理書 3:26-28** — 亞伯尼歌從窯中出來，毫髮無傷

## 神學意義

1. **信仰的堅持**：亞伯尼歌拒絕向金像下跪
2. **神的同在**：烈火窯中神的同在
3. **名字的同化**：巴比倫改名是文化同化的手段
4. **信心的獎賞**：因著信心，經歷了神的拯救
