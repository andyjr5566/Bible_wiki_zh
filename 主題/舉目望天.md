# 舉目望天

## 聖經參考
- 但以理書 4:34
- 詩篇 121:1-2; 123:1
- 路加福音 18:13
- 提摩太前書 2:8

## 概述
「舉目望天」是但以理書第四章描寫尼布甲尼撒王恢復理性的關鍵時刻。經文記載：「日子滿足，我—尼布甲尼撒舉目望天，我的聰明復歸於我，我便稱頌至高者，讚美尊敬活到永遠的神」（v.34）。在經歷了七年如野獸般的生活後，王终于抬起眼來看向天上的神，這個簡單的動作標誌著他內在發生了徹底的改變——從只顧塵世事務的傲慢君王，轉變為承認天上神為至高統治者的謙卑敬拜者。

這個動作具有深刻的象徵意義：
1. **由地上轉向天上**：從此前只看見地上權勢與榮華（象徵之「大樹」）到現在轉向審視宇宙的終极權威。
2. **由自我中心轉向神中心**：不再以自我為 riferimento，而是承認一切權勢與能力的真正來源。
3. **由眼目忙碌轉向心靈專注**：從忙於建城擴土的行動，轉為安靜默察天上的主。
4. **由物質依賴轉向屬靈依賴**：不再仰望人力與權勢，而學習依靠那活到永遠的神。

在聖經中，「舉目望天」常常與尋求神的幫助、承認神為救恩之源連結。詩人宣稱：「我舉目向山上看——我的幫助從何而來？我的幫助從造天地的耶和華而來」（詩121:1-2）。稅收長子禱告時「不敢舉目望天，只捶着胸說：神啊！開恩憐我這個罪人」（路18:13）。這個姿勢表達的是謙卑、依賴和對神的認識。

在尼布甲尼撒的情況中，「舉目望天」不僅是個體姿勢的改變，更象徵著整個生命 orientation 的根本改變：從以自我為中心的統治者，轉為承认天上神為萬有之主的臣民。

## 神學意義
1. **悔改的第一步**：真正的悔改始於轉變我們的注視點——從自我的問題、處境或他人，轉向神的面。就像浪子在遠方「醒過來」時首先做的事是「我要起來去見我父」（路15:18）。
2. **認識神是智慧的開端**：「敬畏耶和華是智慧的開端」（箴9:10）。當我們真正看見神的榮聖、權能和慈愛時，智慧自然隨之而來——正如經文所說：「我的聰明復歸於我」緊接在「舉目望天」之後。
3. **眼睛靈的啟示**：我們的眼睛不僅反映我們關注什麼，也塑造我們成為什麼樣的人。持續 olhar 向世上的榮華會使我們變得世俗；持續 sguardo 向天上的榮光則使我們被逐步變成主的樣式（林後3:18）。
4. **希望的象徵**：即使在最黑暗的處境中，「舉目望天」提醒我們仍有超越當前苦難的盼望——因為那坐在寶座上的神仍然掌管一切。

## 應用與默想
1. 你目前正在將目光放在什麼上面？是環境的困難、他人的評價，還是過去的失敗？試著有意識地把目光轉向天上——那裡有永活的神在注視著你。
2. 建立日常的「舉目望天」習慣：不論是清晨醒來時、工作間隙，或是晚上躺下前，花幾秒時間將你的注意力從地上事物轉向天上的主。這簡單的練習能逐漸重塑你的心態和觀點。
3. 當你看到他人遭遇苦難時，不僅要提供實際幫助，更要幫助他們抬起眼來看見那位在苦難中仍與他們同在的神。
4. 想想看：如果尼布甲尼撒沒有在那個關鍵時刻「舉目望天」，他的故事會如何結束？這提醒我們，我們此刻的眼光方向，正在決定我們未來的命運。
