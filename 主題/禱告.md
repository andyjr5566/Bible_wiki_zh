# 禱告

## 概述

「禱告」（Prayer）是人與上帝之間最基本的溝通方式，是信仰生活的核心。在聖經中，禱告不僅是人向上帝說話，更是上帝啟示祂自己旨意、與人建立關係的途徑。從創世記到啟示錄，禱告貫穿了整個救贖歷史——先祖與上帝同行時呼求祂的名，先知在異象中為百姓代求，詩人在詩篇中傾心吐意，耶穌基督親自教導門徒如何禱告，使徒們在禱告中建立教會。

在但以理書第九章中，[[但以理]]的禱告是聖經中最經典的認罪代求禱告之一：他因讀到[[耶利米]]有關耶路撒冷荒涼七十年的預言，便以禁食、披麻蒙灰的方式向神懇求（但以理書 9:3-19）。神立即差遣天使[[加百列]]回應他的禱告，並啟示了關乎彌賽亞和末世的重要預言——「[[七十個七]]」。本章清楚地表明：「禱告」不僅改變人的處際，更啟動了上帝啟示的闸门。

#### 聖殿與禱告

- [[列王紀上]] 8:22-53 — 所羅門獻殿禱告：七個祈求（無辜受屈、仇敵攻擊、旱災、饑荒等），求上帝從天上垂聽

- 但以理書 6:10 — 但以理在窗戶向耶路撒冷方向禱告，一日三次

#### 但以理書第九章的禱告

但以理書第九章的禱告（9:3-19）是舊約禱告的典範，包含四個要素：

1. **敬拜**（第4節）：承認上帝是「大而可畏」、「守約施慈愛」的
2. **認罪**（第5-15節）：以「我們」與全民認同，不推卸責任，承認上帝的審判是公義的
3. **代求**（第16-19節）：為聖城耶路撒冷、荒涼的聖所、上帝的百姓求恩典
4. **動機**（第17-19節）：不是靠自己的義，而是靠上帝的大憐憫，為上帝榮耀的名的緣故

[[加百列]]在「獻晚祭的時候」（下午三點）飛來，顯示上帝「一發出命令」就立即回應（9:23）。

#### 尼希米記中的禱告

[[尼希米]]也是隨時禱告的典範：

- [[尼希米記]] 1:4-11 — 尼希米得知耶路撒冷城牆被毀，坐下哭泣、禁食禱告數日；他的禱告結構與但以理書第九章相似（敬拜、認罪、代求、抓住應許）
- [[尼希米記]] 2:4 — 在王面前驚惶時，「默禱天上的神」
- [[尼希米記]] 4:4-5 — 面對敵人威脅時禱告
- [[尼希米記]] 4:9 — 「我們禱告我們的上帝，並派人看守防備」：禱告與行動並行
- [[尼希米記]] 5:19 — 「求你記念我所行的，施恩與我」

#### 尼希米記第九章的群體禱告

[[尼希米記]] 9:1-38記載了以色列百姓的群體認罪禱告，回顧了從亞伯拉罕到被擄歸回的救贖歷史。這是舊約中最長的禱告之一，結構清楚：稱頌上帝的創造和揀選（4-8節）、回顧出埃及和曠野經歷（9-21節）、承認當前的苦難（32-37節）、立約跟從上帝（38節）。

## 神學意義

1. **禱告的根基：上帝的主權與慈愛**：禱告不是改變上帝的旨意，而是以上帝的主權和慈愛為信靠的支點。但以理禱告時，上帝已經「發出了命令」（但9:23）；尼希米祈求的是上帝已經應許的（尼1:8-9，引用利26、申30）。真正的禱告是「願你的旨意成就」，因為上帝的智慧和良善遠高過我們所能測度的。

2. **禱告是與上帝的關係，不是宗教儀式**：耶穌責備法利賽人以「重複的話」和「故意叫人看見」的方式禱告（太6:5-8）。禱告的核心是與上帝真誠的相交，而不是外在的形式。詩篇51:17說：「上帝所要的祭就是憂傷的靈；上帝啊，憂傷痛悔的心，你必不輕看。」

3. **代禱：與百姓認同的禱告藝術**：「代求」（Intercession）是聖經禱告中最美麗的形式之一。但以理與全民的罪認同（「我們犯了罪」），摩西甚至求上帝「把這本書從你的冊上塗抹」（出32:32），保羅說「為我弟兄、我骨肉之親，自己被咒詛，與基督分離也願意」（羅9:3）。這種「認同式禱告」反映了基督為罪人代求的精神。

4. **禱告與上帝的公義**：聖經中的悔改認罪禱告，總是承認上帝的公義和審判的合理性。但以理說：「你是公義的」（但9:7, 14）；尼希米說：「你今日所行的，是公義的」（尼9:33）。禱告不是控告上帝不公，而是在上帝的聖潔面前謙卑自己。

5. **禱告與行動的結合**：[[尼希米]]是最好的榜樣——他禱告，但也派人看守城牆（尼4:9）；他祈求上帝紀念，但也自己採取措施（尼4:14）。雅各書2:17說：「信心沒有行為是死的。」禱告和行動相輔相成，不可偏廢。

6. **聖靈在禱告中的角色**：[[羅馬書]] 8:26說：「我們的軟弱有聖靈幫助，我們本不曉得當怎樣禱告，只是聖靈親自用不出來的歎息替我們禱告。」禱告不僅是人的努力，更是聖靈的工作。我們禱告的能力和動因，都是上帝所賜的。

7. **禱告的末世意義**：在末世，禱告更具有特殊的意義。[[啟示錄]] 8:3-4記載了「眾聖徒的禱告」如香升到上帝面前。但以理書中天使的回應也表明，末世的事件（如七十個七的成就）與禱告密切相關（但10:12：「從你第一日专心求明白……，你的言語已蒙應允」）。

## 應用與默想

1. 但以理和尼希米的禱告都是以上帝的話語（聖經）為基礎的。但以理從耶利米書得知七十年的預言，便以此禱告。你的禱告是否以上帝的應許為根基？你是否花時間讀聖經，以至於可以準確地向上帝祈求？

2. 但以理禱告時「與百姓認同」——雖然他自己未必犯了和百姓同樣的罪，但他說「我們犯罪、作孽」。你的禱告是否只關注自己的需要，還是也為你周圍的人、你的教會、你的城市代求？

3. 尼希米在危急中「默禱天上的神」（尼2:4），然後回答王的問題。這提醒我們，禱告不必是長篇大論，也可以是在忙碌生活中的短暫呼求。你是否培養了「隨時隨地」禱告的習慣？

4. 耶穌在客西馬尼園的禱告說出：「不要成就我的意思，只要成就你的意思。」在你目前的處境中，你是否願意將自己的計畫交給上帝，順服祂的時間和方式？

5. 雅各書5:16說「義人祈禱所發的力量是大有功效的」。你是否相信禱告真的「有效」？你是否在禱告上持之以恆，還是容易灰心？
