# 瑪加比書

## 概述

《瑪加比書》是聖經次經（Apocrypha）的一部分，包括《[[瑪加比一書]]》和《[[瑪加比二書]]》。這些書卷記載了[[哈斯蒙尼]]家族領導的猶太人起義（前167年—前160年）以及隨後的事件，是了解兩約之間歷史的重要文獻。

## 聖經出現

### 瑪加比一書

- 記載從安条克四世褻瀆聖殿（前167年）到西蒙統治時期（前142年）的歷史
- 以希伯來文寫成，後被翻譯為希臘文

### 瑪加比二書

- 記載從大祭司奧尼亞斯三世（前187年）到尼迦諾爾戰役（前161年）的歷史
- 以希臘文寫成，是五卷書的摘要

## 神學意義

1. **歷史橋樑**：填補舊約和新約之間的歷史空白
2. **信仰的見證**：記載了猶太人對信仰的堅持
3. **復活教義**：瑪加比二書包含復活的教義（瑪加比二書 7章）
4. **殉道的典範**：記載了殉道者的故事
5. **修殿節的起源**：記載了潔淨聖殿和修殿節的起源
