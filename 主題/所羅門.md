# 所羅門

## 概述

所羅門是以色列聯合王國的第三位國王，大衛的兒子，以智慧和財富聞名於世。他在位期間建造了第一聖殿。

## 聖經出現

- **列王紀上 1-11章** — 所羅門的一生
- **歷代志下 1-9章** — 所羅門建聖殿和統治
- **箴言、雅歌、傳道書** — 所羅門的著作
- **馬太福音 6:29; 12:42** — 耶穌提到所羅門

## 神學意義

1. **智慧的象徵**：所羅門求智慧的故事是謙卑求問的典範
2. **聖殿的建造者**：所羅門聖殿是以色列敬拜的中心
3. **墮落的警告**：所羅門的晚年警告我們，即使是最親近神的人也可能偏離
4. **基督的預表**：所羅門作為和平之君預表基督
