# 敘利亞以法蓮戰爭

## 概述

主前 734-732 年，敘利亞（亞蘭）與北國以色列（以法蓮）結盟攻打南國猶大，試圖廢除亞哈斯王，另立傀儡王。這場戰爭直接導致猶大王亞哈斯求助亞述，改變了整個地區的勢力格局。

## 歷史背景

- **時間**：主前 734-732 年
- **參戰方**：
  - 敘利亞王利辛
  - 北國以色列王比加（以法蓮支派）
  - 南國猶大王亞哈斯
- **目的**：廢亞哈斯，立「他比勒之子」為王（賽 7:6）
- **結果**：亞哈斯求助亞述王提革拉毘列色三世；亞述攻打大馬色、佔領加利利、基列；北國元氣大傷

## 聖經出現

- **何西阿 5:8-9** — 「在基比亞吹角，在拉瑪吹號……以法蓮必變為荒場」
- **何西阿 5:13** — 「以法蓮見自己有病……往亞述去見耶雷布王」
- **何西阿 7:11** — 「以法蓮如無心的鴿子……往亞述去」
- **何西阿 12:1** — 「以法蓮追隨東風……與亞述立約」
- **列王紀下 16:5-9** — 亞哈斯求助亞述
- **以賽亞 7:1-17** — 以馬內利預言的背景
- **歷代志下 28** — 猶大慘敗細節

## 神學意義

1. **內部衝突招致外患**：以色列攻打猶大，反被亞述吞併
2. **政治聯盟的虛假**：倚靠人不倚靠神，反被所倚靠者吞滅
3. **神的主權掌管歷史**：敘利亞-以法蓮聯盟看似強大，實在神手中
4. **何西阿的見證**：先知在戰爭期間事奉，見證北國走向滅亡
