# 神的主權

## 聖經參考
- 但以理書 4:17, 25, 34-35
- 詩篇 103:19; 115:3
- 但以理書 2:20-22
- 提摩太前書 6:15
- 啓示錄 19:16

## 概述
「神的主權」是但以理書第四章的核心主題，貫穿全章始終。這個概念指的是神作為宇宙的創造者和維護者，對一切受造之物擁有絕對的最高權威和統治權。在但以理書第四章中，這個抽象的神學概念透過巴比倫王尼布甲尼撒親身的歷史事件得到生動的體驗和見證。

章節開頭，尼布甲尼撒就宣告：「他的國是永遠的；他的權柄存到萬代」（v.3），並且在夢的解釋中多次重申：「好叫世人知道至高者在人的國中掌權，要將國賜與誰就賜與誰，或立極卑微的人執掌國權」（v.17），以及「等你知道至高者在人的國中掌權，要將國賜與誰就賜與誰」（v.25）。章節結尾，王親身經歷後宣告：「祂的權柄是永有的；他的國存到萬代」（v.34），以及「世上所有的居民都算為虛無；在天上的萬軍和世上的居民中，他都憑自己的旨意行事，無人能擋住他的手，或問他說：你做什麼呢？」（v.34-35）。

這段經歷從理論認識變為生命體驗的過程，充分展示了神主權的幾個面向：

1. **對人間王權的主權**：神不僅創立王權（但以理書2:21），更有權按照祂的旨意提升或廢除君王。尼布甲尼撒親身經歷了從世界上最強大的君王變為如野獸般生活的被廢君，這徹底粉碎了人類對權力永久性的幻想。

2. **對時間與環境的主權**：神精確地安排了這場試煉的時間——不是瞬間發生，也不是永無止境，而是恰好的十二個月緩衝期（給予悔改機會）加上七年管教期（足以產生徹底改變）。這表明神不僅做「什麼」，也精確掌控「何時」和「如何長久」。

3. **對人類心靈的主權**：神不僅改變尼布甲尼撒的外在處境，更深入到他的心智之中——讓他的變為獸心，又在適當時候恢復他的人性智慧。這顯示神的權柄延伸到人內在最深處的領域。

4. **對萬有的主權**：正如結尾所宣告的，「天上的萬軍和世上的居民中，他都憑自己的旨意行事」。神的權柄不受限於特定領域或類型的受造之物，祂是一切存在的绝对主宰。

在聖經整體啟示中，神的主權是一條貫穿始終的主線。從創世記開頭「在起初，神創造天地」到啟示錄結尾「阿們！甚至願主耶穌快快來！」，一貫的信息是：宇宙有一位至高的統治者，祂的旨意最終必會成就。

## 神學意義
1. **敬拜的基礎**：認識神的主權是正確敬拜的根基。只有當我們承認祂是萬有之主時，我們才能真正地將心魂所มี的愛、順服和敬拜獻給祂。
2. **信心的根基**：在患難中堅持信心的關鍵是相信那掌管萬有的神也在看顧我們的個別生活。正如詩人所言：「耶和華為王！願萬民震動！祂坐在基路伯上，路比倫上，願地動盪」（詩99:1）。
3. **倫理的依據**：如果真有一位至高的道德立法者和審判者，則倫理就不僅是人類社會的慣例，而是根基於宇宙的道德秩序之上。這使得道德主張具有客觀約束力。
4. **安慰的源頭**：當世界看似混亂不堪時，知道有一位智慧而良善的主宰在掌掌一切，是極大的安慰與盼望來源。正如使徒保羅所宣告的：「我們曉得萬事都互相效力，叫愛神的人得益處」（羅8:28）。
5. **宣教的動力**：既然有一位對萬民萬國有權柄的神，則我們有責任將這個好消息傳向一切民族——因為祂願意「將國賜與誰就賜與誰」。

## 應用與默想
1. 你目前正面對什麼看似無法控制的情況？試著將這情況交托給那位「在天上的萬軍和世上的居民中，他都憑自己的旨意行事」的神。
2. 在決策時，不僅詢問「這對我來說是否合適？」，更要問「這符合那位掌權者的旨意嗎？」這將幫助我們做出更永恆價值的選擇。
3. 當你看到世界上有不公義或苦難時，記得最終的審判權在乎那位「無人能擋祂手」的神。這不該使我們放鬆對公義的追求，而是應該讓我們在追求公義時有深深的平安——因為最終勝利祂已經應許。
4. 想想看：如果你真正相信並活出「神的主權」這個真理，你今天的態度、決策和人際關係會有什麼不同？
