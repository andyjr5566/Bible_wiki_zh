# 神的棄絕非永遠

## 概述

何西阿書透過「羅阿米」（非我民）→「阿米」（我民）的名字翻轉，宣告神的棄絕有期限，恩典最終得勝。這是整卷書的福音核心。

## 聖經出現

- **何西阿 1:9-10** — 羅阿米（非我民）→「然而……稱為永生神的兒子」
- **何西阿 2:1, 23** — 阿米（我民）、路哈瑪（蒙憐憫）
- **何西阿 3:5** — 「末後的日子……必以敬畏的心歸向耶和華」
- **何西阿 6:1-3** — 「祂撕裂，也必醫治；打擊，也必包裹」
- **何西阿 11:8-11** — 「我怎能捨棄你……我的心在我裡面翻轉」
- **何西阿 14:4** — 「我必醫治他們背道的病，我必以甘心愛他們」
- **耶利米 31:3** — 「我以永遠的愛愛你」
- **羅馬書 11:1-2, 25-26** — 「神棄絕了祂的百姓嗎？斷乎不是……以色列全家都要得救」

## 神學意義

1. **棄絕有期限**：為了管教，非永遠拋棄
2. **恩典勝過審判**：雅 2:13 —「憐憫向審判誇勝」
3. **神的信實**：神的恩賜和呼召沒不後悔（羅 11:29）
4. **何西阿的生活見證**：先知三次買贖歌篾，身歷此真理
5. **終極應驗**：基督的十字架與復活
