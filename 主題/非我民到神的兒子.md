# 非我民到神的兒子

## 概述

何西阿書 1:10 「從前在甚麼地方對他們說『你們不是我的子民』，將來在那裡必對他們說『你們是永生神的兒子』」被保羅（羅 9:25-26）和彼得（彼前 2:10）引用，成為新約解釋外邦人得救、以色列得救的關鍵經文。

## 聖經出現

- **何西阿 1:10** — 「然而，以色列的人數必如海沙……將來在那裡必對他們說『你們是永生神的兒子』」
- **何西阿 2:23** — 「本非我民的，我必對他說：你是我的民；他必說：你是我的神」
- **羅馬書 9:25-26** — 「正如何西阿書上所說：『那本來不是我子民的，我要稱為我的子民；本來不是蒙愛的，我要稱為蒙愛的。』在從前對他們說『你們不是我的子民』的地方，必稱為『永生神的兒子』」
- **彼得前書 2:10** — 「你們從前不是子民，現在是神的子民；從前未曾蒙憐憫，現在卻蒙了憐憫」
- **羅馬書 11:25-26** — 「以色列全家都要得救」

## 神學意義

1. **何西阿的原意**：指北國以色列被擄後的復興
2. **保羅的用法**：應用在外邦人得救，顯明神的救恩計劃超越民族界限
3. **彼得的用法**：應用在教會（含猶太與外邦信徒）的身分確認
4. **神學張力**：原意（以色列復興）與新約用法（外邦人得救）的關係
4. **終極應驗**：羅 11:26 「以色列全家都要得救」——雙重應驗

## 應用與默想

神的救恩計劃超越我們的預期。原本針對以色列的復興應許，在基督裡擴展到萬國萬民。
