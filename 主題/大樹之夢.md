# 大樹之夢

## 聖經參考
- 但以理書 4:10-17
- 但以理書 4:19-26

## 概述
尼布甲尼撒在但以理書第四章所做的夢，其中央意象是一棵極其巨大的樹，位於地中（世界中心），枝葉繁茂，提供庇護和食物給一切生物。這棵樹象徵著尼布甲尼ša王本人及其龐大的權勢。隨後，一位守望的聖者從天而降，宣告要砍倒這棵樹，但留下樹墩，用鐵銅箍住，使王失去理性，如野獸般生活七時（年），直至他承認至高者在人間萬國中掌權。

這個夢與但以理書第二章的金像夢形成呼應：兩者皆為神所賜的啟示夢，但前者針對王的個人驕傲，後者則預言歷史進程。樹的意象在古近東文化中常用來代表國家或君王（參見以西結書31章），其被砍伐卻留下餘根，象徵刑罰是暫時的，旨在帶來悔改與恢復。

## 神學意義
1. **權勢的虛妄**：樹的壯麗表面之下隱藏著脆弱——只要神一命令，它即被徹底毀滅。這提醒我們一切人間權勢終於歸於無有。
2. **神的主權**：夢的解釋明確指出，這一切發生是因為「至高者在人的國中掌權」（v.25）。人間的王權絕非自生自滿，皆源於神的許可與安排。
3. **管教的目的**：樹被砍伐但留下根莖，表明神的懲罰不是要徹底毀滅，而是要帶來悔改。正如園丁修剪果樹是為結果更豐盛，神管教兒女也是為成就聖潔。
4. **人的無助**：當樹被毀時，所有依賴它的生物都失去棲身之所和食物來源。這象徵當人以權勢、財富或地位為依靠時，一旦這些崩塌，將陷入極端的無助與痛苦。

## 應用與默想
1. 你生命中有什麼「大樹」——你視為安全感、身份或成功之源的事物？想象一下，如果神命令將它砍伐，你會如何反應？
2. 這則夢境提醒我們：真正的安全感唯有在神裡面才能找到。當我們將信心放在暫時的事物上時，最終必會經歷失落的痛苦。
3. 神的管教總是帶有恢復的希望——就像樹雖被砍伐卻留下根莖，祂在我們身上的工作永遠不是毀滅而是更新。
4. 你是否曾經歷過「被砍伐的樹」般的經驗？在那段時間裡，你學到了什麼關於神和自己的功課？
