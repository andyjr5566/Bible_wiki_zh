# [[何西阿]]書 第3章 註解

## 一、本章總結

本章是[[何西阿]]婚姻敘事的完結篇。神命令先知再次買贖不貞的妻子[[歌篾]]，象徵神對[[以色列]]不變的愛。前三節是「救贖與管教」，後兩節是「預言與應許」。全章以「[[耶和華]]還是愛他們」開始，以「領受祂的恩惠」結束，完整呈現神的愛之始終。

---

## 二、逐節重點

### 第1節
- **經文**：[[耶和華]]對我說：「你再去愛一個淫婦，就是她情人所愛的；好像[[以色列]]人，雖然偏向別[[神]]，喜愛葡萄餅，[[耶和華]]還是愛他們。」
- **核心行動**：神命令[[何西阿]]「再去愛」——第二次的命令，第一次在第一章
- **「淫婦」的身份**：大多數解經家同意就是[[歌篾]]，她離家淫奔另找情人（二5）
- **「情人所愛的」**：「情人」在此指前夫[[何西阿]]本人（非歌篾的情人），強調她被對方所愛而非她愛對方
- **「葡萄餅」**：用曬乾葡萄製成的餅，在拜[[巴力]]禮儀中用來獻祭；象徵[[以色列]]在真神之外另有所愛
- **「[[耶和華]]還是愛他們」**：整卷書的核心信息——神的愛不因人的不忠而改變
- **話中之光**：
  - 人間的愛是有條件的；神的愛不是因為我們可愛，甚至我們變心祂仍愛我們
  - 許多人表面愛神，實際上在神之外另有所愛

### 第2節
- **經文**：我便用銀子十五舍客勒，大麥一賀梅珥半，買她歸我。
- **贖價分析**：
  - 銀子十五舍客勒 = 半個奴僕的贖價（出二十一32）
  - 大麥一賀梅珥半 ≈ 銀子十五舍客勒
  - 總值約三十舍客勒 = 一個奴僕或婢女的完整贖價
- **背景**：歌篾可能已賣身為妓或積欠債務，需要贖身
- **話中之光**：
  - 神在我們還作罪人的時候，就付出重大代價差遣耶穌基督為我們捨命（羅五8）
  - 我們是重價買來的，要在身上榮耀神（林前六20）

### 第3節
- **經文**：我對她說：「你當多日為我獨居，不可行淫，不可歸別人為妻，我向你也必這樣。」
- **「獨居」的重點**：非指單獨，而是**不可再與別人有來往**，專心與丈夫在一起
- **「多日」**：非永久，但為相當時日，使婦人因淫亂不潔經過一段時間
- **「我向你也必這樣」**：[[何西阿]]也同樣不親近她，即**禁慾**
- **屬靈意義**：自潔的程序，透過禁慾讓歌篾完全回心轉意
- **話中之光**：
  - 與世俗為友就是與神為敵（雅四4）
  - 凡向主有指望的，就潔淨自己（約壹三2-3）

### 第4節
- **經文**：[[以色列]]人也必多日獨居，無君王，無首领，無祭祀，無柱像，無以弗得，無家中的神像。
- **六項「無」分為三組**：
  | 組別 | 項目 | 說明 |
  |------|------|------|
  | **政治** | 無君王 | 無領導中心，必生混亂 |
  | | 無首領 | 無軍事領袖，失去安全保障 |
  | **宗教** | 無祭祀 | 無贖罪方法，對耶和華的祭祀也停止 |
  | | 無以弗得 | 大祭司聖服，後成占卜工具 |
  | **民間信仰** | 無柱像 | 迦南神廟祭壇旁的石柱，代表[[巴力]] |
  | | 無家中的神像 | 「特拉芬」，可移動的小偶像，用於占卜 |
- **象徵意義**：歌篾的「多日獨居」寓意[[以色列]]的被擄
- **話中之光**：
  - 「獨居」意指分別為聖，不讓神以外的一切人、事、物佔據心中地位
  - 務要從他們中間出來，與他們分別（林後六17-18）

### 第5節
- **經文**：後來[[以色列]]人必歸回（或譯：回心轉意），尋求他們的[[神]]─[[耶和華]]和他們的王[[大衛]]。在末後的日子，必以敬畏的心歸向[[耶和華]]，領受祂的恩惠。
- **「歸回」**：從被擄之地歸回迦南美地
- **「他們的王[[大衛]]」**：指[[大衛]]的後裔，即[[彌賽亞]]（耶三十9；結三十四24）
- **「末後的日子」**：指[[彌賽亞]]時代（珥二28；來一2）
- **「敬畏」**：原文為恐懼戰兢的態度，在耶和華恩惠面前含有歡喜感恩與虔恭敬畏
- **靈意註解**：「尋求耶和華和大衛」二者合而為一，即指救主耶穌基督；末日以色列全家都要得救（羅十一26）
- **話中之光**：
  - 我們若轉向神，神就必轉向我們（亞一3；瑪三7）
  - 苦難能幫助我們學會順從（來五8）而敬畏神

## 三、整體神學主題

1. **神不變的愛**：雖以色列偏向別神，耶和華還是愛他們——本書中心思想
2. **先知的行動比喻**：[[何西阿]]買贖[[歌篾]]，具體演示神對百姓的慈愛
3. **管教的意義**：暫時的剝奪（無君王、無祭祀等）是為了最終的復興
4. **[[彌賽亞]]盼望**：預言百姓將歸回，尋求耶和華和大衛的後裔
5. **悔改的步驟**：承認淫亂 → 接受代價 → 自潔獨居 → 歸回尋求神 → 敬畏領受恩惠

## 四、待確認事項

- 「葡萄餅」是否僅指巴力禮儀中的供物，或也包括一般食物
- 「十五舍客勒」的準確現代價值是否需要更多說明
- 「末後的日子」的歷史應驗時間線（部分應驗 vs. 完全應驗）
- 第4節的「多日」是否指整個教會時代
